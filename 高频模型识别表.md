# 大学物理A1 — 高频模型识别表（看到什么→用什么定律）

| 题目关键词/场景 | 识别模型 | 适用定律/方法 | 章节 |
|-----------------|---------|--------------|------|
| 速度、速率、加速度、位移 | 质点运动学 | v=dr/dt, a=dv/dt, a_t/a_n分解 | Ch1 |
| 圆周、半径R、角速度ω、转/秒 | 圆周运动 | a_n=v^2/R, v=Rω, a_t=Rα | Ch1 |
| 平抛、斜抛、仰角θ、射程、最高点 | 抛体运动 | x/y分解, 最高点a_n=g | Ch1 |
| 受力分析、加速度、连接体、滑轮 | 牛顿第二定律应用 | 隔离体法→ΣF=ma(system) | Ch2 |
| 碰撞、打击、子弹射入、瞬间 | 碰撞问题 | 动量守恒(首选!)→如弹性+动能守恒 | Ch2 |
| 弹簧压缩/拉伸、斜面下滑、摆 | 功能关系 | 动能定理/机械能守恒(看是否有摩擦) | Ch2 |
| 爆炸、分裂、反冲(人车、火箭) | 动量守恒 | ΣF_ext=0→Σp=const | Ch2 |
| 杆转动、圆盘转动、滑轮有质量 | 刚体定轴转动 | M=Iα, I查表+平行轴, L=Iω | Ch3 |
| 收臂加速、花滑、转盘走人 | 角动量守恒 | I₁ω₁=I₂ω₂(合外力矩=0) | Ch3 |
| 纯滚动(球/柱滚下斜面) | 平动+转动联合 | v_c=Rω, Ek=mv_c²/2+I_cω²/2 | Ch3 |
| 高速(>0.1c)、飞船、μ子、同时性 | 狭义相对论 | γ因子→时间膨胀/长度收缩/质能方程 | Ch4 |
| 弹簧振子、摆、A,ω,T,f,φ,初始条件 | 简谐振动 | x=Acos(ωt+φ), v=-ωAsin, a=-ω²x | Ch5 |
| 波函数、传播方向、波长、频率 | 平面简谐波 | y=Acos[ω(t-x/u)+φ] | Ch6 |
| 两列波叠加、增强减弱、相干 | 波的干涉 | Δφ=2kπ(加强)或(2k+1)π(减弱) | Ch6 |
| 反射波、固定端、波节波腹 | 驻波 | 驻波方程+半波损失+波节/波腹位置 | Ch6 |
| 频率变、波源运动/观察者运动 | 多普勒效应 | ν'=ν₀·(u±v_R)/(u∓v_S) | Ch6 |
| p,V,T, ν,摩尔,状态方程 | 理想气体宏观描述 | pV=νRT, T=t+273.15 | Ch7 |
| 分子动能、压强微观、能均分 | 气体动理论 | E=iνRT/2, ⟨ε_k⟩=(i/2)kT | Ch7 |
| 吸热放热、做功、等温等容等压绝热 | 热力学过程分析 | ΔU=Q+W→判断哪个量为0→计算 | Ch8 |
| 热机、效率、循环、高温低温热源 | 循环与效率 | η=W_net/Q_in, η_carnot=1-T₂/T₁ | Ch8 |
