第7章 理想气体动理论Kinetic Theory of Ideal Gas — 从宏观到微观的桥梁

一罐氧气瓶能供车间用几天？为什么温度越高气体压强越大？这些问题的答案，藏在万亿个分子的无序碰撞之中。本章搭建从「宏观量 p, V, T」到「微观分子运动」的桥梁 —— 压强是分子撞壁的集体表现，温度是分子平均平动动能的量度，而内能则是所有分子各自由度动能的总和。
TL;DR 理想气体状态方程 pV = νRT 和 p = nkT 是宏观描述；微观上，压强 p = $1/3$nm₀⟨v²⟩（分子撞壁的统计平均），温度 $3/2$kT = $1/2$m₀⟨v²⟩（平均平动动能的量度）。能均分定理给每个自由度分配 $1/2$kT 的平均动能，内能 E = $i/2$νRT 只取决于温度和自由度。麦克斯韦速率分布描述分子速率的统计规律，三种特征速率大小关系：v_p < ⟨v⟩ < v_rms（方均根最大）。

平衡态、状态参量与温度

热力学系统与平衡态 必考 页3页4

热力学系统：由大量微观粒子组成的有限宏观物质系统。系统之外与系统有相互作用的环境称为「外界」。按能量/物质交换分为三类：

  • 开放系统 —— 与外界既有能量交换也有物质交换
  • 封闭系统 —— 与外界有能量交换但无物质交换
  • 孤立系统 —— 与外界既无能量交换也无物质交换

平衡态：在不受外界影响的条件下，孤立系统的宏观性质长时间不随时间变化的状态。注意：平衡态是热动平衡—— 微观粒子仍在做无规则热运动，只是宏观性质的统计平均值不再变化。

易错：平衡态 ≠ 静止。微观分子始终在无规则运动，平衡态只是宏观性质（p, V, T）不再变化，是动态平衡。

状态参量与热力学第零定律 必考 页5页6页7

宏观量：描述大量分子集体行为特征的物理量（如 p, V, T），由微观量的统计平均决定。

微观量：描述单个分子行为的物理量（如分子质量 m₀、速度 v、动能 ε）。

对比维度宏观量 $Macroscopic$微观量 $Microscopic$
描述对象大量分子的集体行为单个分子
典型例子压强 p、体积 V、温度 T分子质量 m₀、速度 v、动能 ε
测量方式直接可测（压力表、温度计等）不能直接测量，通过统计间接推得

三个状态参量：

  • 体积 V —— 气体分子自由运动的空间，单位 m³
  • 压强 p —— 垂直作用在容器壁单位面积上的气体压力，p = F/S，单位 Pa（帕斯卡），1 atm = 1.01325×10⁵ Pa
  • 温度 T —— 表征物体冷热程度的物理量

热力学第零定律：如果两个系统分别与第三个系统处于热平衡，则这两个系统也彼此处于热平衡。这一定律是测温原理的基础 —— 温度计与被测物体达到热平衡时，温度计读数即为被测物体的温度。

温标：摄氏温标与热力学温标 必考 页8页9

摄氏温标 t：以水的冰点（0 ℃）和沸点（100 ℃）为基准，利用水银或酒精的热胀冷缩特性标定。

热力学温标 T：以绝对零度为起点，单位 K（开尔文）。

T = t + 273.15

绝对零度 T = 0 K 对应 t = −273.15 ℃，只能无限逼近而不可达到。

平衡态下，气体的宏观性质是否随时间变化？微观分子是否静止？
参考答案宏观性质不随时间变化；但微观分子仍在做无规则热运动，平衡态是热动平衡，不是静止状态。
热力学温标 T 与摄氏温标 t 的关系是什么？
参考答案T = t + 273.15。绝对零度 T = 0 K 对应 t = −273.15 ℃。

理想气体状态方程

理想气体模型与三大实验定律 必考 页10页11

理想气体：在任何情况下都严格遵守三条实验定律的气体。实际气体在压强不太高（相对大气压）、温度不太低（相对室温）的条件下可近似为理想气体。

三条实验定律（一定质量气体）：

  • 玻意耳-马略特定律（Boyle-Mariotte）：等温条件下，pV = 常数（p ∝ 1/V）
  • 盖-吕萨克定律（Gay-Lussac）：等压条件下，V ∝ T
  • 查理定律（Charles）：等容条件下，p ∝ T

理想气体状态方程 必考 页12页13页14

物质的量：1 mol = 6.022×10²³ 个微粒（阿伏伽德罗常数 N_A）。

由三大定律综合，得到理想气体状态方程：

$$pV = \nu RT = \frac{m}{M}RT$$

其中 ν = m/M 为摩尔数，M 为摩尔质量（kg/mol），R = 8.31 J·mol⁻¹·K⁻¹ 为摩尔气体常量。

引入玻尔兹曼常数 k = R/N_A = 1.38×10⁻²³ J·K⁻¹，以及分子数密度 n = N/V（单位体积分子数），状态方程可改写为：

$$p = nkT$$

标准状态：p₀ = 1.01325×10⁵ Pa，T₀ = 273.15 K，此时 1 mol 理想气体的体积 V_mol = 22.4×10⁻³ m³ = 22.4 L，分子数密度 n₀ = 2.69×10²⁵ m⁻³（洛喜密脱数）。

易错：状态方程 pV = νRT 中的 T 必须是热力学温度（单位 K），不能直接代入摄氏温度！T = t + 273.15。
一瓶氧气体积 V₁=32 L，初始压强 p₁=1.317×10⁷ Pa，当压强降到 p₂=1.031×10⁶ Pa 时需充气。车间每天用 p₃=1.031×10⁵ Pa、V₃=400 L 的氧气，温度不变，问可用几天？
参考答案温度不变，m ∝ pV。初始质量 m₁ ∝ p₁V₁，需充气时剩余 m₂ ∝ p₂V₁，可用质量 Δm ∝ $p₁−p₂$V₁。每天用量 m_daily ∝ p₃V₃。N = Δm/m_daily = $p₁−p₂$V₁/$p₃V₃$ ≈ 9.4 天。

微观模型与压强公式

理想气体微观模型 必考 页16页17

四条基本假设：

  1. 质点假设 —— 分子线度远小于分子间距，可视为质点（忽略大小）
  2. 自由粒子假设 —— 除碰撞瞬间外，分子间及分子与器壁间无相互作用
  3. 弹性碰撞假设 —— 碰撞是完全弹性碰撞，动量守恒、动能守恒
  4. 忽略重力 —— 不计重力影响

概括：理想气体是一群自由无规则运动的弹性小球群。

平衡态统计假设：

  • 均匀分布 —— 分子数密度处处相等
  • 各向同性 —— 分子沿各方向运动的机会均等，速度各分量的各种平均值相等：
$$\bar{v}_x = \bar{v}_y = \bar{v}_z = 0, \quad \overline{v_x^2} = \overline{v_y^2} = \overline{v_z^2} = \frac{1}{3}\overline{v^2}$$

压强公式的物理图像 必考 页18页19

压强的微观本质：气体压强是大量分子不断碰撞器壁的宏观表现。单个分子碰撞器壁 → 动量变化 → 给器壁冲量 → 大量分子统计平均 → 单位面积单位时间内的平均冲量 = 压强。

推导思路（物理图像）：

  1. 一个分子以速度 v_x 撞向器壁 → 弹性碰撞反弹，速度变为 −v_x → 动量变化 Δp = 2m₀v_x
  2. Δt 时间内能撞到面积 A 上的分子数 = nAv_xΔt（只考虑向壁运动的分子）
  3. 总冲量 → 压强 p = F/A → 对大量分子取统计平均
  4. 利用各向同性条件 ⟨v_x²⟩ = ⟨v²⟩/3，最终得到：
$$p = \frac{1}{3} n m_0 \langle v^2 \rangle = \frac{2}{3} n \bar{\varepsilon}_{kt}$$

其中 n 为分子数密度，m₀ 为单个分子质量，⟨v²⟩ 为速率平方的统计平均值。

分子平均平动动能：

$$\bar{\varepsilon}_{kt} = \frac{1}{2} m_0 \langle v^2 \rangle$$
易错：压强公式中的因子 1/3 来源于各向同性假设 ⟨v_x²⟩ = ⟨v²⟩/3，不是随意取的。另外，压强 p 是宏观量，但公式右边的 n、m₀、⟨v²⟩ 都是微观量的统计平均 —— 这是连接宏观与微观的桥梁公式。
为什么说压强是大量分子碰撞器壁的统计结果？单个分子碰撞能产生稳定压强吗？
参考答案单个分子碰撞器壁产生的冲量是间歇、随机的，不能形成稳定的宏观压强。只有大量分子（~10²³量级）持续碰撞，统计平均后才呈现稳定的压强值。因此压强是统计量，对少量分子无意义。

温度与分子动能

温度公式：温度是分子平均平动动能的量度 必考 页20页21页22

将状态方程 p = nkT 与压强公式 p = $2/3$nε̄_kt 联立：

$$\bar{\varepsilon}_{kt} = \frac{1}{2} m_0 \langle v^2 \rangle = \frac{3}{2} kT$$

这是本章最核心的公式之一。它揭示了温度的微观本质：温度是分子平均平动动能的量度。温度越高，分子热运动越剧烈。

由此可得方均根速率（root-mean-square speed）：

$$v_{rms} = \sqrt{\langle v^2 \rangle} = \sqrt{\frac{3kT}{m_0}} = \sqrt{\frac{3RT}{M}} \approx 1.73\sqrt{\frac{RT}{M}}$$
易错：$1$ 温度公式 $3/2$kT 是「平均平动动能」，不是总动能。对有转动自由度的多原子分子，总平均动能大于平动动能。$2$ 方均根速率 v_rms = √$3RT/M$ 中的 M 是摩尔质量，单位用 kg/mol 而非 g/mol。

重要结论：温度相同的不同气体，其分子的平均平动动能相同，但方均根速率不同（因分子质量不同）。

两瓶不同气体（如 He 和 N₂），若温度相同：$1$ 平均平动动能是否相同？$2$ 方均根速率是否相同？$3$ 若分子数密度不同，压强是否相同？
参考答案$1$ 相同——ε̄_kt = $3/2$kT 只取决于温度，与气体种类无关。$2$ 不同——v_rms = √$3RT/M$，M 不同则 v_rms 不同。$3$ 不同——p = nkT，n 不同则 p 不同（即使 T 相同）。

能均分定理与理想气体内能

自由度 重点 页33页34页35

自由度 i：确定物体空间位置所需的独立坐标数。

分子类型平动自由度 t转动自由度 r总自由度 i = t+r例子
单原子分子303He, Ar
刚性双原子分子325H₂, O₂, N₂

易错：$1$ CO₂ 是线性分子（三原子共线），自由度为 5 而非 6。$2$ 双原子分子转动自由度为 2（不是 3），因为绕两个原子连线轴的转动惯量可忽略不计。$3$ 以上均为「刚性分子」模型——不考虑振动自由度。若温度足够高使得振动自由度被激发，自由度还会增加。

能量均分定理 必考 页36页37

能量均分定理：在温度为 T 的平衡态下，物质分子的每一个自由度都具有相同的平均动能，其大小等于 $1/2$kT。

推导来源：单原子分子只有 3 个平动自由度，平均平动动能 $3/2$kT，每个自由度分得 $1/2$kT。统计假设保证各自由度平等。

$$\text{分子平均总动能 } \bar{\varepsilon}_k = \frac{i}{2}kT$$
分子类型i平均总动能平均平动动能平均转动动能
单原子3$3/2$kT$3/2$kT0
双原子5$5/2$kT$3/2$kTkT
多原子63kT$3/2$kT$3/2$kT
易错：无论何种分子，平均平动动能恒为 $3/2$kT（因为平动自由度始终 = 3）。能均分定理增加的只是转动（和振动）部分的动能。

理想气体的内能 必考 页38页39页40

理想气体内能：模型中分子间无相互作用 → 无相互作用势能；刚性分子 → 无振动自由度。因此内能 = 所有分子平均动能之和。

$$E = \frac{i}{2} \nu RT = \frac{i}{2} \frac{m}{M} RT$$

关键结论：理想气体的内能只是温度的单值函数，内能变化只与始末状态有关，与过程无关。

1 mol O₂（刚性双原子，i=5）温度从 300 K 升高到 400 K，内能增加多少？（R = 8.31 J·mol⁻¹·K⁻¹）
参考答案ΔE = $i/2$RΔT = $5/2$ × 8.31 × $400−300$ = 2.5 × 8.31 × 100 = 2077.5 J ≈ 2.08×10³ J。
为什么理想气体内能只是温度的函数？实际气体也是这样吗？
参考答案理想气体模型中分子间无相互作用，无势能项，内能 = 所有分子动能之和。能均分定理给出动能 ∝ T，所以内能 ∝ T。实际气体分子间有相互作用势能，内能还与体积（分子间距）有关，不只是温度的函数。

麦克斯韦速率分布

统计规律与速率分布函数 重点 页24页25页26

统计规律：大量偶然事件整体所遵循的规律。单个分子行为是偶然的，大量分子整体表现出必然的统计规律（如伽尔顿板实验）。

麦克斯韦速率分布函数 f$v$：平衡态下，速率在 v ∼ v+dv 区间内的分子数占总分子数的比例为 f$v$dv。f$v$ 的物理意义是分子处于速率 v 附近单位速率区间的概率密度。

f$v$ 满足归一化条件：

$$\int_0^\infty f$v$\,dv = 1$$

分布曲线的特点：从原点出发 → 上升到极大值 → 逐渐下降 → 无限延伸但趋于零。曲线不对称，右侧尾巴较长。温度升高 → 曲线峰值右移且变扁平（高速分子增多）。

三种特征速率 必考 页24页25页26

速率定义公式数值关系 $以 √(RT/M$ 为单位)用途
最概然速率 v_p 分布函数 f$v$ 取极大值对应的速率 $$v_p = \sqrt{\frac{2kT}{m_0}} = \sqrt{\frac{2RT}{M}}$$ ≈ 1.41 讨论速率分布、碰撞截面
平均速率 ⟨v⟩ 所有分子速率的算术平均值 $$\langle v \rangle = \sqrt{\frac{8kT}{\pi m_0}} = \sqrt{\frac{8RT}{\pi M}}$$ ≈ 1.60 计算平均自由程、碰撞频率

v_p < ⟨v⟩ < v_rms （方均根速率最大，最概然速率最小）

易错：$1$ 三种速率的公式容易混淆 —— v_p 系数为 √2，v_rms 系数为 √3，⟨v⟩ 系数为 √$8/π$。关键记忆：v_rms 来自温度公式（3/2）kT = $1/2$m₀⟨v²⟩，所以 √$3kT/m₀$。$2$ 注意是 v_p < ⟨v⟩ < v_rms 而非其他顺序。$3$ 方均根速率来自 ⟨v²⟩ 开根号，不是先开根号再平均。
温度升高时，麦克斯韦速率分布曲线如何变化？三种特征速率如何变化？
参考答案温度升高 → 曲线峰值右移（向高速方向移动），峰值降低，曲线变扁平（高速分子比例增加）。三种特征速率都增大，增大比例均为 √$T₂/T₁$。

平均自由程与碰撞频率

分子碰撞与平均自由程 重点 页28页29

平均碰撞频率 Z：一个分子在单位时间内与其他分子碰撞的平均次数。数量级约 10⁹ ∼ 10¹⁰ s⁻¹。

考虑其他分子也运动时，平均相对速率为 √2⟨v⟩，碰撞截面为 πd²（d 为分子有效直径 ≈ 10⁻¹⁰ m）：

$$Z = \sqrt{2}\pi d^2 \langle v \rangle n$$

平均自由程 λ：一个分子在连续两次碰撞之间所通过路程的平均值。

$$\lambda = \frac{\langle v \rangle}{Z} = \frac{1}{\sqrt{2}\pi d^2 n}$$

由 p = nkT 得 n = p/$kT$，代入：

$$\lambda = \frac{kT}{\sqrt{2}\pi d^2 p}$$

关键结论：

  • 温度一定时，λ ∝ 1/p（压强越低，平均自由程越大）
  • 压强一定时，λ ∝ T（温度越高，平均自由程越大，因为 n = p/kT 减小）
  • 标准状态下空气分子的 λ ∼ 10⁻⁷ m 量级
易错：λ = ⟨v⟩/Z 中的 ⟨v⟩ 是平均速率（不是方均根速率也不是最概然速率）。Z 表达式中的 √2 因子是因为考虑了分子间的相对运动——用平均相对速率代替平均速率。
保温瓶（暖壶）夹层抽成真空的物理原理是什么？为什么真空度越高保温效果越好？
参考答案抽真空 → 分子数密度 n 降低 → 平均自由程 λ ∝ 1/n 增大 → 分子碰撞频率降低 → 热传导减弱。当 λ 大于夹层间距时，分子几乎不发生碰撞直接从一个壁飞到另一个壁，导热大幅降低。真空度越高，保温效果越好。
