一、矢量基础：描述运动的数学语言 页6页7页8页9页10

kc1.0 Vectors & Units 矢量与单位 必考 页6页7页8页9页10

你用手推一个箱子，说「我用了10N的力」和「我用了10N斜向上30°的力」，效力完全不同。第一个说法是标量，第二个才是矢量——物理量只靠数字远远不够。
标量只有大小，矢量有大小+方向。速度、加速度、力、位矢都是矢量，它们的运算（加法、标乘、矢乘）必须遵守矢量法则。

Why 为何矢量如此重要？

经典力学的核心是运动方程 $$\vec{r} = \vec{r}$t$$，其中位矢 $\vec{r}$ 是矢量。速度 $\vec{v} = d\vec{r}/dt$、加速度 $\vec{a} = d\vec{v}/dt$ 都是矢量。如果不理解矢量的合成与分解，就无法描述曲线的弯折、斜抛、变速——而这些正是第1章的核心。

What 什么是矢量与标量？

  • 标量 Scalar：只有大小（数值+单位），无方向。如质量 $m$、时间 $t$、路程 $s$、温度 $T$。
  • 矢量 Vector：大小+方向。如位矢 $\vec{r}$、位移 $\Delta\vec{r}$、速度 $\vec{v}$、加速度 $\vec{a}$、力 $\vec{F}$。
  • 单位矢量 Unit Vector：大小为1的矢量，仅表方向。如 $\hat{i}, \hat{j}, \hat{k}$ 分别指向 $x,y,z$ 正方向。

How 矢量的运算

  1. 矢量加法：平行四边形法则或三角形法则（头尾相接）。分量相加： $$\vec{A} + \vec{B} = $A_x+B_x$\hat{i} + $A_y+B_y$\hat{j} + $A_z+B_z$\hat{k}$$
  2. 标量积（点乘 / Dot Product）：结果是一个标量 —— 常用于求投影、做功。 $$\vec{A} \cdot \vec{B} = A_x B_x + A_y B_y + A_z B_z = |\vec{A}||\vec{B}|\cos\theta$$ 式中 $\theta$ 是两矢量夹角。
  3. 矢量积（叉乘 / Cross Product）：结果是一个矢量 —— 方向由右手定则确定，大小 $|\vec{A}\times\vec{B}|=|\vec{A}||\vec{B}|\sin\theta$。 $$\vec{A} \times \vec{B} = \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ A_x & A_y & A_z \\ B_x & B_y & B_z \end{vmatrix}$$
易错：点乘结果是标量，叉乘结果是矢量。角速度、力矩的方向由叉乘右手定则确定，不是直观方向。

二、运动的描述：从「质点在哪」到「走多快」 页11页15页16页17页20页21页22页23页24页25页26

kc1.1 Particle & Reference Frame 质点与参考系 必考 页11页15页16

地球公转时可以看作一个「点」，但研究地球自转时就不能。同一个地球，有时是质点，有时不是——这就叫「具体问题具体分析」。
质点是一个只具有质量、没有大小和形状的理想模型。能否看作质点取决于研究问题的尺度——当物体的线度远小于运动尺度时可视为质点。参考系是描述运动的基准，参考系不同则运动描述不同。

What

  • 质点 Particle：忽略大小形状，只保留质量的理想物体。适用条件：物体的几何尺寸 << 运动的空间尺度。
  • 参考系 Reference System：描述物体运动时选作参考的物体（或物体系）+ 一套同步的钟。没有参考系就谈不上运动——运动描述是相对的。
  • 坐标系 Coordinate System：参考系上的量化工具。三维直角坐标系 OXYZ，用 $(x,y,z$\) 表示质点位置。
  • 自然坐标 Natural Coordinate：在已知轨迹上任选固定点 O，沿轨迹规定正方向，弧长 $s$ 即为自然坐标。

判断：能否看作质点？

  • 地球绕太阳公转 --> 是（地球半径 $6.4\times10^6$m << 日地距离 $1.5\times10^{11}$m）
  • 地球自转 --> 否（球上各点位移、速度、加速度方向大小都不同，不可忽略）
  • 火车从北京到上海 --> 是
  • 火车过一座短桥 --> 否
易错：「大物体不是质点，小物体就是质点」是错的。一颗灰尘在研究其内部结构时也不是质点。核心是尺度比较，不是绝对大小。

kc1.2 Position Vector & Displacement 位矢与位移 必考 页17页20

你从教室出发经过食堂到宿舍，走的总路程和直线距离不同。直线距离（从起点直接指向终点）是位移，走过的弯弯绕绕是路程——一个人用导航看的是「剩余位移」，不是已走路程。
位矢从原点指向质点，位移是位矢的变化量（矢量），路程是轨迹的长度（标量）。|Δr| 是位移大小，Δs 是路程，两者一般不等。

What

  • 位矢 Position Vector：从坐标原点 O 指向质点 P 的矢量。 $$\vec{r} = x\hat{i} + y\hat{j} + z\hat{k}$$ 模：$|\vec{r}| = \sqrt{x^2+y^2+z^2}$ 方向余弦：$\cos\alpha=\frac{x}{|\vec{r}|},\ \cos\beta=\frac{y}{|\vec{r}|},\ \cos\gamma=\frac{z}{|\vec{r}|}$，其中 $\alpha,\beta,\gamma$ 分别是 $\vec{r}$ 与 Ox、Oy、Oz 轴的夹角。
  • 运动学方程： $$\vec{r} = \vec{r}$t$, \quad \begin{cases} x = x$t$ \\ y = y$t$ \\ z = z$t$ \end{cases}, \quad s = s$t$$ 三种表示法：位矢法、坐标法、自然法。消去参数 $t$ 可得轨迹方程。
  • 位移 Displacement：一段时间内位矢的增量。 $$\Delta\vec{r} = \vec{r}_B - \vec{r}_A$$ 位移是矢量，大小是 $|\Delta\vec{r}|$（起点到终点的直线距离）。
  • 路程 Path Length：实际轨迹的长度 $\Delta s$，标量，恒 $\ge 0$。
对比项位移 $\Delta\vec{r}$路程 $\Delta s$
矢量/标量矢量标量（恒正）
含义位矢的增量轨迹实际长度
关系$\Delta\vec{r}\le \Delta s$等号仅单向直线运动成立
与路径仅取决于起点和终点取决于实际路径
易错：$|\Delta\vec{r}|$（位移大小）和 $\Delta r$（位矢大小增量，即 $|\vec{r}_B\vec{r}_A|$）是两个不同的量！直线运动沿反向折返时最容易混。

kc1.3 Velocity 速度（平均/瞬时） 必考 页21页22页23

打开手机地图，骑车去上课时有三个关键信息：①当前瞬时速度 15km/h（看码表），②全程平均速度 12km/h（总位移/总时间），③导航显示「平均速率」18km/h（总路程/总时间，因为绕了路）。这三个数不同——你能分清吗？
速度是矢量（位矢对时间的一阶导数），速率是标量（速度的大小）。平均速度 = 位移/时间，平均速率 = 路程/时间。瞬时速度 = 位矢对时间的导数，瞬时速率 = 瞬时速度的大小。

What

  • 平均速度 Average Velocity： $$\vec{v}_{\text{avg}} = \frac{\Delta\vec{r}}{\Delta t} = \frac{\vec{r}$t+\Delta t$-\vec{r}$t$}{\Delta t}$$ 矢量，方向与位移相同。
  • 瞬时速度 Instantaneous Velocity： $$\vec{v} = \lim_{\Delta t\to 0}\frac{\Delta\vec{r}}{\Delta t} = \frac{d\vec{r}}{dt}$$ 方向：轨迹的切线方向（$\Delta t\to 0$ 时 $\Delta\vec{r}$ 无限逼近切线）。
  • 平均速率 Average Speed： $$v_{\text{avg-speed}} = \frac{\Delta s}{\Delta t}$$ 标量，路程/时间。
  • 瞬时速率 Instantaneous Speed： $$v = |\vec{v}| = \frac{ds}{dt} = \sqrt{v_x^2+v_y^2+v_z^2}$$ 瞬时速度的模，标量。

How 直角坐标分量求速度

位矢分量分别对时间求一阶导数即为速度分量： $$\vec{v} = \frac{dx}{dt}\hat{i} + \frac{dy}{dt}\hat{j} + \frac{dz}{dt}\hat{k} = v_x\hat{i} + v_y\hat{j} + v_z\hat{k}$$
对比项平均速度平均速率瞬时速度瞬时速率
定义$\vec{v}_{\text{avg}}=\Delta\vec{r}/\Delta t$$v_{\text{avg}}=\Delta s/\Delta t$$\vec{v}=d\vec{r}/dt$$v=ds/dt=\vec{v}$
矢/标矢量标量矢量标量
大小关系$\vec{v}_{\text{avg}}\le v_{\text{avg}}$——$v=\vec{v}$（瞬时速率=瞬时速度的大小）
易错：平均速度的「大小」不等于平均速率！因为 $|\Delta\vec{r}|\neq\Delta s$。只有单向直线运动时两者才相等。

kc1.4 Acceleration 加速度（平均/瞬时） 必考 页24页25页26

坐电梯启动那一刻你感觉身体「一沉」，匀速运行时毫无感觉，停的那一刻又「一轻」——这三个阶段速度都在变，但变化的「速率」不同。加速度就是描述速度变化快慢的物理量。
加速度是速度对时间的一阶导数，也是位矢对时间的二阶导数。$$\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$$ 加速度大小反映速度变化的快慢，方向反映速度变化的方向。

What

  • 平均加速度： $$\vec{a}_{\text{avg}} = \frac{\Delta\vec{v}}{\Delta t} = \frac{\vec{v}_2 - \vec{v}_1}{\Delta t}$$
  • 瞬时加速度： $$\vec{a} = \lim_{\Delta t\to 0}\frac{\Delta\vec{v}}{\Delta t} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$$

How 加速度的分量与大小

分量： $$a_x = \frac{dv_x}{dt} = \frac{d^2x}{dt^2},\quad a_y = \frac{dv_y}{dt} = \frac{d^2y}{dt^2},\quad a_z = \frac{dv_z}{dt} = \frac{d^2z}{dt^2}$$ 合成： $$\vec{a} = a_x\hat{i} + a_y\hat{j} + a_z\hat{k}$$ 大小： $$a = |\vec{a}| = \sqrt{a_x^2 + a_y^2 + a_z^2}$$ 方向（方向余弦）：$\cos\alpha' = a_x/a,\ \cos\beta' = a_y/a,\ \cos\gamma' = a_z/a$

直线运动总结 页27

沿 x 轴的一维运动退化为： $$x = x$t$,\quad v = \frac{dx}{dt},\quad a = \frac{dv}{dt} = \frac{d^2x}{dt^2}$$
易错：加速度的正负不代表加速或减速！$a$ 与 $v$ 同号 --> 加速（速率增大）；$a$ 与 $v$ 异号 --> 减速（速率减小）。例如 $v=-6\mathrm{m/s},\,a=+2\mathrm{m/s^2}$，虽然 $a>0$ 但因异号，质点实际在减速。

三、圆周与曲线运动：弯弯绕绕里的加速度 页31页32页33页34页35页36页37页38页39页40

kc1.5 Circular & Curvilinear Motion 圆周与曲线运动 必考 页31页32页33页34页35

开车转弯时，即使车速表指针不动（匀速），你也会感到被「甩」向一侧——这说明有加速度！匀速圆周运动中速度大小不变但方向不断改变，这个改变方向的加速度就是法向加速度（向心加速度）。如果加速过弯，还有切向加速度让你速率也变化。
曲线运动的加速度可以分解为切向分量和法向分量：切向加速度 $a_\tau$ 改变速率大小，法向加速度 $a_n$ 改变速度方向。$$a_\tau = \frac{dv}{dt},\quad a_n = \frac{v^2}{R},\quad a = \sqrt{a_\tau^2 + a_n^2}$$

What 切向与法向分解

用自然坐标，速度沿切向： $$\vec{v} = v\hat{\tau} = \frac{ds}{dt}\hat{\tau}$$ 加速度推导（对 $\vec{v}=v\hat{\tau}$ 求导，用到 $d\hat{\tau}/dt = (v/R$\hat{n}\)）： $$\vec{a} = \frac{d\vec{v}}{dt} = \frac{dv}{dt}\hat{\tau} + v\frac{d\hat{\tau}}{dt} = \frac{dv}{dt}\hat{\tau} + \frac{v^2}{R}\hat{n}$$ $$a_\tau = \frac{dv}{dt},\qquad a_n = \frac{v^2}{R}$$ 总加速度：$a = \sqrt{a_\tau^2 + a_n^2}$
对比项切向加速度 $a_\tau$法向加速度 $a_n$
物理意义改变速率的大小改变速度的方向
公式$a_\tau = dv/dt$$a_n = v^2/R$
匀速圆周$a_\tau=0$$a_n=v^2/R=\text{常数}$
变速圆周$a_\tau\neq0$，反映速率变化$a_n=v^2/R$，随速率变
方向沿轨迹切线指向曲率中心（垂直于切线）

What 角量与线量的关系 页35

  • 角速度：$\displaystyle\omega = \frac{d\theta}{dt}$（rad/s）
  • 角加速度：$\displaystyle\alpha = \frac{d\omega}{dt} = \frac{d^2\theta}{dt^2}$（rad/s²）
  • 线速与角速：$v = R\omega$
  • 切向加速度与角加速度：$a_\tau = R\alpha$
  • 法向加速度与角速度：$a_n = \frac{v^2}{R} = v\omega = R\omega^2$

曲率圆视角 — 任意平面曲线运动 页34

任意平面曲线可看作由无数段无穷小圆弧连接而成。曲率圆（密切圆）的半径 $\rho$ 即曲率半径。在任意点，用该点的曲率半径 $\rho$ 替代公式中的 $R$： $$a_n = \frac{v^2}{\rho},\quad a_\tau = \frac{dv}{dt}$$
易错：匀速率圆周运动 $a_\tau=0$，但 $a_n\neq0$，所以加速度不为零！很多同学看到「匀速」就以为加速度为零，这是致命的错误。另外线速度 $v$ 和角速度 $\omega$ 的方向不同：$v$ 沿切线、$\omega$ 沿转轴由右手定则确定。

四、抛体运动：运动叠加原理的经典应用 页41页42页43页44

kc1.6 Projectile Motion 抛体运动 必考 页41页42页43页44

「枪打落猴」：猎人瞄准树上的猴子开枪，子弹射出的同时猴子受惊自由下落——只要在射程内，子弹必然击中猴子，无论仰角和距离如何变化！这就是运动叠加原理的最佳演示。
抛体运动可分解为水平方向的匀速直线运动 + 竖直方向的匀变速运动。核心方法：运动叠加原理。取向上为正，则 $$\vec{v} = $v_0\cos\theta$\hat{i} + $v_0\sin\theta - gt$\hat{j}$$，$$\vec{r} = $v_0 t\cos\theta$\hat{i} + $v_0 t\sin\theta - \frac{1}{2}gt^2$\hat{j}$$

Why 为何用叠加原理？

在经典力学中，一个运动可以看作几个各自独立运动的叠加——水平方向的运动和竖直方向的运动互不影响。这是伽利略首先提出的方法。

What 两种分解方式

  1. 分解一：沿 x 水平 + y 竖直 水平：匀速直线运动 $x = v_0 t\cos\theta$，$v_x=v_0\cos\theta=$ 常数 竖直：匀变速运动 $y = v_0 t\sin\theta - \frac{1}{2}gt^2$，$v_y=v_0\sin\theta-gt$ （已取向上为正，$a_y=-g$）
  2. 分解二：沿初速方向匀速 + 竖直方向自由落体 匀速直线：$\vec{v}_0 t = (v_0\cos\theta\,\hat{i}+v_0\sin\theta\,\hat{j}$t\) 自由落体：$-\frac{1}{2}gt^2\hat{j}$ 叠加：$\vec{r} = \vec{v}_0 t + \frac{1}{2}\vec{g}t^2$

How 关键公式速查

  • 射高：$H = \dfrac{v_0^2\sin^2\theta}{2g}$
  • 射程：$R = \dfrac{v_0^2\sin 2\theta}{g}$（落地高度=发射高度时）
  • 飞行时间（落地高度=发射高度）：$T = \dfrac{2v_0\sin\theta}{g}$
  • 轨迹方程（消去 t）：$y = x\tan\theta - \dfrac{g}{2v_0^2\cos^2\theta}x^2$（抛物线）

「枪打落猴」揭秘

子弹：$y_1 = y_0 - \frac{1}{2}gt^2$（因 $\tan\theta = y_0/x_0$）
猴子：$y_2 = y_0 - \frac{1}{2}gt^2$
--> 任意时刻 $y_1 = y_2$，只要子弹落地前两者始终等高，必然击中。
易错：射程公式 $R=v_0^2\sin2\theta/g$ 仅在发射点和落地点在同一水平面上才成立。若落地点低于/高于发射点，必须从轨迹方程解交点。

五、相对运动与常见力：换个视角看世界 页45页46

kc1.7 Relative Motion & Common Forces 相对运动与常见力 重点 页45页46

你坐在匀速行驶的火车上看窗外，树木和房屋「向后移动」——同一棵树，地面上的人说它静止，你说它向后运动。谁对？都对！这就是相对运动：运动描述因参考系不同而不同。
伽利略变换是低速下参考系间的转换法则。若 K' 系相对 K 系以恒定速度 $\vec{u}$ 运动，则 $\vec{r} = \vec{R} + \vec{r}'$，$\vec{v} = \vec{v}' + \vec{u}$，$\vec{a} = \vec{a}' + \vec{a}_0$。常见力有万有引力、重力、弹力、摩擦力。

What 伽利略变换 页45

设 K' 系相对 K 系以恒定速度 $\vec{u}$ 沿 x 轴运动，且 t=0 时两系原点重合： 坐标变换：$x' = x - ut,\ y' = y,\ z' = z,\ t' = t$ 矢量形式：$\vec{r} = \vec{R} + \vec{r}'$ 速度变换：$\vec{v}_K = \vec{v}_{K'} + \vec{u}$ 即「绝对速度」=「相对速度」+「牵连速度」 加速度变换：$\vec{a}_K = \vec{a}_{K'} + \vec{a}_0$ 若 K' 系相对 K 系匀速直线运动（$\vec{a}_0=0$），则 $\vec{a}_K = \vec{a}_{K'}$——加速度在惯性系间不变。

What 四种常见力 页46

  • 万有引力 Universal Gravitation：存在于任何两个物体之间。$F = G\frac{m_1 m_2}{r^2}$
  • 重力 Weight：地球表面附近物体受到地球的引力——属于万有引力。$G = mg$，$g\approx 9.8\,\mathrm{m/s^2}$
  • 弹性力 Elastic Force：物体接触且变形时产生的恢复力（压力、支撑力、张力、弹簧回复力等）。遵循胡克定律：$F=-kx$
  • 摩擦力 Frictional Force：接触面间有相对运动或相对运动趋势时产生的阻碍力。静摩擦力、动摩擦力（滑动摩擦、滚动摩擦）。
易错：伽利略变换中的速度叠加是矢量叠加（不是代数加减）！比如船横渡河流时，船相对岸的速度 = 船相对水的速度（矢量）+ 水相对岸的速度（矢量），方向不同不能直接数值相加。

Checkpoint 1：位移 vs 路程 vs 位矢增量
质点沿 x 轴运动，运动方程为 $x = 4.5t^2 - 2t^3$（SI）。求第2秒内（t=1到t=2）的：$1$ 位移，$2$ 路程，$3$ 位矢大小的增量。
（提示：先检查 v=0 的时刻看质点是否折返。）
参考答案 $1$ 位移：$\Delta x = x(2$-x$1$ = $18-16$-$4.5-2$ = 2-2.5 = -0.5\,\mathrm{m}\)
$2$ 先求 $v=dx/dt=9t-6t^2$，令 v=0 得 t=0, t=1.5s（在 1455L55 内折返）。
路程 $\Delta s = |x(1.5$-x$1$| + |x$2$-x$1.5$| = |3.375-2.5| + |2-3.375| = 0.875 + 1.375 = 2.25\,\mathrm{m}\)
$3$ 位矢大小增量 $\Delta r = |x(2$x$1$| = 2-2.5 = -0.5\,\mathrm{m}\)
关键结论：位移大小 $|{-0.5}| = 0.5\mathrm{m}$，路程 2.25m，位矢增量 -0.5m——三者不同！路程最大，因为有折返。
Checkpoint 2：切向加速度 vs 法向加速度 vs 总加速度
一质点做半径为 R 的圆周运动，路程 $s = v_0 t - \frac{1}{2}bt^2$（$v_0,b,R$ 为正的常量）。$1$ 求任意时刻的速率 v$t$；$2$ 求切向加速度 $a_\tau$ 和法向加速度 $a_n$；$3$ 何时 $a_\tau = a_n$？并求此时的总加速度大小。
参考答案 $1$ $v = ds/dt = v_0 - bt$
$2$ $a_\tau = dv/dt = -b$（常量，方向与速度相反——匀减速）。$a_n = v^2/R = (v_0-bt$^2/R\)
$3$ 令 $|a_\tau| = a_n$：即 $b = (v_0-bt$^2/R\) --> $v_0-bt = \sqrt{bR}$ --> $t = (v_0-\sqrt{bR}$/b\)
此时 $a = \sqrt{a_\tau^2+a_n^2} = \sqrt{b^2+b^2} = \sqrt{2}b$
关键图示：切向加速度是常数但法向加速度随时间递减（因为速率在减小），总加速度的大小和方向都在变化。

3、弹性力：(Elastic force) The force generated when an object is in

a pair of forces preventing relative movement will be generated between
the contact surfaces 。（静摩擦力、动摩擦力、滑动摩擦力、滚动摩擦
