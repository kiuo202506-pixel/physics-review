# Chapter 4 Special Relativity 狭义相对论 -- 参考答案

### 1. μ 子以 0.998c 的速度从高空飞向地面。在实验室参考系中测得 μ 子的寿命比其静止寿命长，这是因为：

**Answer**: B

**Explanation**: 实验室参考系中 μ 子以高速运动，动钟变慢（时间膨胀），其寿命被放大 γ 倍：Δt = γτ₀。实验测得 μ 子飞行距离远超经典预计，完美验证了时间膨胀。选项 A 说的是事实但时间膨胀才是根本原因；选项 C 错误；选项 D 违反光速不变。

---

### 2. 关于固有时（proper time）Δt₀ 的说法，正确的是：

**Answer**: C

**Explanation**: 固有时 Δt₀ 的定义：两事件在同一地点发生（Δx' = 0）的那个参考系中测得的时间间隔。γ ≥ 1 所以 Δt = γΔt₀ ≥ Δt₀，固有时最短而非最长——A 错误。固有时不需要事件同时——C 错误。D 没有物理意义。

---

### 3. 飞船以 0.8c 飞向距离地球 8 光年的星球。下列哪个说法是正确的？

**Answer**: A

**Explanation**: 地球测旅程时间 Δt = 8 ly/0.8c = 10 年（非固有时，因为出发和到达不在同一地点）。飞船系中这是同一地点的两个事件（飞船自身经历），故飞船上测的时间是固有时：Δt₀ = Δt/γ = 10/1.667 ≈ 6.0 年，小于地球测的 10 年。所以 A 正确（地球看，飞船钟走得更慢，测量的间隔更短）。B：运动方向长度收缩；C：地球测 10 年不是 6.4 年；D：飞船看到距离收缩。

---

### 4. 爱因斯坦火车思想实验中，车厢中点 O 发出闪光。地面观察者看到光先到达车尾后到达车头。这说明了什么？

**Answer**: D

**Explanation**: 爱因斯坦火车实验直接证明了同时性的相对性。车上观察者：光同时到达两端（因等距、光速同）。地面观察者：光速仍是 c，但车尾迎着光走（更近），车头远离光走（更远），所以光先到尾后到头。关键在于光速在地面系中也是 c（光速不变）——选项 A 和 D 都违反了光速不变原理。

---

### 5. 一根 1 m 长的杆静止在 S' 系中，与 x' 轴夹角 30°。S' 相对 S 以速度 u 沿 x 轴运动。S 系中杆与 x 轴夹角变为 45°。则杆在 S 系中的长度最接近：

**Answer**: C

**Explanation**: y 方向无收缩：Δy = Δy'。tan30° = Δy'/Δx' → Δx' = Δy'/tan30°。tan45° = Δy/Δx = 1 → Δx = Δy = Δy'。故 Δx/Δx' = tan30°/1 = 0.577 = \sqrt{1−u²/c²} → u = 0.816c。杆长 L = \sqrt{Δx² + Δy²} = \sqrt{Δy'² + Δy'²} = Δy'·\sqrt{2}。又 L₀ = \sqrt{Δx'²+Δy'²} = 1 m，而 Δx' = Δy'/tan30° → 代回得 L = L₀·(sin30°/sin45°) = 1×(0.5/0.707) = 0.707 m。长度变短了！

---

### 6. 关于同时性的相对性，下列说法正确的是：

**Answer**: B

**Explanation**: 由洛伦兹变换：Δt' = γ(Δt - vΔx/c²)。S 系中同时 Δt=0 但不同地 Δx≠0 → Δt' = -γvΔx/c² ≠ 0，S'系中一定不同时。

如果两事件在 S 系中同地同时（Δx=0, Δt=0），则 Δt' = 0 ——在所有惯性系中都是同地同时的。

同时性的相对性是相对论时空观的本质特征，不是高速近似——即使 v 很小，只要 v≠0 且 Δx≠0（空间距离足够大），Δt' 就可能显著不为零。

> Pitfall: D 选项是典型错误——同时性的相对性是精确结果，不存在「低速时同时性恢复绝对」这种说法。只是低速下不同参考系的时差很小、不易察觉而已。

---

### 7. 在狭义相对论中，真空中光速在所有惯性系中都等于 c，与光源和观察者的相对运动无关。

**Answer**: False/Incorrect

**Explanation**: 这正是光速不变原理的内容——狭义相对论的第二条基本假设。Michelson-Morley 实验证明了这一点。

---

### 8. 在经典力学（伽利略变换）中，同时性是绝对的，即在一个惯性系中同时发生的两个事件，在所有其他惯性系中也同时发生。

**Answer**: False/Incorrect

**Explanation**: 经典绝对时空观中，t'=t 保证了同时性在所有惯性系中的不变性。相对论推翻了这一点——同时性是相对的。

---

### 9. 一根尺子以高速沿长度方向运动时，地面观察者测得其长度比静止时测的长度更长。

**Answer**: True/Correct

**Explanation**: 动尺变短！运动物体在运动方向上长度收缩：L = L₀/γ &lt; L₀。固有长度（静止时测的）是最长的，运动测量值总是更短。

---

### 10. 一艘飞船以高速飞离地球。从飞船上的宇航员看来，地球上的时钟也变慢了（地球上的时间膨胀了）。这是否意味着双方看到的效应是对称的、因而存在矛盾？实际上不矛盾——因为「同时」的定义在飞船和地球参考系中不同。

**Answer**: False/Incorrect

**Explanation**: 这被称为「双生子佯谬」（实际不是佯谬）。时间膨胀是相对的——每个惯性系中的观察者都认为相对于自己运动的钟变慢。这看起来矛盾，但因为同时性是相对的，双方对「什么时候」做比较的定义本身就不同，所以逻辑自洽。双生子佯谬中，如果一方经历了加速（转弯返回），不对称的加速度打破了对称性。

---

### 11. 一艘飞船以 0.8c 的速度相对地球飞行。飞船上的时钟测得某物理过程持续了 6.0 s。在地球上观测，该过程持续多长时间？（γ = 5/3）

**Answer**: See explanation below

**Explanation**: 飞船系中同一地点的事件，飞船钟测得固有时 Δt₀ = 6.0 s。地球系是相对运动的参考系，测得 Δt = γΔt₀ = (5/3)×6.0 = 10.0 s，比固有时长。这体现「动钟变慢」——地球观察者看到飞船上的钟走得慢。

### 12. 一根杆静止时长度为 2.0 m。当它以 0.6c 的速度沿杆长方向运动时，地面上的观察者测得杆的长度为多少？

**Answer**: See explanation below

**Explanation**: 固有长度 L₀ = 2.0 m 是杆静止时的长度。运动时 L = L₀\sqrt{1-v^2/c^2} = 2.0×0.8 = 1.6 m，运动方向缩短了 20%。注意长度收缩只在运动方向，垂直方向不变。

### 13. 一艘飞船以 0.6c 相对地球飞行，飞船上向前发射一个相对于飞船速度为 0.6c 的探测器。求探测器相对地球的速度（以 c 为单位，保留三位小数）。

**Answer**: See explanation below

**Explanation**: 经典相加 0.6c+0.6c = 1.2c 是错的！相对论速度叠加的分母修正项 (1+uv'/c²) 保证了结果 0.882c &lt; c，不超光速。这是光速作为速度上限的体现。

### 14. 在核裂变中，1 mol ²³⁵U 反应前后总静质量亏损 0.215 g。求释放的能量（用科学计数法表示，单位 J，保留 3 位有效数字，格式：1.94×10¹³）。

**Answer**: See explanation below

**Explanation**: 核反应释放的能量来自静质量亏损。ΔE = Δm₀c²。仅 0.215 g 的质量亏损就放出约 1.94×10¹³ J 能量，相当于 625 吨无烟煤燃烧的热量！这展示了 c² 因子的巨大放大效应。

### 15. 一个静止质量为 m₀ 的粒子以 0.6c 运动，其动能是多少？用 m₀c² 的倍数表示。

**Answer**: See explanation below

**Explanation**: 相对论动能 E_k = (γ-1)m₀c²。当 v = 0.6c 时 γ = 1.25，E_k = 0.25m₀c²。如果用经典公式 ½m₀(0.6c)² = 0.18m₀c²，明显低估了。

### 16. S' 系相对 S 系以 0.6c 沿 +x 方向运动。在 S 系中，事件 A 的坐标为 x_A = 0，t_A = 0；事件 B 的坐标为 x_B = 3×10^8 m，t_B = 1.0 s。求 S...

**Answer**: See explanation below

**Explanation**: 用洛伦兹变换 Δt' = γ(Δt − uΔx/c²)。在 S 系中 Δt = 1.0 s，但 S' 系测到的时间间隔只有 0.5 s——因为在 S' 系中两事件发生在不同位置（Δx' ≠ 0），所以 Δt' 不是固有时。时空混合项 uΔx/c² 修正了时间间隔。

### 17. γ = 1.25 时，粒子动能与静能之比 E_k/E₀ 是多少？

**Answer**: See explanation below

**Explanation**: 动能与静能之比 E_k/E₀ = γ−1，这是一个简洁而重要的关系。γ = 1.25 对应 v = 0.6c，此时动能只有静能的 25%。当 v → c 时 γ → ∞，动能远超静能。

### 18. 静止 μ 子的平均寿命 τ₀ = 2.2×10⁻⁶ s。来自高空的宇宙线 μ 子以 v = 0.998c 高速飞向地面。问：(1) 在实验室系中测得 μ 子的平均寿命是多少？(2) 它能飞行的平均距离...

**Answer**: See explanation below

**Explanation**: (1) 固有时间 τ₀（在 μ 子静止系中测得的寿命）。实验室系中：τ = γτ₀。γ = 1/√(1-0.998²) = 1/√(1-0.996004) = 1/√0.003996 = 1/0.06321 ≈ 15.8。τ = 15.8 × 2.2×10⁻⁶ = 3.48×10⁻⁵ s。

(2) 飞行距离：L = v·τ = 0.998×3×10⁸×3.48×10⁻⁵ = 1.04×10⁴ m ≈ 10.4 km。

这是 μ 子地面实验的经典验证——μ 子诞生于约 10 km 高空，若没有时间膨胀效应，它只能飞行 cτ₀ = 660 m 就到不了地面。

> Pitfall: 固有时间 τ₀ 是在 μ 子自身参考系（它认为自己是静止的）中测得的寿命。实验室看 μ 子运动 → 钟变慢 → 寿命延长。不要弄反方向：如果误把实验室时间当固有时，τ = τ₀/γ，μ 子寿命反而变短，那就飞不到地面了——与实验事实矛盾。

### 19. 一根米尺在 S'系中与 x' 轴成 30° 角。S'系相对 S 系沿 x 轴以速度 v 运动。在 S 系中测得该尺与 x 轴成 45° 角。求相对速度 v。

**Answer**: See explanation below

**Explanation**: 在 S'系（固有系）：L₀ₓ = L₀ cos30° = 1 × √3/2 = √3/2 m；L₀_y = L₀ sin30° = 1 × 1/2 = 0.5 m。

在 S 系：长度收缩仅发生在 x 方向（运动方向）：Lₓ = L₀ₓ√(1-v²/c²)；L_y = L₀_y = 0.5 m（y 方向不收缩）。

S 系中角度：tan45° = L_y/Lₓ = 1 → L_y = Lₓ。

所以 0.5 = (√3/2)·√(1-v²/c²) → √(1-v²/c²) = 0.5/(√3/2) = 1/√3。

1 - v²/c² = 1/3 → v²/c² = 2/3 → v = c·√(2/3) ≈ 0.816c。

> Pitfall: 长度收缩只在运动方向发生！垂直于运动方向的长度不变。此题是利用「不收缩的 y 方向」作为桥梁来求「收缩的 x 方向」的比例，从而反解 v。

### 20. 一个电子的静质量 m₀ = 9.11×10⁻³¹ kg。求：(1) 电子以 v = 0.99c 运动时的动能；(2) 若该动能由经典公式 ½mv² 计算，误差有多大？

**Answer**: See explanation below

**Explanation**: (1) γ = 1/√(1-0.99²) = 1/√0.0199 = 1/0.1411 ≈ 7.09。相对论动能：E_k = (γ-1)m₀c² = 6.09 × 9.11×10⁻³¹ × (3×10⁸)² = 6.09 × 8.20×10⁻¹⁴ = 4.99×10⁻¹³ J ≈ 3.12 MeV。

(2) 经典公式：E_k_classical = ½m₀v² = 0.5 × 9.11×10⁻³¹ × (0.99×3×10⁸)² = 0.5 × 9.11×10⁻³¹ × 8.82×10¹⁶ = 4.02×10⁻¹⁴ J。

相对论结果 / 经典结果 ≈ 12.4 倍。经典公式严重低估！

> Pitfall: 当 v > 0.1c 时，经典动能 ½mv² 的误差就不可忽略。v = 0.99c 时误差超过 10 倍。相对论动能必须用 (γ-1)m₀c²。注意：不是 mc² 减去 ½mv²，而是 mc² - m₀c²。
