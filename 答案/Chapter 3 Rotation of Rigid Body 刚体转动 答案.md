# Chapter 3 Rotation of Rigid Body 刚体转动 -- 参考答案

### 1. 关于刚体模型，下列说法正确的是：

**Answer**: D

**Explanation**: ①题型判断：概念辨析题。②刚体定义：理想的力学模型，受力后形状和大小不变，体内任意两点间距恒定。③逐项：第一项正确；刚体与质量大小无关（小物体也可以是刚体）；受力不变形是刚体定义的核心；只有平动时各点速度才相同，转动时不同。④依据：刚体定义。⑤易错点：把刚体误认为「很硬的物体」——硬不等于刚体，刚体是完全无形变的理想模型。

> Pitfall: 混淆「刚体」与「质点」：质点无形状大小，刚体有形状大小但不变形。

### 2. 关于转动惯量，下列说法正确的是：

**Answer**: B

**Explanation**: ①题型判断：概念理解题。②转动惯量取决于质量、质量分布和转轴位置三因素。③平行轴定理必须以质心轴 $J_C$ 为基准；④质量大不一定 $J$ 大（$r^2$ 的权重可能使小质量大半径物体 $J$ 更大）。⑤$J$ 单位 kg·m²，力矩单位 N·m = kg·m²/s² — 量纲不同。逐项：第一项错（依赖转轴）；第二项正确；第三项错（与分布有关）；第四项错（单位不同）。⑥易错：把 $J$ 当成纯质量属性。

> Pitfall: 转动惯量单位是 kg·m² 而非 N·m；平行轴定理必须从质心轴出发。

---

### 3. 关于角动量守恒定律，下列说法正确的是：

**Answer**: A

**Explanation**: ①题型判断：守恒条件辨析题。②角动量守恒条件：合外力矩为零（而非合外力为零）。③合外力为零但各力作用线不共点时力矩未必为零；合外力矩为零但各力平行时合外力也未必为零——两个条件独立。④$J\omega$ 守恒时若 $J$ 变化则 $\omega$ 也变。⑤内力矩成对出现、之和为零，不能改变总角动量。逐项：第一项错（需外力矩为零）；第二项正确；第三项错（$J$ 可变）；第四项错（内力矩之和为零）。⑥易错：混淆合外力为零与合外力矩为零。

> Pitfall: 角动量守恒条件是合外力矩为零，不是合外力为零——两条件相互独立。

### 4. 关于力矩的定义 $\vec{M} = \vec{r} \times \vec{F}$，以下理解正确的是：

**Answer**: C

**Explanation**: ①题型判断：概念理解题。②$M = Fr\sin\theta$，只在 $\theta=90°$ 时 $M=Fr$。③力臂是转轴到力作用线的垂直距离（不是到作用点）。④力平行于 $\vec{r}$ 时 $\sin\theta=0$ → $M=0$，而沿 $\vec{r}$ 方向的分力无转动效果。⑤力矩方向由右手法则确定，垂直于 $\vec{r}$ 和 $\vec{F}$ 所在平面。逐项：第一项错（需 $\theta=90°$）；第二项错（力臂是到力作用线的垂直距离）；第三项正确（若「平行于转轴」理解为力沿 $\vec{r}$ 方向即力通过轴心，则力矩为零）；第四项错（力矩方向垂直 $\vec{r}$ 和 $\vec{F}$ 平面）。⑥易错：力臂是垂距而非连线长。

> Pitfall: 力臂是转轴到力作用线的垂直距离，不是到力作用点的距离。

### 5. 一人站在可自由旋转的转台上，两手各握重物伸开。当他把双手收回胸前时（忽略摩擦），以下说法正确的是：

**Answer**: B

**Explanation**: 系统合外力矩为零 → 角动量守恒 Jω = 常数。双手收回 → 质量分布靠近转轴 → J 减小 → ω 增大。转动动能 Ek = ½Jω² = L²/(2J)，J 减小意味着 Ek 增大——人的肌肉做了功，将生化能转化为转动动能。

> Pitfall: 角动量守恒 ≠ 动能守恒！花样滑冰运动员收臂加速时角动量不变但转动动能增大——增大的动能来自肌肉做功。这是高频概念辨析题。

### 6. A、B 为两个完全相同的定滑轮（半径 R，转动惯量 J）。A 滑轮挂一质量为 M 的物体，B 滑轮受拉力 F = Mg。不计绳重和摩擦，关于两滑轮角加速度的大小关系：

**Answer**: C

**Explanation**: B 滑轮：拉力 F = Mg 直接产生力矩 TR = MgR，所以 MgR = Jβ_B。

A 滑轮：挂的物体有向下的加速度 a，绳中张力 T &lt; Mg（因为 Mg - T = Ma > 0）。由 T·R = Jβ_A 且 a = Rβ_A，联立得 β_A = MgR/(J + MR²)，小于 β_B = MgR/J。

物理直觉：挂物体时，物体的重力一部分用于加速物体自身、一部分通过绳张力传给滑轮；直接拉力 F=Mg 时全部力都给了滑轮。

> Pitfall: 千万不要把「挂质量 M 的物体」和「施加力 F=Mg」等效！前者绳张力 T &lt; Mg，后者直接作用力 = Mg。这是一个高频易错概念判断题。

### 7. 刚体的平面平行运动可以分解为质心平动和绕质心转动的叠加。

**Answer**: False/Incorrect

**Explanation**: ①题型判断：判断题。②任何复杂刚体运动均 = 质心平动 + 绕质心转动。③平面平行运动（如车轮滚动）是这一分解的典型代表。④依据：刚体运动学定理（Chasles 定理）。⑤易错点：混用绕质心转动和绕任意点转动——只有绕质心的转动 + 质心平动才是标准分解。

> Pitfall: 质心平动 + 绕质心转动是标准分解，而非绕任意点。

---

### 8. 一质点做匀速直线运动时，它对空间中任意一点的角动量都为零。

**Answer**: True/Correct

**Explanation**: ①题型判断：判断题。②质点做匀速直线运动时，对不在其运动直线上的点，$\vec{r}$ 与 $\vec{v}$ 不平行 → $L=rmv\sin\theta \neq 0$。③只有质点运动方向恰好通过参考点（$\vec{r}\parallel\vec{v}$）时角动量才为零。④例如：高速公路上直行的汽车对路旁电线杆有角动量。⑤易错：把直线运动等同于角动量为零——匀速直线运动的角动量对大多数参考点都是非零常量。

> Pitfall: 直线运动的质点对大多数点都有角动量（$\vec{r}$ 和 $\vec{v}$ 一般不平行）。

---

### 9. 纯滚动中静摩擦力不做功，因此包含纯滚动的系统机械能守恒。

**Answer**: False/Incorrect

**Explanation**: ①题型判断：判断题。②纯滚动时接触点瞬时速度为零 → $dA = f_s \cdot 0 \cdot dt = 0$，静摩擦力不做功。③若系统只有保守力（重力）做功，则机械能守恒。④注意：前提是只有保守力做功——如果还有手推、刹车等外力，机械能就不守恒。⑤题干在此语境下正确——纯滚动 + 保守力系统机械能守恒。⑥易错：把「有摩擦力」等同于「机械能不守恒」——滑动摩擦做负功导致机械能损失，但纯滚动的静摩擦不做功。

> Pitfall: 纯滚动中静摩擦力存在但不做功（作用点瞬时速度为零），与滑动摩擦做负功截然不同。

### 10. 一飞轮以 900 r/min 的转速旋转。制动后均匀减速，经 10 s 停止。求：(1) 角加速度；(2) 制动过程中飞轮转过的圈数。

**Answer**: See explanation below

**Explanation**: ①题型判断：匀变速转动计算。②角速度换算 rpm→rad/s 是关键第一步（因子 $2\pi/60$）。③用无 t 公式 $\omega^2 - \omega_0^2 = 2\alpha\Delta\theta$ 求角位移最便捷。④转数 $N = \Delta\theta / (2\pi)$，不是 $\Delta\theta$ 本身。⑤易错：忘记转数与弧度的换算；单位不统一。

> Pitfall: rpm换算 rad/s 时因子是 $\frac{2\pi}{60} \approx 0.1047$，不是 $\frac{\pi}{60}$；转数 $N = \theta / (2\pi)$，差 $2\pi$ 倍。

### 11. 一均匀圆盘（质量 m=5 kg，半径 R=0.3 m）可绕过中心且垂直盘面的轴转动。在盘边缘沿切线方向施加恒力 F=10 N。求：(1) 圆盘的转动惯量；(2) 力矩大小；(3) 角加速度。

**Answer**: See explanation below

**Explanation**: ①题型判断：转动定律 $M=J\alpha$ 直接应用。②圆盘绕中心垂直轴 $J=\frac{1}{2}mR^2$ 是必背公式。③力沿切线方向时 $\theta=90°$，$M=FR$。④$\alpha = M/J$ 与 $a = F/m$ 完全平行——只需变量替换。⑤易错：忘记圆盘的 $J$ 公式是 $\frac{1}{2}mR^2$ 不是 $mR^2$；混淆力矩方向。

> Pitfall: 圆盘 $J=\frac{1}{2}mR^2$ 而非 $mR^2$（圆环才是 $mR^2$）；力不垂直于位矢时需乘 $\sin\theta$。

### 12. 一质量为 M=4 kg、半径为 R=0.2 m 的均匀圆盘定滑轮，绕以轻绳，绳一端悬挂质量 m=2 kg 的物体。系统从静止释放，忽略轴摩擦。求：(1) 物体的加速度 a；(2) 绳中张力 T。

**Answer**: See explanation below

**Explanation**: ①题型判断：定滑轮+重物经典综合题。②关键步骤：(a) 质点用 $F=ma$；(b) 滑轮用 $M=J\alpha$；(c) 约束 $a=R\alpha$ 连接两套方程。③注意有质量的滑轮两端张力不等——$T$ 同时出现在两个方程中，联立消去。④若 M=0（轻滑轮），$a=g$、$T=0$。⑤易错：忘了约束方程 $a=R\alpha$；把滑轮当质点列 $F=ma$。

> Pitfall: 有质量滑轮的绳两端张力不相等；必须写约束方程 $a=R\alpha$。

### 13. 一根长 L=1 m、质量 m=2 kg 的均匀细杆，一端装有一个质量 M=1 kg、半径 R=0.1 m 的均匀圆盘（盘心在杆端处）。求整个系统对过杆另一端且垂直于杆的轴的转动惯量。

**Answer**: See explanation below

**Explanation**: ①题型判断：组合体 + 平行轴定理。②杆对端点轴 $J=\frac{1}{3}mL^2$ 直接使用。③盘对过杆端点的轴需用平行轴定理：$J = J_C + Md^2$，其中 $d$ 是盘心到转轴的距离（此处为杆长 L）。④组合体转动惯量 = 各部分对同一轴的 $J$ 之和。⑤易错：忘记对盘用平行轴定理，直接用 $\frac{1}{2}MR^2$ 而不加 $Md^2$。

> Pitfall: 组合体中远离轴的部件必须用平行轴定理：$J=J_C+Md^2$ 中的 $d$ 是部件质心到转轴的距离。

### 14. 花样滑冰运动员双臂伸展时转动惯量为 $J_1=4.0\,\text{kg·m}^2$，角速度 $\omega_1=3.0\,\text{rad/s}$。收臂后转动惯量变为 $J_2=1.5\,\tex...

### 15. 一根长 $l=1.0\,\text{m}$、质量 $M=3.0\,\text{kg}$ 的均匀细杆，上端挂在光滑水平轴 O 上可自由转动。质量为 $m=0.05\,\text{kg}$ 的子弹以水平速...

**Answer**: See explanation below

**Explanation**: ①题型判断：子弹打木棒——角动量守恒 + 完全非弹性碰撞。②不能用动量守恒！悬挂点 O 的约束反力是外力，动量不守恒。③对 O 轴角动量守恒：合外力矩为零（重力沿铅直方向，力臂在水平方向分量为零）。④碰撞后 $J_{总}$ 包含杆和嵌在其中的子弹。⑤易错：试图用动量守恒；子弹的角动量漏了乘 $l$；遗漏子弹嵌入后也要贡献 $J$。

> Pitfall: 「子弹打悬挂棒」不能用动量守恒（有约束反力），必须用对悬挂点的角动量守恒。

### 16. 一均匀实心球（$J_c=\frac{2}{5}mR^2$）从倾角 $\theta=30°$ 的斜面顶端由静止纯滚动下滑。求：(1) 球的质心加速度 $a_c$；(2) 若球的质量 m=1 kg，斜面高...

**Answer**: See explanation below

**Explanation**: ①题型判断：纯滚动综合题（动力学+能量法）。②动力学法：质心方程+转动方程+纯滚动约束，三式联立。③能量法更简洁——纯滚动静摩擦不做功，机械能守恒。④有效惯性因子：对实心球为 $1+J_c/(mR^2)=1+2/5=7/5$，$a_c = g\sin\theta/(7/5) = 5g\sin\theta/7$。⑤易错：忘了转动动能；用 $v=\sqrt{2gh}$（仅滑动的公式）。

> Pitfall: 纯滚动总动能 = 平动动能 + 转动动能；$a_c \neq g\sin\theta$（转动惯量减小了加速度）。

### 17. 一根均匀细杆长 L=1 m、质量 m=2 kg，一端固定于光滑水平轴可自由转动。初始杆水平静止，然后释放。求杆转到竖直位置时的：(1) 角速度 $\omega$；(2) 质心速度。

**Answer**: See explanation below

**Explanation**: ①题型判断：刚体机械能守恒。②刚体重力势能变化 = $mgh_c$（质心高度变化）。③初始水平（$\omega_0=0$）→ 末态竖直：$mgL/2 = \frac{1}{2}J\omega^2$。④$J$ 对端点轴 = $\frac{1}{3}mL^2$。⑤得 $\omega = \sqrt{3g/L}$。⑥易错：势能参考高度取错（应用质心高度而非端点）；用 $J=\frac{1}{12}mL^2$ 而不检查轴的位置。

> Pitfall: 刚体重力势能是 $mgh_c$（质心高度）；端点轴的 $J=\frac{1}{3}mL^2$，不是 $\frac{1}{12}mL^2$。

**Answer**: See explanation below

**Explanation**: ①题型判断：人在转盘上走——角动量守恒 + 相对运动。②初始静止 → 总角动量始终为零。③关键难点：地面参考系中角速度与相对角速度的关系 $\omega_{\text{人地}} = \omega_{\text{人盘}} + \omega_{\text{盘地}}$。④人走一圈的条件是 $\theta_{\text{人盘}} = 2\pi$ 而非 $\theta_{\text{人地}} = 2\pi$。⑤更简便的解法：用角度替代角速度积分——$J_{\text{人}}\theta_{\text{人地}} + J_{\text{盘}}\theta_{\text{盘地}} = 0$，且 $\theta_{\text{人地}} = \theta_{\text{人盘}} + \theta_{\text{盘地}} = 2\pi + \theta_{\text{盘地}}$。联立得 $\theta_{\text{盘地}} = -\frac{2m}{2m+M}\cdot 2\pi = -\frac{120}{125}\times 2\pi \approx -6.03\,\text{rad}$。

### 19. 长为 L、质量为 M 的均匀细杆可绕过其一端 O 的竖直轴自由转动，转动惯量 J = ML²/3。质量为 m、速率为 v₀ 的子弹水平射入杆的另一端并留在杆内。求：(1) 射入后系统的角速度 ω；(2...

**Answer**: See explanation below

**Explanation**: (1) 子弹射入瞬间，外力矩为零（O 点无力矩，重力对 O 有力矩但在竖直面内不影响水平转动），角动量守恒：mv₀L = (ML²/3 + mL²)ω → ω = mv₀/[(M/3 + m)L]。

### 20. 匀质圆盘（质量 m、半径 R）在水平粗糙桌面上绕中心轴以角速度 ω₀ 旋转。桌面摩擦系数为 μ。（圆盘 J = mR²/2）求圆盘停止前转过的圈数。

**Answer**: See explanation below

**Explanation**: 面密度 σ = m/(πR²)。取半径 r 处宽 dr 的圆环：质量 dm = σ·2πr·dr，所受摩擦力 df = μ·dm·g = 2πμσg·r·dr，摩擦力矩 dM = r·df = 2πμσg·r²·dr。

总阻力矩 M = ∫₀^R 2πμσg·r²dr = (2/3)πμσgR³ = (2/3)μmgR。

β = M/J = [(2/3)μmgR]/[mR²/2] = 4μg/(3R)。

0 - ω₀² = 2(-β)·θ → θ = 3Rω₀²/(8μg)，n = θ/(2π) = 3Rω₀²/(16πμg)。

> Pitfall: 圆盘摩擦问题摩擦力矩需要积分——因为不同半径处力臂不同。直接写 M = μmgR 是错的（这是力乘半径，但摩擦力非均布于边缘）。正确做法是取微元积分。
