# Chapter 5 Mechanical Vibration 机械振动 -- 参考答案

### 1. 在简谐振动中，速度 v 与位移 x 的相位差为：

**Answer**: A

**Explanation**: x = Acos(ωt+φ)，v = -ωAsin(ωt+φ) = ωAcos(ωt+φ+π/2)，速度相位领先位移 π/2。

---

### 2. 弹簧振子，t = 0 时 x₀ = A/2，且向 x 轴负方向运动（v₀ &lt; 0），则初相位 φ 为：

**Answer**: C

**Explanation**: cosφ = x₀/A = 0.5 → φ = ±π/3。v₀ = -ωAsinφ &lt; 0 → sinφ &gt; 0 → φ 在一二象限 → φ = π/3。

---

### 3. 关于受迫振动和共振，下列说法正确的是：

**Answer**: D

**Explanation**: 稳态受迫振动频率 = 策动力频率（不等于固有频率）。共振时振幅取有限最大值（受阻尼限制），阻尼越小共振峰越尖锐。

---

### 4. 两个互相垂直的同频率 SHM：x = Acos(ωt)，y = Acos(ωt + π/2)。合振动的轨迹是：

**Answer**: B

**Explanation**: Δφ = π/2，A₁ = A₂ = A，轨迹为圆 x² + y² = A²。随 t 增大，x 从 A→0→-A→0→A，y 从 0→-A→0→A→0，质点沿顺时针方向运动。

---

### 5. 两个同方向同频率的 SHM：x₁=0.05cos(10t+π/3) m，x₂=0.05cos(10t-π/3) m。合振动的振幅为：

**Answer**: A

**Explanation**: 用旋转矢量法：Δφ = π/3-(-π/3) = 2π/3。A² = A₁²+A₂²+2A₁A₂cosΔφ = 0.05²+0.05²+2×0.05×0.05×cos(2π/3) = 0.0025+0.0025-0.0025 = 0.0025。A=0.05 m。
物理直觉：两个等幅振动相位差 2π/3(120°)，旋转矢量几乎对消。若相位差为 π 则 A=0(完全对消)；若为 0 则 A=0.10 m(同相叠加)。

---

### 6. 简谐振动中，加速度与位移成正比且方向相反，即 a = -ω²x。

**Answer**: False/Incorrect

**Explanation**: a = -ω²x 是 SHM 的动力学判据，从 a = d²x/dt² = -ω²Acos(ωt+φ) 直接得到。

---

### 7. 弹簧振子的周期 T = 2π√(m/k)，因此质量 m 越大，周期越长。

**Answer**: False/Incorrect

**Explanation**: T = 2π/ω = 2π√(m/k)，T ∝ √m，质量越大周期越长——惯性大，振动慢。

---

### 8. 受迫振动达到稳态后，振动频率等于系统的固有频率。

**Answer**: False/Incorrect

**Explanation**: 错误。稳态受迫振动的频率 = 策动力的频率（不是固有频率）。固有频率只在共振（策动力频率≈固有频率）时才近似相等。阻尼越小、共振峰越尖锐。

---

### 9. 两个同方向同频率的 SHM：x₁ = 0.03cos(ωt)，x₂ = 0.04cos(ωt + π)。则合振动的振幅为 ______ m。

**Answer**: -1

**Explanation**: Δφ = π（反相），合振幅 A = |A₁ - A₂| = 0.01 m。

---

### 10. 在旋转矢量表示法中，矢量 A→ 以角速度 ω 逆时针旋转，其在 x 轴上的投影为 x = A______。

**Answer**: -1

**Explanation**: 旋转矢量 A→ 以 ω 逆时针旋转，t 时刻与 x 轴夹角为 ωt+φ，投影 x = Acos(ωt+φ)。

---

### 11. 从牛顿第二定律出发，推导弹簧振子简谐振动的微分方程，并指出角频率 ω 与系统参数的关系。

**Answer**: See explanation below

**Explanation**: 从 F = -kx = ma 出发，令 ω² = k/m，得到 d²x/dt² + ω²x = 0。ω = √(k/m)。

### 13. 弹簧振子，m = 0.2 kg，k = 80 N/m，振幅 A = 0.05 m。求位移 x = 0.03 m 时系统的动能。

### 14. 两个同方向的简谐振动，频率分别为 f₁ = 256 Hz，f₂ = 260 Hz。若它们同时作用于同一质点，求拍频（单位：Hz）。

### 15. 一质点做 SHM，周期 T = 2 s，t = 0 时位移 x₀ = -0.02 m 且速度 v₀ = 0。以平衡位置为原点，求该质点的运动方程（用 cos 表示）。

**Answer**: See explanation below

**Explanation**: v₀ = 0 说明质点正好在振幅位置，A = |x₀| = 0.02 m，φ = π。方程为 x = 0.02cos(πt + π)。

### 16. 某振动系统固有角频率 ω₀ = 20 rad/s，阻尼因子 β = 2 s⁻¹。求该系统发生位移共振时策动力的角频率 ω_r（单位：rad/s，保留根号或保留整数）。

**Answer**: See explanation below

**Explanation**: 位移共振频率 ω_r = √(ω₀² - 2β²) = √(400-8) = √392 ≈ 19.8 rad/s，略小于固有频率。
