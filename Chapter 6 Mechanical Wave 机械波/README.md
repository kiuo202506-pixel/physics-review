# Chapter 6 Mechanical Wave 机械波

## 核心公式速查

| Concept | Formula | Notes/Conditions |
|---------|---------|------------------|
| Wave Speed-Wavelength-Frequency | `u = lambda*f = omega/k` | k = 2*pi/lambda (wave number) |
| Wave Function (+x) | `y = A*cos[omega*(t-x/u) + phi]` | = A*cos(omega*t - k*x + phi), phase lags with x |
| Phase Difference (spatial) | `Delta_phi = 2*pi*Delta_x/lambda` | Path difference -> phase difference |
| Wave Energy | `KE = PE (sync, unlike spring oscillator)` | Energy density w = rho*A^2*omega^2*sin^2(...) |
| Intensity (Energy Flux) | `I = rho*u*A^2*omega^2/2` | I proportional to A^2 |
| Constructive Interference | `Delta_phi = +/-2k*pi OR delta = +/-k*lambda` | In phase, max amplitude |
| Destructive Interference | `Delta_phi = +/-(2k+1)*pi OR delta = +/-(2k+1)*lambda/2` | Out of phase, min amplitude |
| Standing Wave (fixed end) | `y = 2A*sin(k*x)*cos(omega*t)` | Node at x=0. Half-wave loss! |
| Nodes & Antinodes | `Nodes: x = k*lambda/2, Antinodes: x = (2k+1)*lambda/4` | Adjacent nodes separated by lambda/2 |
| Doppler Effect | `nu' = nu0 * (u +/- v_R)/(u -/+ v_S)` | Numerator: v_R towards=+. Denominator: v_S towards=-. |
