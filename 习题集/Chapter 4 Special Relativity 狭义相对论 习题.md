# Chapter 4 Special Relativity 狭义相对论 -- 精编习题

> 共 20 题

## 选择题 (6题)

1. μ 子以 0.998c 的速度从高空飞向地面。在实验室参考系中测得 μ 子的寿命比其静止寿命长，这是因为：
   A. μ 子在运动中质量变大导致衰变变慢
   B. 时间膨胀效应——运动时钟变慢
   C. μ 子飞行轨迹变长
   D. 光速在 μ 子参考系中变化了

2. 关于固有时（proper time）Δt₀ 的说法，正确的是：
   A. 固有时是在任何惯性系中测得的最长的时间间隔
   B. 固有时测量时两事件必须同时发生
   C. 固有时是两事件在同一地点发生的那个参考系中测得的时间间隔
   D. 固有时是两事件在任意两个参考系中测量的时间差的平均值

3. 飞船以 0.8c 飞向距离地球 8 光年的星球。下列哪个说法是正确的？
   A. 从地球看，飞船钟测的旅程时间小于地球上测的旅程时间
   B. 从地球看，飞船的长度不变
   C. 地球测到的旅程时间为 6.4 年
   D. 飞船上测到的地球-星球距离仍然是 8 光年

4. 爱因斯坦火车思想实验中，车厢中点 O 发出闪光。地面观察者看到光先到达车尾后到达车头。这说明了什么？
   A. 地面参考系中光速与车厢运动方向有关
   B. 地面参考系中的信号传播速度超过光速
   C. 在列车参考系中光速比地面参考系中小
   D. 同时性是相对的——列车参考系中同时的事件在地面参考系中可以不同时

5. 一根 1 m 长的杆静止在 S' 系中，与 x' 轴夹角 30°。S' 相对 S 以速度 u 沿 x 轴运动。S 系中杆与 x 轴夹角变为 45°。则杆在 S 系中的长度最接近：
   A. 1.414 m
   B. 1.0 m
   C. 0.707 m
   D. 0.5 m

6. 关于同时性的相对性，下列说法正确的是：
   A. 在一个惯性系中同时同地发生的两事件，在另一惯性系中一定不同时
   B. 在一个惯性系中同时不同地发生的两事件，在另一惯性系中一定不同时
   C. 同时性是绝对的，与参考系无关
   D. 同时性的相对性只存在于高速运动，低速时同时性是绝对的

## 判断题 (4题)

7. 在狭义相对论中，真空中光速在所有惯性系中都等于 c，与光源和观察者的相对运动无关。
   (   ) True / False

8. 在经典力学（伽利略变换）中，同时性是绝对的，即在一个惯性系中同时发生的两个事件，在所有其他惯性系中也同时发生。
   (   ) True / False

9. 一根尺子以高速沿长度方向运动时，地面观察者测得其长度比静止时测的长度更长。
   (   ) True / False

10. 一艘飞船以高速飞离地球。从飞船上的宇航员看来，地球上的时钟也变慢了（地球上的时间膨胀了）。这是否意味着双方看到的效应是对称的、因而存在矛盾？实际上不矛盾——因为「同时」的定义在飞船和地球参考系中不同。
   (   ) True / False

## 计算题 (10题)

11. 一艘飞船以 0.8c 的速度相对地球飞行。飞船上的时钟测得某物理过程持续了 6.0 s。在地球上观测，该过程持续多长时间？（γ = 5/3）

12. 一根杆静止时长度为 2.0 m。当它以 0.6c 的速度沿杆长方向运动时，地面上的观察者测得杆的长度为多少？

13. 一艘飞船以 0.6c 相对地球飞行，飞船上向前发射一个相对于飞船速度为 0.6c 的探测器。求探测器相对地球的速度（以 c 为单位，保留三位小数）。

14. 在核裂变中，1 mol ²³⁵U 反应前后总静质量亏损 0.215 g。求释放的能量（用科学计数法表示，单位 J，保留 3 位有效数字，格式：1.94×10¹³）。

15. 一个静止质量为 m₀ 的粒子以 0.6c 运动，其动能是多少？用 m₀c² 的倍数表示。

16. S' 系相对 S 系以 0.6c 沿 +x 方向运动。在 S 系中，事件 A 的坐标为 x_A = 0，t_A = 0；事件 B 的坐标为 x_B = 3×10^8 m，t_B = 1.0 s。求 S' 系中两事件的时间间隔 Δt'。已知 γ = 1.25。

17. γ = 1.25 时，粒子动能与静能之比 E_k/E₀ 是多少？

18. 静止 μ 子的平均寿命 τ₀ = 2.2×10⁻⁶ s。来自高空的宇宙线 μ 子以 v = 0.998c 高速飞向地面。问：(1) 在实验室系中测得 μ 子的平均寿命是多少？(2) 它能飞行的平均距离是多少？

19. 一根米尺在 S'系中与 x' 轴成 30° 角。S'系相对 S 系沿 x 轴以速度 v 运动。在 S 系中测得该尺与 x 轴成 45° 角。求相对速度 v。

20. 一个电子的静质量 m₀ = 9.11×10⁻³¹ kg。求：(1) 电子以 v = 0.99c 运动时的动能；(2) 若该动能由经典公式 ½mv² 计算，误差有多大？
