Chapter 5 Mechanical Vibration 机械振动

Hook
一座桥、一把吉他、一次心跳——它们都在振动。当 Taylor Swift 拨动琴弦，琴码推拉面板，空气压缩再稀疏，声波传入你耳中。物理学家说：这不是魔法，这是 简谐振动。而一旦你掌握 x=Acos$ωt+φ$，你就同时理解了钟摆、弹簧、甚至量子谐振子。
TL;DR
本章学一个方程：x = Acos$ωt+φ$。从这个方程出发，求导得速度 v=-ωAsin$ωt+φ$，再求导得加速度 a=-ω²Acos$ωt+φ$=-ω²x。所有特征量（A/ω/T/f/φ）都从它定义；所有能量（½kA²）都从它推导；所有合成（同向/垂直/拍）都是它的组合；阻尼 + 策动力 + 共振是它的扩展。核心思路：先找 A、ω、φ、T、f 五个量，其余全部代入公式。

一、简谐振动基本方程 页3页4页5

1.1 弹簧振子：建立物理图像

想象一个弹簧振子（spring oscillator）：轻质弹簧一端固定，另一端连着质量 m 的木块，水平放置在光滑表面上。木块偏离平衡位置后释放，它就在 -A 到 +A 之间来回振动——这就是我们要研究的简谐振动（Simple Harmonic Motion, SHM）。

弹簧振子是理解 SHM 的最佳物理模型——本章一切公式最终都可以映射回「弹簧 + 木块」这个图像。

从受力分析入手：胡克定律给出 F = -kx（负号表示力始终指向平衡位置），结合牛顿第二定律：

$$ F = -kx = ma \quad \Rightarrow \quad a = -\frac{k}{m}x $$

令 ω² = k/m，得到 SHM 的微分方程：

$$ \frac{d^2x}{dt^2} + \omega^2 x = 0 $$

解这个微分方程，得到运动方程（kinematic equation）：

$$ \boxed{x = A\cos$\omega t + \varphi$} $$

这是本章最核心的公式。用 cos 不用 sin——cos 直接反映 t=0 时 x=Acosφ，物理意义清晰。若题干明确 t=0 时 x=0，则可写为 sin 形式，本质等价。

1.2 速度和加速度：一次求导 + 二次求导 必考

对 x = Acos$ωt+φ$ 求一阶导数得速度：

$$ v = \frac{dx}{dt} = -\omega A \sin$\omega t + \varphi$ $$

速度最大值 vmax = ωA，出现在平衡位置（x=0）时。

再求导得加速度：

$$ a = \frac{dv}{dt} = \frac{d^2x}{dt^2} = -\omega^2 A \cos$\omega t + \varphi$ = -\omega^2 x $$

加速度最大值 amax = ω²A，出现在最大位移处（x=±A）。

易错：a = -ω²x 是判断一个运动是否为 SHM 的判据——若某个运动满足「加速度与位移成正比、方向相反」，它就是简谐振动。不要混淆「a=-ω²x」（SHM 判据）和「a=-ω²Acos$ωt+φ$」（展开式）。

位移、速度、加速度的相位关系表：

物理量 表达式 振幅 相位领先关系
位移 x Acos$ωt+φ$ A 基准（0）
速度 v -ωAsin$ωt+φ$ = ωAcos$ωt+φ+π/2$ ωA 领先 x 相位 π/2
加速度 a -ω²Acos$ωt+φ$ = ω²Acos$ωt+φ+π$ ω²A 领先 x 相位 π（即反相）

记住：速度领先位移 π/2（v 先到峰值，再 1/4 周期后 x 到峰值）；加速度与位移恰好反相（a 最大时 x 在最负处）。

1.3 SHM 的判据 重点

判断一个运动是否为简谐振动，只需检查三种等价形式之一：

  • 线性力：F = -kx（力与位移成正比、方向相反）
  • 加速度：a = -ω²x
  • 角量：M = -Kθ，α = -ω²θ（用于单摆、复摆等扭摆）

例如：单摆在 θ < 5° 时，sinθ ≈ θ，M = -mgLθ，满足 M = -Kθ，因而是 SHM。

Checkpoint 1：一质点沿 x 轴做 SHM，运动方程为 x = 0.05cos$4πt + π/3$（SI 制）。求：$1$ 振幅、$2$ 角频率、$3$ 周期、$4$ 频率、$5$ t=0 时的速度。
参考答案

$1$ A = 0.05 m
$2$ ω = 4π rad/s
$3$ T = 2π/ω = 2π/$4π$ = 0.5 s
$4$ f = 1/T = 2 Hz
$5$ v = -ωAsin$φ$ = -4π × 0.05 × sin$π/3$ = -0.2π × √3/2 ≈ -0.544 m/s

二、特征量：从方程中提取一切 页6页7页8

2.1 五个特征量 必考

给定 x = Acos$ωt+φ$，这五个量完全描述了一次简谐振动：

名称 符号 定义 求解方法
振幅 Amplitude A 离开平衡位置的最大距离 由初始条件决定：A = √$x₀² + v₀²/ω²$
角频率 Angular freq. ω = 2π/T = 2πf 由系统本身性质决定：弹簧 ω=√$k/m$；单摆 ω=√$g/L$
周期 Period T 完成一次完整振动的时间 T = 2π/ω
频率 Frequency f 每秒振动次数 f = 1/T = ω/$2π$
初相位 Initial phase φ t=0 时的相位，决定初始运动状态 cosφ = x₀/A，再用 v₀ 的符号确定象限

2.2 初相位 φ 的确定：用 v₀ 符号定象限 重点

确定 φ 是考试中的高频易错点。步骤：

  1. 代入 t=0：x₀ = Acosφ --> cosφ = x₀/A，得 φ 的两个候选值（±|φ| 或 φ 和 2π-φ）
  2. 代入 v₀ = -ωAsinφ，用 v₀ 的正负号排除一个候选值
易错：只用 cosφ = x₀/A 确定不了 φ 的唯一值（cos 在一、四象限和二三象限各有两个角对应同一个余弦值），必须再用 v₀ 的符号（即 sinφ 的符号）做二次判定。v₀ > 0 意味着 sinφ < 0（第三/四象限）；v₀ < 0 意味着 sinφ > 0（第一/二象限）。
Checkpoint 2：某 SHM，A=0.1m，ω=10 rad/s。t=0 时 x₀=0.05m，v₀=-0.866m/s。求 φ。
参考答案

cosφ = x₀/A = 0.05/0.1 = 0.5 --> φ = π/3 或 5π/3（即 -π/3）
v₀ = -ωAsinφ = -10×0.1×sinφ = -sinφ = -0.866 --> sinφ = 0.866 = √3/2
sinφ > 0 --> φ 在第一或第二象限 --> φ = π/3

Checkpoint 3：弹簧振子，k=100 N/m，m=1 kg。t=0 时从平衡位置向正方向运动，v₀=2 m/s。求运动方程。
参考答案

ω = √$k/m$ = √$100/1$ = 10 rad/s
t=0: x₀=0 --> cosφ=0 --> φ=π/2 或 3π/2
v₀ = -ωAsinφ = 2 > 0 --> sinφ < 0 --> φ = 3π/2（即 -π/2）
由 v₀ = ωA 得 A = v₀/ω = 2/10 = 0.2 m
x = 0.2cos$10t + 3π/2$ = 0.2cos$10t - π/2$ m

三、简谐振动的能量 页12页13

3.1 动能 + 势能 = 常数 必考

SHM 中只有保守力（弹簧弹力）做功，机械能守恒。

动能：

$$ E_k = \frac{1}{2}mv^2 = \frac{1}{2}m\omega^2 A^2 \sin^2$\omega t + \varphi$ $$

势能：

$$ E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2 \cos^2$\omega t + \varphi$ $$

利用 k = mω²：

$$ \boxed{E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2} $$

总能量只取决于振幅 A 和劲度系数 k，与时间无关。动能和势能各自以 2ω 的频率振荡，一个最大时另一个为零——能量在两种形式之间周期性转换。

3.2 能量图解读

在一个周期内：

  • x = ±A 时：Ek = 0，Ep = E = ½kA²（全部是势能）
  • x = 0 时：Ek = E = ½kA²，Ep = 0（全部是动能）
  • 任意位置：Ek/E = 1 - $x/A$²，Ep/E = $x/A$²
易错：E = ½kA² 中的 k 是弹簧的劲度系数，不是动能符号。总能量的表达式与时间无关——无论在哪一时刻，只要知道 A 和 k，能量就确定了。
Checkpoint 4：弹簧振子，m=0.5 kg，k=200 N/m，振幅 A=0.04 m。求：$1$ 总机械能；$2$ 位移 x=0.02 m 时的动能。
参考答案

$1$ E = ½kA² = 0.5 × 200 × $0.04$² = 0.16 J
$2$ x=0.02 m 时，Ep = ½kx² = 0.5 × 200 × $0.02$² = 0.04 J
Ek = E - Ep = 0.16 - 0.04 = 0.12 J

四、振动的合成与特殊振动 页16页17页18页20页21页22

4.1 同方向同频率的合成 重点

两个 SHM 在同一方向、同频率：

$$ x_1 = A_1\cos$\omega t + \varphi_1$, \quad x_2 = A_2\cos$\omega t + \varphi_2$ $$

合振动 仍是简谐振动，频率仍为 ω：

$$ x = x_1 + x_2 = A\cos$\omega t + \varphi$ $$

其中：

$$ A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos$\varphi_2 - \varphi_1$} $$ $$ \tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2} $$

两种特殊情况：

  • 同相（Δφ = φ₂-φ₁ = 0 → cosΔφ=1）：A = A₁+A₂（振幅最大，振动加强）
  • 反相（Δφ = π → cosΔφ=-1）：A = |A₁-A₂|（振幅最小，若 A₁=A₂ 则 A=0，质点静止）

4.2 拍（Beat）：同方向频率接近的合成 重点

当两个 SHM 频率接近但不相等时（ω₁ ≈ ω₂）：

$$ x = 2A\cos\left$\frac{\omega_2 - \omega_1}{2}t\right$ \cdot \cos\left$\frac{\omega_2 + \omega_1}{2}t + \phi\right$ $$

这表现为「高频载波 + 低频包络」：合振动的振幅以 |ω₂-ω₁|/2 的频率缓慢周期性变化，形成时强时弱的「拍」现象。

拍频（beat frequency）：

$$ f_{\text{拍}} = |f_2 - f_1| \quad \text{或} \quad \omega_{\text{拍}} = |\omega_2 - \omega_1| $$

拍频等于两分振动频率之差的绝对值。注意：ω拍 = |ω₂-ω₁|（是角频率）；而包络的角频率是 |ω₂-ω₁|/2。考试中直接问「拍频」通常答频率差。

易错：拍频 = |f₂-f₁|，不是 $f₁+f₂$/2。$f₁+f₂$/2 是平均频率（载波频率），对应音高。拍频是强度变化的频率。

4.3 相互垂直的同频率合成：李萨如图形 重点

两个方向互相垂直、同频率的 SHM：

$$ x = A_1\cos$\omega t + \varphi_1$, \quad y = A_2\cos$\omega t + \varphi_2$ $$

合振动轨迹方程：

$$ \frac{x^2}{A_1^2} + \frac{y^2}{A_2^2} - \frac{2xy}{A_1A_2}\cos$\varphi_2 - \varphi_1$ = \sin^2$\varphi_2 - \varphi_1$ $$

四种典型相位差对应的轨迹：

Δφ = φ₂-φ₁ 轨迹形状 运动方向
0 直线 y = $A₂/A₁$x（一三象限） 沿直线来回
π 直线 y = -$A₂/A₁$x（二四象限） 沿直线来回
π/2 正椭圆（x²/A₁² + y²/A₂² = 1） 顺时针
3π/2 或 -π/2 正椭圆（同上） 逆时针

Δφ 为其他值时，轨迹为倾斜的椭圆。当 A₁ = A₂ 且 Δφ = π/2 时，轨迹为圆。李萨如图形是示波器显示的基础。

4.4 阻尼振动（Damped Oscillation） 重点

当存在阻力 f = -γv（γ 为阻力系数）时，振动方程为：

$$ \frac{d^2x}{dt^2} + 2\beta\frac{dx}{dt} + \omega_0^2 x = 0 $$

其中 阻尼因子 β = γ/$2m$，固有角频率 ω₀ = √$k/m$。

三种阻尼状态：

状态 条件 运动特点
弱阻尼（under damped） β < ω₀ x = A₀e⁻ᵝᵗcos$ωt+φ$，振幅指数衰减，周期 T = 2π/√$ω₀²-β²$ > T₀
临界阻尼（critically damped） β = ω₀ 不振，最快回到平衡位置
过阻尼（over damped） β > ω₀ 不振，缓慢回到平衡位置
易错：阻尼振动不是简谐振动（振幅随时间衰减），但在弱阻尼下可近似视为「振幅指数衰减的准周期运动」。阻尼使周期变长（T > T₀）。

4.5 受迫振动与共振 重点

在周期性外力（策动力 F = F₀cosωt）作用下：

$$ \frac{d^2x}{dt^2} + 2\beta\frac{dx}{dt} + \omega_0^2 x = h\cos\omega t, \quad h = F_0/m $$

稳态解：x = Acos$ωt+φ$（频率等于策动力频率），其中：

$$ A = \frac{h}{\sqrt{$\omega_0^2 - \omega^2$^2 + 4\beta^2\omega^2}} $$

共振（Resonance）：当策动力频率 ω = ωr 时，振幅 A 达最大值。

  • 位移共振频率：ωr = √$ω₀² - 2β²$ ≈ ω₀（当阻尼很小时）
  • 共振振幅：Ar = h/$2β√(ω₀²-β²$)
  • 阻尼越小，共振峰越尖锐（sharp resonance）；阻尼大时共振峰平缓。

共振的应用：钢琴/小提琴共鸣箱增强声音、收音机调谐；共振的危害：桥梁/建筑物在共振频率下可能被破坏（如 Tacoma Narrows Bridge）、发动机频率接近车身固有频率时剧烈抖动。

Checkpoint 5：两个同方向 SHM：x₁ = 0.06cos$10t + π/6$，x₂ = 0.08cos$10t - π/3$。求合振动的振幅。
参考答案

Δφ = φ₂ - φ₁ = -π/3 - π/6 = -π/2，cos$Δφ$ = cos$-π/2$ = 0
A = √$A₁² + A₂² + 2A₁A₂cos(Δφ$) = √$0.06² + 0.08²$ = √0.01 = 0.10 m

Checkpoint 6：两个音叉同时发声，频率分别为 440 Hz 和 444 Hz。求拍频。
参考答案

f拍 = |f₂ - f₁| = |444 - 440| = 4 Hz
