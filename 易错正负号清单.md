# 大学物理A1 — 易错正负号/方向清单

| 物理量/场景 | 正确约定 | 常见错误 |
|------------|---------|---------|
| v的符号(1D运动) | 沿选定正方向为正,反之为负 | 不看正方向,v_-和v_+符号颠倒 |
| 动量变化Δp | Δp=mv₂-mv₁(矢量差!) | 反向弹回时写成v₂-v₁=0,实际Δp=2mv |
| 冲量方向 | 冲量方向=Δp方向(≠力的方向,如果力方向在过程中改变了) | 误以为冲量方向恒等于力的方向 |
| 功的正负 | 力与位移夹角<90°→正; >90°→负 | 摩擦力做负功写成正功 |
| 重力势能零点 | 可任意选取;零点变了势能数值变了但ΔEp不变 | 零点选取后改用不同的零点对比时出错 |
| 保守力做功 | W_c=-ΔE_p(保守力做正功→势能减少) | 写成W_c=+ΔE_p,正负号反了 |
| 力矩方向(右手定则) | 逆时针为正(右手四指→拇指向上为正) | 忘了用右手定则,凭直觉瞎写 |
| 角动量方向 | 与角速度同向(右手定则) | 忘记角动量是矢量(轴向) |
| 时间膨胀方向 | Δt=γΔt₀, 运动钟变慢→Δt>Δt₀ | 把Δt₀(固有时)和Δt搞反→Δt=Δt₀/γ |
| 长度收缩方向 | L=L₀/γ, 仅在运动方向上收缩 | 认为三维都缩(错!),y和z方向不变 |
| 速度叠加符号 | u_x'=(u_x-v)/(1-u_x*v/c^2) | 经典加减:0.8c+0.6c=1.4c→超光速! |
| 振动初相φ的范围 | 由cosφ=x₀/A得|φ|∈[0,π]→再由v₀符号定象限 | 只用cosφ→得到±φ→忘记用v₀定正负 |
| 波传播方向与括号符号 | (t-x/u):+x方向; (t+x/u):-x方向 | (t-x/u)写成-x方向传播 |
| 半波损失条件 | 波疏→波密反射→相位突変π | 自由端反射无半波损失(混淆固定端和自由端) |
| 多普勒效应分子/分母符号 | 分子v_R趋近取+,分母v_S趋近取- | 分子分母符号搞反→ν变大写成ν变小 |
| 热力学第一定律 | ΔU=Q+W (W:外界对系统做功>0) | 混淆ΔU=Q-W版本→W符号全反 |
| 绝热过程W的计算 | W=-ΔU=-nC_VΔT(对外做功>0时ΔT<0→W>0) | 直接用W=nRTln(V₂/V₁)→这是等温公式! |
| 循环效率η的分母 | η=W_net/Q_in(Q_in只计吸热) | 把放热量也加进分母→η虚高 |
| 熵变计算 | 不可逆过程的ΔS用可逆过程路径算 | 直接用Q/T→Q/T只是熵变的一部分,不完整 |
