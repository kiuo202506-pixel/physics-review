# 大学物理A1 — SI单位检查清单

## 基本量

| 物理量 | 符号 | SI单位 | 量纲 |
|--------|------|--------|------|
| 长度/位移 | x, s, L, r | m(米) | [L] |
| 质量 | m, M | kg(千克) | [M] |
| 时间 | t, T, τ | s(秒) | [T] |
| 温度 | T | K(开尔文) | [Θ] |
| 物质的量 | n, ν | mol(摩尔) | [N] |
| 角度 | θ | rad(弧度), 1rad=180°/π | 无量纲 |

## 导出量

| 物理量 | 公式 | SI单位 | 换算/注意 |
|--------|------|--------|----------|
| 速度 | v=dx/dt | m/s | 1km/h = 1/3.6 m/s |
| 加速度 | a=dv/dt | m/s^2 | g = 9.80 m/s^2 |
| 力 | F=ma | N(牛顿) | 1N = 1kg*m/s^2 |
| 动量 | p=mv | kg*m/s |  |
| 冲量 | I=∫Fdt | N*s | 1N*s = 1kg*m/s |
| 功/能量/热量 | W, E, Q | J(焦耳) | 1J = 1N*m; 1eV=1.60e-19J |
| 功率 | P=dW/dt | W(瓦特) | 1W = 1J/s |
| 力矩 | M=r×F | N*m | 力矩和功单位相同但物理含义不同 |
| 转动惯量 | I=Σmr^2 | kg*m^2 |  |
| 角速度/角频率 | ω | rad/s | 1rpm = 2π/60 rad/s |
| 角动量 | L=Iω | kg*m^2/s |  |
| 压强 | p=F/A | Pa(帕斯卡) | 1atm=1.013e5Pa; 1bar=1e5Pa |
| 频率 | f | Hz(赫兹) | 1Hz = 1/s |
| 波长 | λ | m |  |
| 波数 | k=2π/λ | rad/m |  |
| 气体常量 | R | J/(mol*K) | R=8.31; k=R/N_A=1.38e-23J/K |
| 熵 | S | J/K |  |

## 考前必检

- [ ] 所有物理量代入公式前单位已统一为SI(kg/m/s/K/Pa)
- [ ] 角度量已转换为rad而非degree
- [ ] 最终答案的量纲与所求物理量一致
- [ ] 数值数量级合理(g=9.8, R=8.31, k=1.38e-23, c=3e8, N_A=6.02e23)
- [ ] 特殊值验证(如v<<c时相对论公式应退化到经典结果)
