第8章 热力学基础 Thermodynamics Basis

为什么一杯热水放久了会凉，而同样一杯凉水不会自动变热？为什么摩擦生热轻而易举，而用热来驱动机器却需要锅炉和冷凝器两套设备？这些「方向性」问题，牛顿力学和能量守恒都回答不了——热力学正是为此而生。

TL;DR — 热力学第一定律（ΔE = Q + W）是能量记账本：系统内能变化 = 吸热 + 外界做功。热力学第二定律管方向：热量不能自发从低温传向高温；不可能从单一热源吸热全部转化为功而不产生其他影响。卡诺定理给出了热机效率的天花板 η = 1 − T₂/T₁。解题核心思路：先判断过程类型→套对应公式。

一、为什么要学热力学

热力学处理的是大量分子集体行为的宏观规律。它不关心单个分子的速度和位置，而是关心压强 P、体积 V、温度 T 和内能 E 这些可测量的宏观量。

热力学的三大支柱：

  • 第一定律（能量守恒）→ 告诉你「能不能发生」——能量是否平衡
  • 第二定律（方向性）→ 告诉你「会不会自发发生」——过程有无方向限制
  • 卡诺定理（效率上限）→ 告诉你「能做到多好」——理论效率不可逾越

二、功、热量与内能 —— 三个极易混淆的量 页3页4页5

2.1 功（Work）必考

功是过程量，不是状态函数。依赖具体路径，同样的初末态走不同路径，功的大小不同。

$$dA = p\,dV, \qquad A = \int_{V_1}^{V_2} p\,dV$$

A 等于 P-V 图上过程曲线下的面积。dA > 0（dV > 0）表示系统对外做正功；dA < 0（dV < 0）表示外界对系统做功。

易错：A = ∫p dV 只适用于准静态过程。非准静态过程不能用此式积分，但等容过程（dV=0）无论准静态与否都有 A = 0。
功是不是状态函数？不是——从 A 到 B 走不同路径，P-V 曲线下面积不同，功不同。

2.2 热量（Heat）必考

热量也是过程量，因温差而传递的能量。与功一样依赖过程路径，不是状态函数。「物体含有多少热量」是错误说法——热量只在传递过程中有意义。

2.3 内能（Internal Energy）必考

内能是状态函数，只取决于系统的当前状态（对理想气体只取决于温度）。

$$E = \frac{M}{M_{\text{mol}}} \cdot \frac{i}{2}RT, \qquad \Delta E = \frac{M}{M_{\text{mol}}} \cdot \frac{i}{2}R\,\Delta T$$

M 为气体质量，Mmol 为摩尔质量，i 为自由度（单原子 i=3，双原子刚性 i=5，多原子刚性 i=6），R = 8.31 J/$mol·K$。ΔE 只由初末态温度差决定，与路径无关。

物理量功 A热量 Q内能 E / ΔE
类型 过程量 过程量 状态函数
与路径 有关 有关 无关（只取决于温度）

「内能是状态函数」这句话的具体含义是什么？——初末态确定后 ΔE 就唯一确定，无论中间走等温还是等压还是绝热路径。

三、热力学第一定律 —— 能量记账本 页8页9

3.1 数学表述 必考

$$\Delta E = Q + W$$

其中 Q 为系统从外界吸收的热量，W 为外界对系统做的功。注意：PPT 用 ΔE = Q + A，但此处的 A 是系统对外做的功，即 ΔE = Q − A对外。这里统一用 ΔE = Q + W 的符号体系，W 为外界对系统的功。

符号约定（与 PPT 一致）：

  • Q > 0：系统吸热；Q < 0：系统放热
  • A > 0（PPT）：系统对外做正功 → 按 ΔE = Q − A 使用
  • W > 0：外界对系统做正功 → 按 ΔE = Q + W 使用
易错：考试时注意题目用的是 ΔE = Q − A（A 为系统对外做功）还是 ΔE = Q + W（W 为外界对系统做功）。两者等价，只是 A = −W。读题先判断用的是哪个符号体系。

另一表述：第一类永动机（不消耗任何能量而持续对外做功的机器）不可能制成。

3.2 对微小过程 补充

$$dQ = dE + dA = dE + p\,dV$$

把第一定律拆成微分量形式，便于对各种过程积分计算。

内能增加100J，同时对外做功50J，系统吸热多少？——ΔE = Q − A → 100 = Q − 50 → Q = 150J。

四、四大准静态过程 —— 解题核心 页13

解题口诀（用户偏好）：拿到热力学题→先判断过程类型→套对应规则：等温→ΔE=0、等容→W=0、绝热→Q=0、循环→ΔE=0。

4.1 等容过程（Isovolumetric, dV = 0）必考

$$A = 0, \qquad Q_V = \Delta E = \frac{M}{M_{\text{mol}}} \cdot \frac{i}{2} R \,\Delta T = \frac{M}{M_{\text{mol}}} C_V \Delta T$$

体积不变→不做功→所有吸热全用来增加内能。

等容摩尔热容：CV = $i/2$R。单原子 CV = $3/2$R，双原子刚性 CV = $5/2$R。

4.2 等压过程（Isobaric, dp = 0）必考

$$A = p$V_2 - V_1$ = \frac{M}{M_{\text{mol}}} R \,\Delta T$$ $$Q_P = \Delta E + A = \frac{M}{M_{\text{mol}}} \cdot \frac{i+2}{2} R \,\Delta T = \frac{M}{M_{\text{mol}}} C_P \Delta T$$

系统吸热→一部分增加内能，一部分对外做功。

迈耶公式（Mayer's formula）：CP = CV + R = $i+2$R/2。1 mol 理想气体等压升温 1K 比等容多吸 8.31J 热量——多出的正好用来对外做功。

$$\text{比热容比（绝热指数）：} \gamma = \frac{C_P}{C_V} = \frac{i+2}{i}$$

单原子 γ = 5/3 ≈ 1.67；双原子刚性 γ = 7/5 = 1.40；多原子刚性 γ = 8/6 ≈ 1.33。

易错：等压升温时，Q = ΔE + A > ΔE。有同学误以为「吸热只增加内能」，忘了等压过程还要对外做 pΔV 的功。

4.3 等温过程（Isothermal, dT = 0）必考

$$\Delta E = 0, \qquad Q = A = \frac{M}{M_{\text{mol}}} RT \ln\frac{V_2}{V_1} = \frac{M}{M_{\text{mol}}} RT \ln\frac{p_1}{p_2}$$

温度不变→内能不变（ΔE=0）→吸热全部用来对外做功。A 可用 pV = $M/Mmol$RT 和 p = $M/Mmol$RT/V 代入积分得到。

易错：等温过程 Q = A 不是普遍结论——仅当系统为理想气体时成立（因为只有理想气体的内能仅取决于温度）。

4.4 绝热过程（Adiabatic, Q = 0）必考

绝热过程中系统与外界无热量交换。对外做功完全以内能减少为代价：A = −ΔE。

$$A = -\Delta E = -\frac{M}{M_{\text{mol}}} C_V $T_2 - T_1$ = \frac{M}{M_{\text{mol}}} C_V $T_1 - T_2$$

准静态绝热方程（三组等价形式）：

$$pV^{\gamma} = \text{const}_1, \qquad V^{\gamma-1}T = \text{const}_2, \qquad p^{\gamma-1}T^{-\gamma} = \text{const}_3$$

注意三个 const 互不相同。

绝热线比等温线陡：在 P-V 图上，绝热线曲率更大。原因：绝热膨胀时温度同时下降，压强比等温膨胀降得更快。

易错：绝热自由膨胀（气体冲入真空）是非准静态过程→绝热方程不适用！但 Q=0, A=0（真空→无反抗），所以 ΔE = 0, T 不变。与准静态绝热膨胀（对外做功→T↓）完全不同！
绝热自由膨胀 VS 准静态绝热膨胀：前者 Q=0, A=0, ΔE=0, T 不变；后者 Q=0, A>0（对外做功）, ΔE<0, T 下降。

4.5 四大过程对比总表 必考

过程 条件 过程方程 Q W（外界对系统） ΔE 特点
等容 dV = 0 p/T = const $M/Mmol$CVΔT 0 $M/Mmol$CVΔT Q = ΔE，功为零
等压 dp = 0 V/T = const $M/Mmol$CPΔT −pΔV $M/Mmol$CVΔT Q = ΔE − W，三项均非零
等温 dT = 0 pV = const $M/Mmol$RT ln$V₂/V₁$ −$M/Mmol$RT ln$V₂/V₁$ 0 Q = −W，内能不变

4.6 解题流程 高频

三步法解热力学计算题：

  1. 判断过程类型（等容/等压/等温/绝热/循环？）→ 确定一个 Q/W/ΔE 中哪个为零
  2. 用状态方程 pV = nRT 求出未知状态参数
  3. 用第一定律 + 过程特性 计算各量
1 mol 双原子理想气体等温膨胀，V₁→2V₁，T=300K。求 Q。——ΔE=0，Q=A=RTln2=8.31×300×0.693≈1728J。

五、循环过程、热机与制冷机 页30

5.1 循环过程特性 必考

系统经历一系列变化后回到初始状态→ΔE = 0（内能不变）。在 P-V 图上，循环是一条闭合曲线。

$$\Delta E_{\text{cycle}} = 0, \qquad Q_{\text{net}} = A_{\text{net}}$$

净功（Anet）= 闭合曲线包围的面积。顺时针→正循环（热机），对外做正功；逆时针→逆循环（制冷机），外界对系统做正功。

5.2 热机效率 必考

$$\eta = \frac{A_{\text{net}}}{Q_1} = 1 - \frac{Q_2}{Q_1}$$

Q1 为循环中所有吸热过程的热量之和（取绝对值），Q2 为所有放热过程的热量之和（取绝对值）。

易错：η = 1 − Q₂/Q₁ 中的 Q₁ 和 Q₂ 都是取绝对值后的数值（正数），不是代数意义上的代数量。

5.3 制冷系数 COP 重点

$$e = \frac{Q_2}{A_{\text{net}}} = \frac{Q_2}{Q_1 - Q_2}$$

Q2 为从低温热源吸收的热量，Anet 为外界对系统做的净功。

热机循环中，总吸热 Q₁=500J，总放热 Q₂=350J。效率 η = ?——η = 1 − 350/500 = 0.30 = 30%。

六、卡诺循环与卡诺定理 页24页25页26

6.1 卡诺循环的四步分解 必考

卡诺循环由两个等温过程 + 两个绝热过程组成，工作在 T₁（高温）和 T₂（低温）两个热源之间。

1
等温膨胀
A→B, T=T₁
从高温热源 吸热 Q₁
Q₁ = nRT₁ ln$V₂/V₁$
2
绝热膨胀
B→C, Q=0
温度降至 T₂
V₂^$γ−1$T₁ = V₃^$γ−1$T₂
3
等温压缩
C→D, T=T₂
向低温热源 放热 Q₂
Q₂ = nRT₂ ln$V₃/V₄$
4
绝热压缩
D→A, Q=0
温度回升至 T₁
V₁^$γ−1$T₁ = V₄^$γ−1$T₂

6.2 卡诺效率 必考

$$\eta_{\text{Carnot}} = 1 - \frac{T_2}{T_1}$$

T₁ 为高温热源温度，T₂ 为低温热源温度（单位：K）。

关键推导步骤：

  1. 等温膨胀吸热：Q₁ = nRT₁ ln$V₂/V₁$
  2. 等温压缩放热：Q₂ = nRT₂ ln$V₃/V₄$
  3. 利用绝热方程得 V₂/V₁ = V₃/V₄ → Q₂/Q₁ = T₂/T₁
  4. 代入 η = 1 − Q₂/Q₁ = 1 − T₂/T₁

6.3 卡诺定理 重点

  • 工作在相同高低温热源之间的一切可逆热机，效率均等于卡诺效率 1 − T₂/T₁。
  • 不可逆热机的效率小于卡诺效率。
  • 卡诺效率仅取决于两个热源的温度，与工作物质无关。
$$\eta_{\text{Carnot refrigerator}} = \frac{T_2}{T_1 - T_2}$$

卡诺制冷机的制冷系数。

易错：提高热机效率的途径是增大高低温热源温差（提高 T₁ 或降低 T₂），而不是增加工作物质质量或改变工质种类。卡诺效率与工质无关！
T₁=500K, T₂=300K 的卡诺热机，效率多少？——η = 1 − 300/500 = 40%。如果实际效率只有 25%，正常吗？——正常，因为实际热机为不可逆，效率必然小于卡诺效率。

七、热力学第二定律 页20页21页22

7.1 两种经典表述 必考

开尔文表述（Kelvin Statement）

不可能从单一热源吸取热量全部转化为功而不产生其他影响。

→ 第二类永动机（η=100% 的热机）不可能制成。

克劳修斯表述（Clausius Statement）

热量不可能自动地从低温物体传到高温物体而不产生其他影响。

→ 冰箱能把热量从低温搬到高温——但消耗了电能（产生了其他影响）。

两种表述的对比：
开尔文表述克劳修斯表述
涉及过程热→功转换热量传递
关键词单一热源、全部转化、无其他影响自动地、低温→高温、无其他影响
否定对象第二类永动机自发传热反向
实质热功转换不可逆热传导不可逆
等价性两种表述等价——违反其一则必违反其二
易错：热力学第二定律最常考的错误说法——「热量不能从低温物体传到高温物体」（缺少「自动地/不产生其他影响」）。冰箱就是把热量从低温搬到高温，但它消耗了电能！

7.2 可逆与不可逆过程 必考

  • 可逆过程：系统反向进行时能重复正向过程的每一状态，且不引起外界任何变化。条件：无限缓慢（准静态）+ 无摩擦/无耗散。
  • 不可逆过程：自然界绝大多数实际过程都是不可逆的。

典型的不可逆过程：

  • 摩擦生热（功→热）
  • 热传导（高温→低温）
  • 气体自由膨胀（无反抗膨胀）
  • 扩散过程

重要结论：

  • 卡诺循环是可逆循环。
  • 可逆过程一定是准静态过程；但准静态过程不一定可逆（如有摩擦的缓慢过程是准静态但不可逆）。
  • 有摩擦的过程一定不可逆。
易错：（1）可逆过程必为准静态过程 ✓；（2）准静态过程必为可逆过程 ✗（缺「无耗散」条件）；（3）不可逆过程就是不能反向进行的过程 ✗（可以反向，只是反向时会引起其他变化）；（4）有摩擦的过程必为不可逆过程 ✓。
判断正误：「热力学第二定律说明，功可以全部转化为热，但热不能全部转化为功。」——错！热可以全部转化为功（如等温膨胀 Q=A），但不能在循环中从单一热源吸热全部转化为功而不产生其他影响。关键词是「单一热源」「不产生其他影响」。

八、熵与熵增加原理 页28页29页30

8.1 熵的定义 重点

熵（Entropy）是状态函数，度量系统的无序程度。对于可逆过程：

$$dS = \frac{dQ_{\text{rev}}}{T}, \qquad \Delta S = S_2 - S_1 = \int_1^2 \frac{dQ_{\text{rev}}}{T}$$

单位：J/K。dQrev 为可逆过程中系统吸收的热量。因为熵是状态函数，ΔS 只取决于初末态，与是否可逆路径无关——计算 ΔS 时可以设计一个连接初末态的可逆过程来算。

8.2 熵增加原理 必考

$$\Delta S_{\text{孤立系统}} \geq 0$$

孤立系统（或绝热系统）中，自发过程总是朝着熵增加的方向进行。「=」对应可逆过程，「>」对应不可逆过程。熵增加原理是热力学第二定律的数学表述。

8.3 理想气体熵变计算 高频

对于理想气体可逆过程，可以推导出：

$$\Delta S = \frac{M}{M_{\text{mol}}} C_V \ln\frac{T_2}{T_1} + \frac{M}{M_{\text{mol}}} R \ln\frac{V_2}{V_1}$$

此式适用于任意连接状态 $T₁, V₁$ 到 $T₂, V₂$ 的熵变计算，因为熵是状态函数。

各过程熵变特例：

  • 等温过程：ΔS = nR ln$V₂/V₁$
  • 等容过程：ΔS = nCV ln$T₂/T₁$
  • 绝热可逆过程：ΔS = 0（等熵过程）
  • 绝热自由膨胀：ΔS = nR ln$V₂/V₁$ > 0（不可逆→熵增）
易错：绝热可逆过程 ΔS = 0（等熵），但绝热不可逆过程（如自由膨胀）ΔS > 0！只因为「绝热」不能推出「熵不变」——还需要「可逆」。
1 mol 理想气体等温膨胀，体积加倍。ΔS = ?——ΔS = R ln 2 ≈ 8.31 × 0.693 ≈ 5.76 J/K。
