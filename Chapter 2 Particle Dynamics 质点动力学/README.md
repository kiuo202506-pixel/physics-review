# Chapter 2 Particle Dynamics 质点动力学

## 核心公式速查

| Concept | Formula | Notes/Conditions |
|---------|---------|------------------|
| Newton's 1st Law | `F=0 => v=constant` | Defines inertial frames and force |
| Newton's 2nd Law | `F_net = m*a` | Vector equation, instantaneous, inertial frames only |
| Newton's 3rd Law | `F_A_on_B = -F_B_on_A` | Equal magnitude, opposite, on different bodies |
| Momentum | `p = m*v` | Vector, kg*m/s |
| Impulse-Momentum Theorem | `I = integral(F*dt) = Delta_p` | Time integral of force |
| Conservation of Momentum | `Sum(F_ext) = 0 => Sum(p) = constant` | Primary tool for collisions |
| Work | `W = integral(F dot dr)` | Space integral of force, path-dependent |
| Kinetic Energy Theorem | `W_net = Delta_Ek = mv2^2/2 - mv1^2/2` | Scalar equation, universally valid |
| Conservative Force | `closed_loop_integral(F_c dot dr) = 0` | Path-independent work, defines potential energy |
| Potential Energy | `Ep = mgh or kx^2/2 or -GMm/r` | Gravitational/elastic/universal gravitation |
| Conservation of Mechanical Energy | `W_nonconservative = 0 => Ek+Ep = constant` | Only conservative forces do work |
| Average force from momentum | `F_avg = Delta_p/Delta_t` | Collision problems: ignore gravity when F_impact >> mg |
