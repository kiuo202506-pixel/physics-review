牛顿第一定律与惯性参考系 页3页4页5页6页7页8 必考

站在公交车上，司机突然刹车——你为什么会往前倾倒？不是有人推你，是你的身体「拒绝」改变原有的运动状态。
任何物体在合外力为零时，保持静止或匀速直线运动状态不变。第一定律不是 F=0 时的特例——它独立完成了两件大事：定义了惯性参考系，并定性定义了力（力是改变运动状态的根源，不是维持运动的原因）。

Why：没有惯性系就没有牛顿力学

牛顿第二定律 F=ma 中，加速度 a 必须相对于惯性系测量。如果参考系本身在加速（非惯性系），F=ma 直接使用会得到荒谬结果——比如在急刹车的公交车上，你感觉被「往前推」，但没有一个真实的力在推你。第一定律划定了舞台边界。

What：惯性与惯性系的精确含义

  • 惯性（Inertia）：物体保持原有运动状态的「顽固性」，是物质最基本的属性之一。质量 m 是惯性大小的定量量度——m 越大，速度越难改变。
  • 惯性参考系：牛顿定律成立的参考系。判据：不受外力作用的物体在此参考系中保持静止或匀速直线运动。
实用惯性系的精度层级
惯性系非惯性因素角速度量级 $rad/s$实用程度
地面参考系地球自转$\omega \approx 7.3\times10^{-5}$最常用，近似惯性系
地心参考系地球绕太阳公转$\omega \approx 2.0\times10^{-7}$较地面系严格
日心参考系太阳绕银河中心$\omega \approx 8.0\times10^{-16}$更严格
FK4 参考系——当前最佳（1535 颗恒星平均位形基准）
易错：运动学中，参考系可以任意选取（运动描述是相对的，选哪个系都合法）；动力学中，牛顿定律只能在惯性系中使用。在非惯性系中直接套 F=ma 会得到错误结果——必须引入惯性力才能修正。

在匀速行驶的高铁车厢内，桌面上一个苹果保持静止。以地面为参考系，苹果受哪些力？合力多大？以高铁为参考系呢？高铁是严格的惯性系吗？

参考答案 以地面为系：受重力 mg 向下、桌面支持力 N 向上，合力为零（v=常数，a=0）。以高铁为系：同样受 mg 和 N，合力也为零——高铁匀速运动，相对地面的加速度为零，因此高铁也是惯性系。但地面只是近似惯性系（地球自转 $\omega\approx7.3\times10^{-5}$ rad/s），严格来说高铁也不是严格惯性系。

判断：「牛顿第一定律其实就是牛顿第二定律在 F=0 时的特例。」

参考答案 错误。第一定律独立地定义了惯性参考系——这一功能是第二定律无法完成的。没有第一定律划定惯性系，第二定律中的加速度就没有参照基准。同时第一定律还定性定义了「力」——力是改变运动状态的原因（而非维持运动的原因），这也是第二定律未明确阐述的。

牛顿第二定律 F=ma 页9页10页18页19页20页21页22 核心枢纽

推动一辆空购物车和一辆装满货的购物车——用同样的力，空车一下子飞出去，满载车慢悠悠起步。力的「效果」不仅取决于力有多大，还取决于对象的「质量」——即惯性有多大。
$$\vec{F}_{\text{合}} = m\vec{a}$$ 合外力 = 质量 x 加速度。这是经典力学的核心方程——将力（原因）与加速度（结果）建立了定量、瞬时、矢量性的联系。且力满足叠加原理：几个力同时作用，合力产生的加速度等于各力单独产生的加速度之矢量和。

Why：为什么 F=ma 是「枢纽」？

牛顿之前，人们普遍认为力维持运动（亚里士多德）。伽利略用思想实验、牛顿用数学语言终结了这个两千年的误解：力不维持速度，力改变速度（产生加速度）。F=ma 是一切后续定律（动量定理、动能定理、角动量定理）的微分源头——对时间积分得到动量定理，对空间积分得到动能定理。

What：定律的四个本质属性

  • 矢量性：a 的方向始终与合外力方向相同，不是大小关系而是矢量等式。
  • 瞬时性：F 和 a 是同一瞬时的一一对应——F 变了，a 同时改变，没有延迟。
  • 叠加性：$\vec{F}_{\text{合}} = \sum\vec{F}_i$，合力产生的加速度 = 各分力单独产生的加速度的矢量和。但力的叠加原理的成立不能自动地导致运动的叠加——若 F 是 v 的函数，每个分力单独产生的运动和合力产生的运动并不线性叠加。
  • 适用范围：仅适用于惯性系、仅适用于质点模型。

How：坐标系分量式

不同坐标系下的牛顿第二定律
坐标系分量方程最佳场景
直角坐标$F_x = ma_x,\; F_y = ma_y,\; F_z = ma_z$直线运动、斜面、平抛
自然坐标$F_t = m\frac{dv}{dt},\; F_n = m\frac{v^2}{\rho}$圆周运动、一般曲线

根据力函数的形式选择方程类型：恒力 --> $F=ma$；力是速度的函数 F$v$ --> $F(v$=m\frac{dv}{dt}\)；力是位置的函数 F$r$ --> $F(r$=m\frac{d^2r}{dt^2}\)。

力学标准解题流程（全体例题通用）：
① 选对象（单质点 or 系统？） --> ② 画受力图（隔离体法，标出全部力，不遗漏不重复） --> ③ 判断过程（直线/曲线？恒力/变力？瞬时/积累？） --> ④ 选定律（F=ma / 动量定理 / 动能定理 / 守恒定律，见对比表） --> ⑤ 列方程（写分量式，明确正方向） --> ⑥ 检查（量纲、特殊值极限、物理合理性）
易错：力的叠加原理成立 ≠ 运动的叠加原理自动成立。若力函数形式复杂（如 F=F$v$ 非线性），合力产生的运动 ≠ 各分力单独产生运动之和。

质量 m=2 kg 的物体受到两个力 $\vec{F}_1=3\hat{i}$ N、$\vec{F}_2=4\hat{j}$ N。求加速度的大小和方向。

参考答案 $\vec{F}_{\text{合}}=(3\hat{i}+4\hat{j}$\) N，$|\vec{F}_{\text{合}}|=\sqrt{3^2+4^2}=5$ N。$a=F_{\text{合}}/m=5/2=2.5$ m/s²，方向与 x 轴夹角 $\theta=\arctan(4/3$\approx 53^\circ\)。这是由「力叠加」求「加速度叠加」的直接应用。

例题：定滑轮连接体 页23页24页25 必考

两个不同质量的物体用轻绳跨过光滑定滑轮相连——哪个加速下降？加速度多大？绳中张力在两边一样吗？这是大学物理的「Hello World」级基准题。
定滑轮连接体：隔离每个物体单独列 F=ma，利用绳长不变的约束 $a_1 = a_2$（大小相等），联立得 $a = \frac{m_1 - m_2}{m_1 + m_2}g$ 和张力 $F_T = \frac{2m_1 m_2}{m_1 + m_2}g$。

标准例题（PPT E2.1$1$）

轻绳跨过光滑轴承的定滑轮，两端分别悬挂 $m_1$ 和 $m_2$（设 $m_1 > m_2$）。滑轮和绳质量不计，绳不可伸长。求加速度 a 和绳中张力 $F_T$。

所用定律：牛顿第二定律 F=ma（隔离体法）。为什么不用动量/能量？因为恒力作用求瞬时加速度——F=ma 最直接。

解题步骤：
  1. 选对象：分别隔离 $m_1$ 和 $m_2$
  2. 画受力图：$m_1$ 受 $m_1 g$（向下）、$F_T$（向上）；$m_2$ 受 $m_2 g$（向下）、$F_T$（向上）
  3. 判断过程：$m_1 > m_2$，$m_1$ 向下加速、$m_2$ 向上加速。绳不可伸长 → $a_1=a_2=a$（大小）
  4. 列方程（各自沿加速度方向为正）： $$\begin{aligned}m_1 g - F_T &= m_1 a \quad $\text{对 }m_1\text{，向下为正}$\\F_T - m_2 g &= m_2 a \quad $\text{对 }m_2\text{，向上为正}$\end{aligned}$$
  5. 联立解： $$a = \frac{m_1 - m_2}{m_1 + m_2}g,\qquad F_T = \frac{2m_1 m_2}{m_1 + m_2}g$$

张力小于较重物体的重力大于较轻物体的重力：$m_2 g < F_T < m_1 g$。

变式：电梯中的滑轮（PPT E2.1$2$）

整个装置放在以加速度 $a_0$ 上升的电梯中。物体相对电梯的加速度： $$a_r = \frac{m_1 - m_2}{m_1 + m_2}$g + a_0$,\qquad F_T = \frac{2m_1 m_2}{m_1 + m_2}$g + a_0$$ 只把 g 换成 $g+a_0$，物理本质：电梯加速上升等效于重力场变强。

变式：电梯斜面滑块（PPT E2.2）

电梯以 $a_1$ 上升，光滑斜面倾角 $\theta$，物体从长 l 的斜面顶滑到底：

物体相对斜面的加速度：$a_2 = (g + a_1$\sin\theta\)

压力（牛顿第三定律）：$N = m(g + a_1$\cos\theta\)

到底时间：$t = \sqrt{\frac{2l}{(g + a_1$\sin\theta}}\)

易错：对 $m_2$ 列方程时正方向容易取反。建议每个物体独立选正方向（沿各自加速度方向为正），不要强制统一。如果统一取向下为正，则 $m_2$ 的方程是 $m_2 g - F_T = -m_2 a$——多一个负号，容易遗漏。

牛顿第三定律与作用力-反作用力辨析 页11页15页16页17 必考

你用手掌抵住墙面用力推——墙纹丝不动，但你的手掌有强烈的被压迫感。墙「没动」不代表墙没给你反作用力。事实上，墙推你的力精确等于你推墙的力。
$\vec{F}_{A \to B} = -\vec{F}_{B \to A}$——作用力与反作用力大小相等、方向相反、沿同一直线、作用在不同物体上，同时存在同时消失，属于同一性质的力。它们绝不能互相抵消——因为作用在不同物体上。
平衡力 vs 作用力与反作用力（高频混淆点）
比较项目平衡力作用力与反作用力
作用对象同一物体两个不同物体
力的性质可以不同（重力 vs 支持力）必须相同（都是弹力/引力）
能否抵消能（合力为零，物体平衡）不能（作用对象不同，无法合成）
依赖关系不一定同时存在同时存在、同时消失
效果物体保持静止或匀速（a=0）各自使对方产生加速度
实例书放桌上：重力 + 支持力（都作用在书上）书压桌的力 ↔ 桌撑书的力

一本书静止在水平桌面上。书的「重力 mg」和「桌面对书的支持力 N」是一对作用力与反作用力吗？如果各拿一个弹簧秤挂在一起对拉，两秤的读数为什么总是相等？

参考答案 重力和支持力不是作用力与反作用力——它们都作用在同一物体（书）上，是一对平衡力。真正的反作用力是：书对桌面的压力（书→桌）↔ 桌对书的支持力（桌→书）；书的重力是地球对书的引力，其反作用力是书对地球的引力。弹簧秤对拉读数相等是因为牛顿第三定律——无论两人拉力是否相等，两秤之间的相互作用力始终等大反向。

动量、冲量与动量定理 页30页31页32页34页35页36 核心枢纽

车祸中，为什么要加装气囊？同样的速度变化（比如从 60 km/h 到 0），碰撞时间从 0.01 秒（硬碰硬）延长到 0.1 秒（有气囊缓冲），受力就降低到原来的 1/10——这不是工程直觉，是动量定理的精确预言。
动量 $\vec{p} = m\vec{v}$（矢量），冲量 $\vec{I} = \int_{t_1}^{t_2}\vec{F}\,dt$（力对时间的积分），满足 $$\vec{I} = \Delta\vec{p} = m\vec{v}_2 - m\vec{v}_1$$ 合外力的冲量等于动量的增量——即动量定理。

Why：为什么需要动量定理——牛顿第二定律还不够吗？

牛顿第二定律是瞬时关系——告诉我们某一时刻的 F 和 a 如何对应。但在碰撞、打击问题中，力在极短时间内剧烈变化，精确知道每一瞬时的力既不可能也不必要。动量定理将「力在时间上的累积效应（冲量）」与「运动状态的整体变化（动量差）」直接挂钩——我们不关心中间每一时刻的力，只关心总效果。

What：精确的数学定义

  • 动量 $\vec{p} = m\vec{v}$：矢量，单位 kg·m/s，方向 = 速度方向。
  • 恒力冲量 $\vec{I} = \vec{F}\Delta t$：方向与力相同。
  • 变力冲量 $\vec{I} = \int_{t_1}^{t_2}\vec{F}\,dt$：方向由积分结果整体决定，不能说「冲量方向与力方向一致」（力方向在过程中可能变化）。
  • 平均力：$\bar{F}_x = \frac{\int F_x dt}{\Delta t} = \frac{\Delta p_x}{\Delta t}$

分量式与核心结论

$$I_x = \int_{t_1}^{t_2} F_x\,dt = mv_{2x} - mv_{1x},\quad I_y = \int_{t_1}^{t_2} F_y\,dt = mv_{2y} - mv_{1y}$$

某方向受到冲量，该方向的动量就改变——这是分量独立性的直接推论。碰撞中冲力远大于重力（数量级差几十倍），重力常可忽略。动量定理仅对惯性系成立（由牛二导出）。

质量为 0.5 kg 的小球以 10 m/s 水平射向墙壁，以 8 m/s 反向弹回，碰撞时间 0.02 s。求墙对球的平均作用力。提示：注意速度的符号（矢量差不是标量差）。

参考答案 取反弹方向为正：$v_1 = -10$ m/s，$v_2 = +8$ m/s。$I = mv_2 - mv_1 = 0.5\times 8 - 0.5\times(-10$ = 4 + 5 = 9\) N·s。$\bar{F} = I/\Delta t = 9/0.02 = 450$ N。若误用 $0.5\times(8-10$ = -1\) N·s，相当于忽略了速度方向的矢量性质。

动量定理例题：篮球落地碰撞 页37页38 高频

一个篮球从 2 米高自由落下，以同样速率反弹——短短 0.019 秒的触地时间里，地面给了篮球多大的平均力？
篮球触地问题用动量定理：落地速度 $v = \sqrt{2gh}$，反弹速度等大反向，动量变化 $\Delta p = 2mv$。$(\bar{F} - mg$\Delta t = 2mv\)，得 $\bar{F} = \frac{2mv}{\Delta t} + mg \approx 382$ N——约为篮球自重的 66 倍。

PPT E2.8 详解

质量 m=0.58 kg，从 h=2 m 静止下落，以相同速率反弹，触地时间 $\Delta t = 0.019$ s。求地面平均作用力。

所用定律：动量定理（碰撞问题——时间极短、力复杂变化、冲力远大于重力）。

解题步骤：
  1. 落地速度：$v = \sqrt{2gh} = \sqrt{2 \times 9.80 \times 2} = 6.26$ m/s
  2. 取向上为正：触地前 $v_1 = -6.26$ m/s（向下），反弹后 $v_2 = +6.26$ m/s（向上）
  3. 动量定理：$(\bar{F} - mg$\Delta t = mv_2 - mv_1 = m$v_2 - v_1$ = 2mv\)
  4. $\bar{F} = \frac{2mv}{\Delta t} + mg = \frac{2\times 0.58 \times 6.26}{0.019} + 0.58\times 9.80$
  5. $\bar{F} = 382.2 + 5.68 \approx 3.82\times 10^2$ N

382.2 N >> 5.68 N（mg），冲力约为重力的 66 倍。在粗略估计中重力可忽略，但严格计算中必须计入重力修正（力平衡项）。

易错：初末速度大小相等，$(v_2 - v_1$\) 不是 0 而是 $2v$。错误来源在于将矢量误当标量——速度方向变了，动量方向也变了。若取向下为正，则 $v_1 = +v$，$v_2 = -v$，$\Delta p = -2mv$，数值相同。

PPT E2.9 变式：斜抛落地碰撞（分方向处理）

质点以水平速度 $v_0$ 从高度 $y_0$ 抛出，碰撞后反弹最高 $y_0/2$，水平速度变 $v_0/2$，求地面的竖直和水平冲量。

竖直冲量（向上为正）：$I_y = m\sqrt{2g\cdot y_0/2} - (-m\sqrt{2gy_0}$ = m\sqrt{gy_0}\,$1 + \sqrt{2}$\)

水平冲量：$I_x = m(v_0/2$ - mv_0 = -\frac{1}{2}mv_0\)（负号表示地面给质点的水平冲量与初速方向相反——地面摩擦减速了小球）

动量守恒定律 页39 必考

你站在光滑冰面上，抱着一块石头。把石头向前猛推出去——你自己会向后退。往前推石头之前你和石头的总动量是零；推出去之后，你后退的动量和石头前进的动量加起来——仍然精确为零。
$$\sum\vec{F}_{\text{外}} = 0 \;\Longrightarrow\; \sum m_i \vec{v}_i = \text{常数}$$ 当质点系合外力为零时，系统总动量保持不变。内力可以重新分配各质点的动量，但不能改变总动量。且动量守恒是分量式的——某方向外力为零则该方向动量的分量守恒。

Why：为什么动量守恒比逐个列 F=ma 更「高级」？

处理多体碰撞或分裂问题时，逐一隔离每个物体列 F=ma 需要知道全部内力细节——而内力往往大而复杂且我们并不关心。动量守恒定律只需判断一个条件（合外力是否为零），就可以用一个代数方程直接关联初末状态——不需要知道中间发生了什么。它是比牛顿第二定律更高层次的规律——甚至在现代物理中，即使没有「力」这个概念，动量守恒依然成立。

What：守恒条件的三种情况

  • 严格守恒：系统合外力为零 → 总动量矢量守恒（大小和方向都不变）。
  • 分量守恒：合外力在某个方向的分量为零 → 该方向动量分量守恒。如光滑水平面上的碰撞，水平动量守恒但竖直方向不守恒（有重力）。
  • 近似守恒：碰撞、爆炸过程中，内力远大于外力，极短时间内外力冲量可忽略 → 近似认为动量守恒。

比较项目动量守恒机械能守恒
守恒条件合外力为零（或某方向分量为零）只有保守力做功（非保守力不做功）
判断是对什么对系统外部的力（外力）对系统内部非保守力的功

两个小球在光滑水平面上发生碰撞。下列说法正确的是：$1$碰撞中动量一定守恒 $2$碰撞中机械能一定守恒 $3$碰撞中动能一定守恒。

参考答案 只有 $1$ 正确。光滑水平面 → 水平方向无外力 → 动量守恒。$2$$3$ 都不对：只有完全弹性碰撞机械能（动能）才守恒；一般非弹性碰撞中，部分动能转化为内能，机械能不守恒，但动量依然守恒。这是两类守恒定律最核心的区分点。

动量守恒例题：火箭分裂 + 人车反冲 页40页41页42 高频

一枚火箭在太空中以 2.5 km/s 水平飞行，突然分为前后两截。分开后仪器舱相对容器舱以 1 km/s 的水平速度向前飞出——你能算出两截各自相对地面的速度吗？
火箭分裂问题用动量守恒 + 相对速度约束：$(m_1+m_2$v = m_1 v_1 + m_2 v_2\) 和 $v_1 - v_2 = v'$，联立得 $v_2 = v - \frac{m_1}{m_1+m_2}v'$，$v_1 = v_2 + v'$。

PPT E2.11 火箭分裂

火箭以 v=2500 m/s 水平飞行，分为仪器舱 $\(m_1=100$ kg) 和火箭容器 $\(m_2=200$ kg)。仪器舱相对容器舱的水平速度为 $v'=1000$ m/s。求两舱相对惯性系 S 的速度。

所用定律：动量守恒（分裂中合外力为零）。为什么不选其他定律？ 分裂是内力驱动的分离过程，动量守恒直接给出两体速度关系，无需知道内力细节。

解题步骤：
  1. 分裂前总动量：$(m_1+m_2$v\)
  2. 分裂后总动量：$m_1 v_1 + m_2 v_2$
  3. 动量守恒：$(m_1+m_2$v = m_1 v_1 + m_2 v_2\)
  4. 相对速度约束：仪器舱在前，向前飞出 → $v_1 = v_2 + v'$
  5. 代入： $$v_2 = v - \frac{m_1}{m_1+m_2}v' = 2500 - \frac{100}{300} \times 1000 = 2167 \approx 2.17 \times 10^3\text{ m/s}$$ $$v_1 = v_2 + v' = 2167 + 1000 = 3167 \approx 3.17 \times 10^3\text{ m/s}$$
易错：相对速度约束方程的方向。仪器舱在前、容器舱在后，分裂后仪器舱应更快（向前飞出），所以用 $v_1 = v_2 + v'$。如果写反成 $v_1 = v_2 - v'$，得出的结果物理上不合理（轻的前舱反而比重后舱慢）。画示意图确认每个速度的物理方向。

PPT E2.12 人车反冲

光滑水平面上，车长 L 质量 M，车上站人质量 m，初始静止。人从车一端走到另一端，人和车各自相对地面移动多远？

所用定律：动量守恒（水平无外力，虽然人走路脚底有摩擦，但那是人与车之间的内力——不影响总动量守恒）。关键理解：人走路靠车给人的摩擦力（内力），这个力让人加速、同时让车反向加速——总质心不动。

解题步骤：
  1. 设人相对地面速度为 v，车相对地面速度为 V（反向）：$0 = mv - MV$
  2. 积分：$\int mv\,dt = \int MV\,dt$ → $mx = MX$
  3. 几何约束（人相对车走了 L）：$x + X = L$
  4. 解得：$$X = \frac{m}{M+m}L,\quad x = \frac{M}{M+m}L$$

车退的距离与人的质量成正比：人越重（m 越大），车退得越多。若人极轻 m→0，车几乎不动（X→0），人走 L。若人极重 m→∞，车退 L，人几乎不动。

功与动能定理 页43页44页45页46页47页48 必考

推箱子在地板上走——推力做正功，摩擦力做负功。推力让箱子加速，摩擦力让它停下。力和位移的「点积」把力的空间效果量化成一个标量——这就是「功」。
功 $W = \int_a^b \vec{F} \cdot d\vec{r}$ 是力对空间的累积效应（标量，只有正负），而 $$W_{\text{合}} = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$ 合外力的功 = 动能的增量。这是功与运动状态变化的定量桥梁。

Why：功和动能定理填补了「力对空间累积」的空白

动量定理 --> 「力持续了多久改变多少动量」
动能定理 --> 「力推了多远改变多少动能」
两个定理互补——同一道力学题可能需要分方向用动量定理、分能量用动能定理。

What：功的完整定义

  • 恒力功：$W = F\,|\Delta\vec{r}|\,\cos\theta = \vec{F} \cdot \Delta\vec{r}$
  • $\theta = 0^\circ$：正功最大（力与位移同向）；$0^\circ<\theta<90^\circ$：正功（推着走）
  • $\theta = 90^\circ$：不做功（向心力，瞬时与位移垂直）
  • $90^\circ<\theta<180^\circ$：负功（摩擦力、阻力）
  • 变力功：微元积分： $$W_{ab} = \int_a^b \vec{F} \cdot d\vec{r} = \int_a^b $F_x dx + F_y dy + F_z dz$$ 自然坐标：$W_{ab} = \int_a^b F_t\,ds$

四个关键注意点

  • 功是标量——只有正负（+/-），没有方向
  • 功是过程量——与路径有关（保守力例外）
  • 功与参考系有关——位移依赖于参考系选择
  • 质点系动能定理：$W_{\text{外}} + W_{\text{内}} = \Delta E_k$——内力可以改变系统总动能！
易错：内力合力为零 ≠ 内力功之和为零。一对作用力与反作用力等大反向，但若两质点的位移不同，这对力的总功可以不为零。典型例子：炸弹爆炸——弹片相互推开，内力做正功，把化学能转化为弹片的动能。用力为零但位移不同 → 功的代数和 ≠ 零。

人匀速提起 10 kg 箱子到 2 m 高度。求人的拉力和重力各做多少功？合力做了多少功？这验证了什么？

参考答案 匀速 → $F_{\text{拉}} = mg = 98$ N。$W_{\text{拉}} = 98 \times 2 = 196$ J（正功），$W_G = -mg \times 2 = -196$ J（负功——重力方向与位移相反）。$W_{\text{合}} = 196 + (-196$ = 0\)。合力做零功 ↔ 动能不变（$\Delta E_k = 0$，匀速），验证动能定理。人消耗了生物能，转化为重力势能——但这涉及非保守内力（人体肌肉），动能定理只描述了质点动能变化。

动能定理例题：钉入木头 + 链条滑落 页49页50页51 高频

一锤子把铁钉打入木板 1 cm。再用同样的力量打第二次——铁钉还能再进 1 cm 吗？直觉可能说「是」，但木板阻力随深度增大，变力做功告诉你的答案是：只多进约 0.41 cm。
阻力与深度成正比（$f = kx$）时，变力做功非线性。第一次做功 $W = \frac{1}{2}kx_0^2$，第二次做功相同但积分区间从 $x_0$ 到 $x$——$x = \sqrt{2}x_0$，增量仅 $\Delta x \approx 0.414$ cm。

PPT E1：锤击铁钉（变力做功 + 动能定理）

木板对铁钉的阻力与进入深度成正比（$f = kx$）。第一次打入 $x_0 = 1$ cm。同等锤击力下第二次还能打入多深？

所用定律：动能定理。锤击使钉子获得初动能 $\frac{1}{2}mv^2$，阻力做负功消耗动能直至速度为零。

解题步骤：
  1. 第一次：$\frac{1}{2}mv^2 = \int_0^{x_0} kx\,dx = \frac{1}{2}kx_0^2$
  2. 第二次：同样初动能，钉子从 $x_0$ 继续深入至 $x$： $$\frac{1}{2}kx_0^2 = \int_{x_0}^x kx\,dx = \frac{1}{2}k$x^2 - x_0^2$$
  3. 解得 $x^2 = 2x_0^2$，$x = \sqrt{2}x_0$
  4. 第二次打入深度：$\Delta x = x - x_0 = (\sqrt{2}-1$x_0 \approx 0.414\) cm

物理直觉：阻力随深度线性增大，越深越难打。同样的锤击能量，后面每次打入的深度递减。

PPT E2：链条从桌面滑落（变力做功 + 动能定理联用）

链条长 l 质量 m，部分长度 a 下垂桌外，桌面摩擦系数 μ，从静止释放。求全部离开桌面时的速度。

所用定律：动能定理（有变力功和摩擦力功）。为什么不只用机械能守恒？因为有摩擦力（非保守力），机械能不守恒，只能用动能定理。

解题步骤：
  1. 摩擦力功（变力，随桌上链条长度变化）：$W_f = -\int_a^l \frac{\mu mg}{l}(l-x$\,dx = -\frac{\mu mg$l-a$^2}{2l}\)
  2. 重力功（下垂部分越来越多）：$W_G = \int_a^l \frac{mg}{l}x\,dx = \frac{mg(l^2-a^2$}{2l}\)
  3. 动能定理（$v_0 = 0$）：$W_G + W_f = \frac{1}{2}mv^2$
  4. $v = \sqrt{\frac{g}{l}\left1455L55}\)
易错：摩擦力功的积分变量。链条从初始下垂长度 a 运动到全部离开（下垂 l），桌上链条长度从 l-a 减到 0。积分变量是下垂长度 x，桌上部分对应的正压力为 $\frac{mg}{l}(l-x$\)，不要写错为 $\frac{mg}{l}(a-x$\)。

保守力与势能 页52页53页54页55页56页57 必考

爬山时，走盘山缓坡还是直上石阶——你出的汗不一样，但重力做的功精确相等。因为重力不管你走了什么路线，它只关心起点和终点的高度差。这就是「保守」的含义。
保守力沿任意闭合路径做功为零 $\oint\vec{F}_{\text{保}} \cdot d\vec{r} = 0$，做功只与始末位置有关，与具体路径无关。由此可定义势能：$E_p(a$ = \int_a^{\text{零点}} \vec{F}_{\text{保}} \cdot d\vec{r}\)，保守力做正功 = 势能减少。

常见例子重力、弹力（弹簧）、万有引力摩擦力、爆炸力、空气阻力

势能的本质理解

  • 势能是系统属性：重力势能 mgh 属于「物体 + 地球」系统，不能说「物体的势能」。弹簧势能属于「物体 + 弹簧」系统。
  • 势能绝对值无意义：只有势能差有意义（与零点选取无关）。「大石头在地上不可怕，在房顶上才可怕」——怕的是势能差。
  • 保守力做正功 → 势能减少；保守力做负功 → 势能增加。
  • 保守力的功 = 势能的减少量：$W_{\text{保}} = E_{p1} - E_{p2} = -\Delta E_p$
易错：不是所有「力与位置有关」的力都是保守力。摩擦力的大小和方向可以随位置变化，但它做功总与路径有关、不能定义势能。保守力的本质判据是封闭路径积分为零，而非简单看力是否含位置变量。

判断：$1$ 摩擦力永远不能做正功。$2$ 只有保守力才能做功。$3$ 摩擦力是非保守力，所以摩擦力做的功转化为热。

参考答案 $1$ 错误——摩擦力可以做正功。例：传送带上的物体被静摩擦力加速前进，静摩擦力对物体做正功。$2$ 错误——任何力都可以做功。$3$ 正确——非保守力做功不存储为势能，而是耗散为热等不可逆形式。但要注意：静摩擦力是保守的吗？不是——静摩擦力不做功（无相对位移），做正功的是动摩擦力吗？也不是——静摩擦力有做功的情况（如汽车加速时地面静摩擦力对轮胎做正功），但静摩擦力的功不存储为势能。

机械能守恒定律 页58页59 必考

一个小球从光滑圆弧面顶端滑下——越滑越快（势能转动能），到达最低点时速度最大，然后沿另一侧弧面上升——越升越慢（动能转回势能）。如果弧面始终光滑，它会升到和出发高度完全相同的位置停下来——能量一分没少。
$$E_k + E_p = \text{常数}$$ 当系统满足所有外力不做功 + 所有非保守内力不做功时，系统的总机械能（动能 + 全部势能）保持不变。动能和势能之间可以互相转化，但总量不变。

Why：机械能守恒的条件为什么比动量守恒更严？

动量守恒：合外力为零即可——内力怎么闹（摩擦、爆炸、弹力）都行。
机械能守恒：不仅要外力不做功，连非保守内力也不能做功——内部摩擦、塑性形变都不许出现。
这就是为什么碰撞中动量总是守恒的（无外力时），而机械能只在完全弹性碰撞中守恒。

守恒条件的精确表述

当 $W_{\text{外}} = 0$ 且 $W_{\text{非保内}} = 0$ 时： $$E = \frac{1}{2}mv^2 + mgh + \frac{1}{2}kx^2 - \frac{GMm}{r} = \text{常数}$$ 能量转换链：下落：$mgh \to \frac{1}{2}mv^2$（势能→动能）｜上升：$\frac{1}{2}mv^2 \to mgh$（动能→势能）｜弹簧压缩-释放：$\frac{1}{2}mv^2 \to \frac{1}{2}kx^2 \to \frac{1}{2}mv^2$

易错：机械能守恒是对整个系统、对整个过程的每一时刻而言的——不是只看初末状态就够了。只比较始末态能量相等不能推出机械能一定守恒（可能中途有非保守力做过功并恰好抵消）。

PPT E4：光滑木钉上的链条滑落

长 2l 的匀质链条平衡悬挂在光滑圆柱形木钉上，从静止滑动。求链条离开木钉时的速度。

所用定律：机械能守恒（光滑 → 无摩擦 → 无非保守力做功；无其他外力的功）。为什么不用动量定理？ 链条运动过程中各部分的位移和速度不一致（上面下降、下面上升），质心位置在移动，用能量法处理最自然。

解题步骤：
  1. 线密度 $\lambda$，总质量 $m = 2\lambda l$
  2. 取 O 点（木钉）为势能零点。初始：链条对称，质心在 O 下方 l/2，$E_{p0} = -2(\lambda l$g \cdot $l/2$ = -\lambda l^2 g\)
  3. 末态：链条竖直下垂，质心在 O 下方 l，$E_p = -(\lambda\cdot 2l$g \cdot l = -2\lambda l^2 g\)
  4. 机械能守恒：$E_{p0} = E_p + E_k$ → $-\lambda l^2 g = -2\lambda l^2 g + \frac{1}{2}(2\lambda l$v^2\)
  5. 解得：$v = \sqrt{lg}$

质量 m 的小球从半径为 R 的光滑半圆轨道顶端由静止滑下。求：$1$到达最低点的速度；$2$轨道对小球的支持力。

参考答案 $1$ 机械能守恒（支持力始终垂直于速度不做功）：$mgR = \frac{1}{2}mv^2$ → $v = \sqrt{2gR}$。
$2$ 最低点牛二（向心力）：$N - mg = m\frac{v^2}{R}$ → $N = mg + m\frac{2gR}{R} = 3mg$。最低点支持力是重力的 3 倍——这是过山车/荡秋千最底端感到「超重」的物理原因。
