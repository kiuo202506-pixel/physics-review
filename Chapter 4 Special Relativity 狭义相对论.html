Chapter 4: Special Relativity狭义相对论 -- 当光速不变时，时空不再绝对

想象你站在地面上，一列火车以 0.5c 疾驰而过。车厢正中央的灯同时向车头和车尾射出两束光。车上的乘客看到光同时到达两端 -- 但地面上的你却看到光先到达了车尾。

同一件事，两个观察者看到的时间顺序不同。这不是错觉，而是宇宙的真实规则。爱因斯坦在 1905 年用这个思想实验改写了人类对时空的所有直觉。

更震撼的实验事实：宇宙线产生的 μ 子以 0.998c 从天而降。按经典物理，它静止寿命只有 2.2×10-6 s，飞不到 700 米就该衰变了。但它确实飞了 10 km 到达地面 -- 因为高速运动让它「活得更久」。这每天都在大气层中发生的现象，就是相对论的时间膨胀。

狭义相对论建立在两条基石之上：物理定律在所有惯性系中形式相同（相对性原理）；真空中光速 c = 3×10⁸ m/s 对所有惯性系观测者都一样（光速不变原理）。由此导出的后果：同时性是相对的、运动时钟变慢（Δt = γΔt₀，动钟变慢）、运动物体在运动方向上缩短（L = L₀/γ，动尺变短）、速度叠加有分母约束从不超光速、质量随速度增大（m = γm₀）、E = mc²。一切效应的核心是 γ = 1/√$1 − v²/c²$ —— v ≪ c 时 γ ≈ 1 回到经典世界，v → c 时 γ → ∞ 进入极端相对论。

Why：为什么要推翻经典时空观？

在 19 世纪末，经典物理学大厦似乎已经完美：牛顿力学、麦克斯韦电磁学、热力学三大体系相互独立又彼此融洽。但有一个裂缝：麦克斯韦方程组预言的光速 c = 3×10⁸ m/s 究竟是相对哪个参考系的？

如果光速遵从伽利略速度叠加（v' = v − u），那么测量到的光速应该随地球运动方向而变化。但 Michelson-Morley 实验反复证明：无论你怎么测、往哪个方向测，光速始终是 c，与地球运动无关。这是一个与经典直觉剧烈冲突的判决性实验结果。

爱因斯坦的选择不是修补旧理论，而是重新定义时空本身。他把「光速不变」这令人困惑的实验事实直接升格为公理，然后问：如果光速确实对所有观察者都一样，那时间、空间、同时性必须是什么样子的？答案就是狭义相对论。

What：狭义相对论的核心内容

一、从经典时空到相对论

页3页4页5页6页7页8经典绝对时空观与伽利略变换 核心

在牛顿力学中，时空是绝对且彼此独立的背景舞台。经典时空观的三大支柱：

  • 绝对同时性 -- S 系中两事件同时发生（t₂ − t₁ = 0），在 S' 系中也同时（t'₂ − t'₁ = 0）。同时与参考系无关。
  • 绝对时间间隔 -- Δt' = Δt，全宇宙的钟滴答同步。无论谁测同一段时间，结果都一样。
  • 绝对空间间隔 -- Δd' = Δd，尺子长度与谁测量无关。空间与时间彼此独立，互不相关。

设惯性系 S' 相对 S 沿 +x 轴以速度 u 匀速运动，t = t' = 0 时两原点重合。伽利略变换给出：

$$x' = x - ut,\quad y' = y,\quad z' = z,\quad t' = t$$

其中 u 是 两参考系的相对速度。t' = t 这一条就是「时间绝对」的数学表达。对其求导得到速度叠加 v' = v − u，加速度不变 a' = a——这正是牛顿第二定律 F = ma 在所有惯性系中形式相同的原因。

但麻烦来了：麦克斯韦方程组在伽利略变换下不保持形式不变；更致命的是，Michelson-Morley 实验证明光速与参考系无关，但伽利略变换却预言光速应为 c ± u -- 直接矛盾。物理学需要新的时空观。

易错：伽利略变换中 t' = t 意味着「同时性在所有惯性系中都是绝对的」——这是我们日常直觉的来源，但在高速世界是完全错误的。记住相对速度 u 的符号方向：S' 沿 +x 相对 S 运动，S→S' 用 x' = x − ut；S'→S 用 x = x' + ut'。代错符号会让整个结果反向。
经典时空观下，「同时」「时间间隔」「空间间隔」三者各自是绝对的还是相对的？
参考答案 三者全部绝对，且与参考系选择无关，时间与空间彼此独立——这就是「绝对时空观」。狭义相对论将彻底推翻这一点。

页9页10页11狭义相对论的两条基本假设 必考

1905 年，爱因斯坦提出两条假设，彻底重塑了物理学的时空根基：

1. 相对性原理（Principle of Relativity）
物理定律在所有惯性参考系中具有相同的数学形式。所有惯性系都是等价的。
这是伽利略相对性原理的推广——从「力学定律」推广到「所有物理定律」（包括电磁学、光学）。没有任何惯性系是特殊的，不存在「绝对静止」参考系。

2. 光速不变原理（Principle of Invariance of Light Speed）
真空中的光速 c = 3×10⁸ m/s 在所有惯性系中都是相同的，与光源和观察者之间的相对运动无关。
这意味着不管你以多快的速度追光，光对你来说永远是 c。你以 0.9c 追光，你测到的光速仍然是 c，不是 0.1c。这是反直觉的核心。

易错：别把「相对性原理」和「光速不变原理」搞混。相对性原理是「定律形式相同」——不限于力学、推广到所有物理；光速不变是「光速数值相同」——无论光源和观察者怎么运动。两者独立，缺一不可。另外，光速不变不是说「光速恒为 c」就结束了，而是光速在任何惯性系中都一样——这个「都」字是关键。
光速不变原理和伽利略速度叠加规则冲突在哪里？爱因斯坦是如何解决这个矛盾的？
参考答案 按伽利略规则，观测者以速度 u 运动时测到的光速应为 c ± u，但光速不变原理要求始终为 c。爱因斯坦的解决方案是：不是改光速，而是改时空——时间不再是绝对的，长度不再是绝对的。洛伦兹变换取代伽利略变换，保证光速在任何惯性系中都是 c。

页16页17页18γ 因子——解读所有相对论公式的钥匙 必考

在所有相对论公式中，有一个因子重复出现——它就是 洛伦兹因子 γ（Lorentz factor）：

$$\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$$

其中 v 是 两个参考系的相对速度，c 是光速。γ 决定了相对论效应有多强。理解 γ 的行为 = 理解整个相对论。

γ 的物理直觉：

  • v ≪ c（日常速度）→ γ ≈ 1，所有相对论效应几乎消失，回到牛顿世界。汽车 100 km/h ≈ 0.000000093c → γ ≈ 1.000000000000004
  • v = 0.6c → γ = 1/√$1−0.36$ = 1/√0.64 = 1.25
  • v = 0.8c → γ = 1/√$1−0.64$ = 1/√0.36 = 5/3 ≈ 1.667
  • v = 0.99c → γ = 1/√0.0199 ≈ 7.09，效应已非常显著
  • v = 0.998c（μ 子的速度） → γ ≈ 15.8，寿命延长近 16 倍
  • v → c → γ → ∞，时间无限膨胀、长度缩为零——有质量物体永远不可能达到光速

γ ≥ 1 恒成立 —— 这是解读所有相对论公式的关键：

  • 时间膨胀：Δt = γ Δt₀ → Δt ≥ Δt₀，「动钟变慢」意味着运动参考系测出的时间间隔更大
  • 长度收缩：L = L₀ / γ → L ≤ L₀，运动物体在运动方向上变短
  • 相对论质量：m = γ m₀ → m ≥ m₀，物体越运动质量越大
易错：很多同学记混 Δt 和 L 的公式方向——Δt = γΔt₀（乘 γ，时间间隔变大），L = L₀/γ（除 γ，长度变小）。记忆诀窍：带着「动」的那个量（动钟、动尺）是「被压缩」的一方——动钟走慢意味着同样的变化需要更长的间隔读数（乘 γ）；动尺变短意味着长度值变小（除 γ）。γ 永远 ≥ 1，所以乘它的变大、除它的变小。
当 v = 0.8c 时，γ 是多少？解释为什么 v = 0.99c 时 γ 急剧增大而 v = 0.01c 时 γ 几乎等于 1。
参考答案 v = 0.8c：γ = 1/√$1−0.64$ = 1/0.6 = 5/3 ≈ 1.667。
v = 0.99c：γ = 1/√$1−0.9801$ = 1/√0.0199 ≈ 7.09。
v = 0.01c：γ = 1/√$1−0.0001$ = 1/√0.9999 ≈ 1.00005 ≈ 1。
根源：v²/c² 在 v 较小时非常小，分母 √$1−v²/c²$ ≈ 1；当 v 接近 c 时，分母接近 0，γ 急剧增大（以 1/√$1−v²/c²$ 的形式发散）。

二、洛伦兹变换与时空效应

页14页15页16页17页18洛伦兹变换 必考

当光速不变和相对性原理同时成立时，连接两个惯性系的不是伽利略变换，而是 洛伦兹变换：

正变换（S → S'，S' 相对 S 以速度 u 沿 +x 运动）：

$$x' = \gamma$x - ut$,\quad y' = y,\quad z' = z,\quad t' = \gamma\left$t - \frac{u}{c^2}x\right$$

逆变换（S' → S，只需把 u 换成 −u）：

$$x = \gamma$x' + ut'$,\quad y = y',\quad z = z',\quad t = \gamma\left$t' + \frac{u}{c^2}x'\right$$

这意味着什么？

  • 时间和空间不再独立——t' 依赖于 x，x' 依赖于 t。时空是统一的四维整体。
  • 当 u ≪ c 时，γ → 1 且 u/c² → 0，洛伦兹变换退化为伽利略变换：x' = x − ut，t' = t。相对论包含经典力学为低速特例。
  • u ≥ c 时 γ → ∞ 或虚数——意味着任何物体速度不能超过 c。光速是宇宙的绝对速度上限。
  • 洛伦兹变换的设计保证了光速不变：如果 S 中光速为 c，代入变换后 S' 中也是 c。
对比维度伽利略变换洛伦兹变换
坐标 x'x' = x − utx' = γ$x − ut$
时间 t't' = t（独立于空间）t' = γ$t − ux/c²$（与空间耦合）
速度叠加v' = v − u（无上限）v' = $v − u$/$1 − uv/c²$（上限 c）
适用范围v ≪ c 的低速世界所有 v < c 的运动
时空关系时空独立时空统一
速度上限无c（光速）
易错：洛伦兹变换的正逆变换中，符号变化规则——正变换是 x' = γ$x − ut$，t' = γ$t − ux/c²$；逆变换把 u 改成 −u：x = γ$x' + ut'$，t = γ$t' + ux'/c²$。不要在使用 u 符号和一撇的归属上搞混。另外 γ 的分母是 √$1 − u²/c²$，不是 1 − u/c！
已知 S' 相对 S 以 0.8c 沿 +x 运动。S 中一事件坐标为 $x=0, t=1 s$，求 S' 中的坐标（x', t'）。
参考答案 γ = 1/√$1−0.8²$ = 1/√0.36 = 1/0.6 = 5/3 ≈ 1.667
x' = γ$x − ut$ = $5/3$$0 − 0.8c×1$ = −$4/3$c ≈ −4×10⁸ m
t' = γ$t − ux/c²$ = $5/3$$1 − 0.8c×0/c²$ = 5/3 ≈ 1.667 s
在 S 系中该事件在原点发生（x=0），在 S' 系中它在负方向 −$4/3$c 处发生——时空坐标互相转化！

爱因斯坦火车——同时性不是绝对的

一列火车以速度 u 匀速行驶（S' 系）。车厢正中间 O 处有一盏灯。当 O 与地面上的观测者重合时（t = t' = 0），灯发出闪光。

S' 系（火车上）：光以相同速度 c 向两端传播，A 和 B 到 O 距离相等 → 光同时到达车头 A 和车尾 B → 两事件同时发生。

S 系（地面）：光速对地面也是 c！但 B（车尾）迎着光运动（光源向它靠近）、A（车头）远离光运动（光源离它远去）→ 光先到达 B 后到达 A → 两事件不同时！

结论：同时性是相对的。在一个惯性系中异地同时发生的两个事件，在另一个惯性系中一般不同时。

从洛伦兹变换看：若 S 系中异地同时（Δx ≠ 0，Δt = 0），则

$$\Delta t' = \gamma\left$\Delta t - \frac{u}{c^2}\Delta x\right$ = -\gamma\frac{u}{c^2}\Delta x \neq 0$$

只有同时且同地（Δx=0 且 Δt=0）的事件，在所有惯性系中才都同时。

易错：虽然同时性是相对的，但因果关系不会颠倒。如果事件 A 是事件 B 的原因，信号传递速度 ≤ c，则在任何惯性系中 A 都先于 B 发生。数学上因为 |Δx/Δt| ≤ c < c/$u/c$ = c²/u，所以 Δt' 的符号与 Δt 始终相同。因果不倒，时间不倒流。

时间膨胀——动钟变慢

设 S' 系中同一位置（Δx' = 0）发生两个事件，时间间隔为 Δt₀。这个在 同一位置 测得的时间间隔称为 固有时（proper time）Δτ——事件发生地的钟测出的时间间隔。

在 S 系中测得的时间间隔 Δt 为：

$$\Delta t = \frac{\Delta t_0}{\sqrt{1 - u^2/c^2}} = \gamma \Delta t_0$$

因为 γ ≥ 1，所以 Δt ≥ Δt₀——运动参考系测出的时间间隔大于固有时。这就是「动钟变慢」：对地面观察者来说，运动物体上的钟走得慢。固有时间隔是最短的时间间隔。

μ 子验证（实验事实！）：

  • μ 子静止寿命 τ₀ = 2.2×10⁻⁶ s（在 μ 子自身参考系中是固有时）
  • 以 v = 0.998c 飞行：γ = 1/√$1−0.998²$ ≈ 15.8
  • 地面测得的寿命：Δt = γτ₀ ≈ 15.8 × 2.2×10⁻⁶ = 3.5×10⁻⁵ s
  • 能飞行的距离：s = vΔt = 0.998 × 3×10⁸ × 3.5×10⁻⁵ ≈ 10,420 m > 8000 m
  • μ 子能到达地面！ 地面实际测到 μ 子流率约 500 m⁻²/s，完美验证了时间膨胀。

半人马座 α 星之旅：飞船以 0.999c 飞向 4.3 光年外的半人马座 α 星。

  • 地球测的旅程时间：Δt = 4.3 ly / 0.999c ≈ 4.55 年（出发和到达不在同一地点 → 不是固有时）
  • 飞船测的旅程时间（固有时）：Δt₀ = Δt/γ = 4.55 × √$1−0.999²$ ≈ 0.203 年 ≈ 74 天
  • 时间膨胀让在有生之年进行星际航行成为可能！
易错：
① 固有时 Δt₀ 是在两事件同一地点发生的那个参考系中测得的时间间隔，不是「静止的钟」——是和事件一起静止的钟。判断方法：问自己「这两个事件在哪个参考系中是同一地点发生的？」
② Δt = γΔt₀，γ ≥ 1 所以 Δt ≥ Δt₀ —— 运动参考系测出的时间间隔更大。不要把「动钟变慢」理解成数值也变小了！动钟走慢意味着跑一圈的间隔读数被拉长 → 数值更大。
③ 时间膨胀是相对的——S 系的人觉得 S' 的钟慢了，S' 系的人也觉得自己看到 S 的钟慢了。这看起来矛盾，但因为在不同参考系中「同时」本身就不同，所以逻辑自洽。
④ 时间膨胀和长度收缩不能同时用于同一个对象——哪个量在哪个系是「原」的，就决定用哪个公式。
飞船以 0.6c 飞行，船上钟测得某过程历时 10 s。地面观察者测得这个过程历时多久？哪个是固有时？
参考答案 过程发生在飞船上同一地点，故船上的 10 s 是固有时 Δt₀。
γ = 1/√$1−0.36$ = 1/0.8 = 1.25。
地面测得 Δt = γΔt₀ = 1.25 × 10 = 12.5 s，比固有时长——地面看到飞船上的过程变慢了。
μ 子实验中，为什么按经典力学算 μ 子到不了地面（只能飞约 660 m），按相对论算却能到？
参考答案 经典力学认为 μ 子寿命固定为 2.2×10⁻⁶ s，飞行距离 = 0.998c × 2.2×10⁻⁶ ≈ 660 m < 8000 m，到不了地面。
相对论中，地面系测得 μ 子寿命被 γ ≈ 15.8 放大：Δt = γτ₀ = 3.5×10⁻⁵ s，飞行距离 ≈ 10420 m > 8000 m，能到达地面。实验测到地面 μ 子流率，完美验证时间膨胀。

页23页24页25长度收缩——动尺变短 必考

一根杆静止在 S' 系中沿 x' 轴放置，其 固有长度（proper length） L₀ = x'₂ − x'₁ 是在杆静止的参考系中测得的长度——这是杆最长的长度。

在 S 系中测量这根运动的杆，必须 同时 记录杆两端的位置（Δt = 0）：

$$L = L_0\sqrt{1 - u^2/c^2} = \frac{L_0}{\gamma}$$

因为 γ ≥ 1，所以 L ≤ L₀——固有长度最长，运动物体在运动方向上缩短了。

关键限制：长度收缩只在运动方向上发生！垂直于运动方向（y、z）的长度不变。这意味着：

  • 一根斜放的杆，只有 x 分量收缩，y 分量不收缩 → 角度也变了！
  • 运动物体的体积会变化，密度会变化
  • 一个球以接近光速运动时，看起来像被压扁的椭球（在运动方向上变扁），但垂直于运动方向看起来仍然是圆的

μ 子的故事也可以用长度收缩来解读：在 μ 子自身的参考系中，它寿命还是 τ₀，但迎面冲来的 8000 m 大气层被压缩为 L = 8000/γ ≈ 507 m，μ 子只需飞 507 m，当然能到达地面。时间膨胀和长度收缩是同一物理现象的两种表述方式。

易错：
① 固有长度 L₀ 是物体静止参考系中测的长度，它最长。运动长度 L = L₀/γ 总是更短。
② 长度收缩只在运动方向，不是三维都缩！y 和 z 方向不变。一根斜杆的角度在不同参考系中会不同。
③ 测量运动物体的长度必须同时记录两端位置——这是操作定义。但在另一参考系中「同时」本身就不同，所以测出的长度也不同。长度收缩的根源正是同时性的相对性。
④ 时间是「乘 γ 放大」（Δt = γΔt₀），长度是「除 γ 缩小」（L = L₀/γ）——两者方向相反，别套混！
固有长度 1 m 的尺子沿长度方向以 0.8c 运动，地面测得它多长？
参考答案 γ = 1/√$1−0.64$ = 1/0.6 = 5/3。
L = L₀/γ = 1 / $5/3$ = 0.6 m。比固有长度短了 40%。
一根 1 米长的杆静止在 S' 中，与 x' 轴成 30°。S 系中测量杆与 x 轴成 45°，求 S' 相对 S 的速度 u 和 S 系中杆的长度。
参考答案 y 方向无收缩：tan30° = Δy'/Δx'，tan45° = Δy/Δx = Δy'/Δx。
故 Δx/Δx' = tan30°/tan45° = 0.577。
由长度收缩：Δx = Δx'√$1−u²/c²$ → √$1−u²/c²$ = 0.577 → u = 0.816c。
杆在 S 系中长度：L = L₀·$sin30°/sin45°$ = 1×$0.5/0.707$ = 0.707 m。

页40页41页42页43页44页45相对论速度叠加——不超光速 核心

经典直觉：火箭以 0.8c 飞行，火箭又向前发射 0.5c 的子弹，子弹相对地面的速度 = 0.8c + 0.5c = 1.3c？错！

相对论速度叠加（S' 相对 S 以速度 u 沿 +x 运动）：

$$\upsilon_x = \frac{\upsilon'_x + u}{1 + \frac{u\upsilon'_x}{c^2}},\qquad \upsilon'_x = \frac{\upsilon_x - u}{1 - \frac{u\upsilon_x}{c^2}}$$

y 和 z 方向也有修正（因为 dt 和 dt' 不同）：

$$\upsilon_y = \frac{\upsilon'_y \sqrt{1 - u^2/c^2}}{1 + \frac{u\upsilon'_x}{c^2}}$$

验证火箭射子弹：u = 0.8c，υ'ₓ = 0.5c

$$\upsilon_x = \frac{0.5c + 0.8c}{1 + \frac{0.8c \cdot 0.5c}{c^2}} = \frac{1.3c}{1.4} = 0.928c$$

0.928c，不是 1.3c！ 分母的修正项 uv'/c² 就是保证不超光速的机制。

验证光速自洽：令 υ' = c，υ = $c+u$/$1+u/c$ = c·$c+u$/$c+u$ = c——光速叠加任何速度还是 c。完美！

易错：
① 速度叠加公式的分母永远不要忘了！经典 0.8c + 0.5c = 1.3c 是错的。
② y、z 方向的速度变换有额外的 √$1−u²/c²$ 因子——因为时间膨胀影响了 dt。
③ 地面观测的两火箭接近速率 1.25c 不是任何物体的运动速度——它是同一参考系中两个物体坐标差的变化率，可以超过 c，不违反相对论。「相对速度」（用速度叠加公式算）和「接近速率」（同一参考系中的速度差）是两个不同的概念。
原子以 0.5c 远离观察者，沿运动方向射出相对原子 0.8c 的电子。求电子相对观察者的速度。为什么不是 1.3c？
参考答案 u = 0.5c，υ'ₓ = 0.8c。
υₓ = $0.8c + 0.5c$/$1 + 0.5×0.8×c²/c²$ = 1.3c/1.4 = 0.928c。
不是 1.3c 的分母修正项 $1 + uυ'/c²$ 保证结果永远不超过 c。经典相加只在 v ≪ c 时近似成立。
两艘火箭在地面观测中分别以 0.5c 和 0.75c 相向飞行。求 B 相对于 A 的速度。地面观测到的接近速度是多少？接近速度是否违反相对论？
参考答案 S（地面）：u_A = 0.5c，υ_B = −0.75c。
S'（A）：υ' = $−0.75c − 0.5c$/$1 − 0.5×(−0.75$) = −1.25c/1.375 = −0.91c。
B 相对 A 的速度为 0.91c，不超光速！
地面测的接近速度 = 0.5c + 0.75c = 1.25c，这是两个物体在同一参考系中各自速度的算术和，不是任何一个物体的运动速度，不违反相对论。

三、相对论动力学：E = mc²

页47页48页49页50页51页52页53页54页55页56页57页58相对论质量、动量与质能关系 必考

质量随速度变化

从完全非弹性碰撞+动量守恒+速度叠加公式可严格推出，要保持动量守恒，质量必须随速度变化：

$$m = \frac{m_0}{\sqrt{1 - \upsilon^2/c^2}} = \gamma m_0$$

其中 m₀ 是 静止质量（rest mass），m 是 相对论质量。物体运动越快，质量越大。要加速一个已接近光速的物体，需要越来越大的力（因为质量在增大）。这就是为什么有质量物体永远达不到光速——需要无穷大的能量。

相对论动量与动力学方程

$$\vec{p} = m\vec{\upsilon} = \frac{m_0\vec{\upsilon}}{\sqrt{1 - \upsilon^2/c^2}} = \gamma m_0\vec{\upsilon}$$

$$\vec{F} = \frac{d\vec{p}}{dt} = \frac{d$m\vec{\upsilon}$}{dt} = m\frac{d\vec{\upsilon}}{dt} + \vec{\upsilon}\frac{dm}{dt}$$

注意：m 随速度变化，所以 F = m dv/dt + v dm/dt，不再是简单的 F = ma。

相对论动能——不是 ½m₀v²！

从动能定理 E_k = ∫F·dr 严格积分得到：

$$E_k = mc^2 - m_0c^2 = $\gamma - 1$m_0c^2$$

当 υ ≪ c 时做泰勒展开：

$$E_k \approx \frac{1}{2}m_0\upsilon^2 + \frac{3}{8}m_0\frac{\upsilon^4}{c^2} + \cdots$$

低速极限的首项就是经典动能 ½m₀v²！这是理论自洽性的重要检验。

E = mc²——质能关系

$$\boxed{E = mc^2}$$

物理学最著名的方程。质量为 m 的物体具有总能量 E = mc²。

  • 静能（rest energy）：E₀ = m₀c²，即使物体静止不动也拥有巨大能量
  • 总能：E = E₀ + E_k = mc² = γ m₀c²，静能+动能

核反应与质量亏损

核反应前后的能量守恒：

$$m_{01}c^2 + E_{k1} = m_{02}c^2 + E_{k2}$$

释放的能量：

$$\Delta E = E_{k2} - E_{k1} = $m_{01} - m_{02}$c^2 = \Delta m_0 c^2$$

其中 Δm₀ = m₀₁ − m₀₂ 是 静质量亏损（mass defect）。

裂变实例（PPT E13）：1 mol ²³⁵U 裂变质量亏损 Δm = 0.215 g

$$\Delta E = 0.215 \times 10^{-3} \times $3 \times 10^8$^2 = 1.935 \times 10^{13}\ \text{J}$$

相当于 625 吨无烟煤燃烧的热量！这就是核能的来源——微小的质量亏损乘以 c² 就是巨大的能量。

聚变实例（PPT E14）：质子+中子→氘核，Δm₀ = 3.97×10⁻³⁰ kg，放出 ΔE ≈ 3.57×10⁻¹³ J。聚合成 1 kg 氘核释放约 1.07×10¹⁴ J。

能量-动量关系（能量三角形）

消去速度 υ，得到能量与动量的普适关系：

$$E^2 = E_0^2 + $pc$^2 = $m_0c^2$^2 + $pc$^2$$

能量三角形：E 是斜边，E₀ = m₀c² 是一条直角边，pc 是另一条直角边。直观！

极端情况：

  • υ ≪ c（低速）：pc ≪ E₀，E ≈ E₀ + p²/$2m₀$，回到经典动能
  • υ → c（极端相对论）：pc ≫ E₀，E ≈ pc
  • 光子：m₀ = 0，E₀ = 0，E = pc（光子能量完全来自动量）
易错：
① 动能不是 ½m₀υ²！当 υ = 0.8c 时，经典动能 = 0.32m₀c²，相对论动能 = $γ−1$m₀c² = 0.667m₀c²——差了超过一倍！高速下必须用 E_k = $γ−1$m₀c²。
② E = mc² 中的 m 是动质量 γm₀，不是静质量 m₀。对于静止物体 E₀ = m₀c²。
③ 计算核反应释放能量用静质量亏损 Δm₀（反应前后静质量之差），不是动质量之差。
④ $pc$² = E² − E₀²：已知动能和静能，可以直接求动量，不必先算速度。
一个电子（m₀ = 9.11×10⁻³¹ kg）被加速到 0.99c。（1）求 γ；（2）求相对论动能；（3）如果用经典公式算动能，相差多少倍？
参考答案 $1$ γ = 1/√$1−0.99²$ = 1/√0.0199 ≈ 7.09
$2$ E_k = $γ−1$m₀c² = 6.09 × 9.11×10⁻³¹ × $3×10⁸$² ≈ 5.0×10⁻¹³ J
$3$ 经典动能：½m₀υ² = 0.5×9.11×10⁻³¹×$2.97×10⁸$² ≈ 4.0×10⁻¹⁴ J
比值：5.0×10⁻¹³ / 4.0×10⁻¹⁴ ≈ 12.5 倍！经典公式在高速下严重低估。
写出相对论能量-动量关系，并说明对光子（m₀ = 0）的结果。
参考答案 E² = $m₀c²$² + $pc$²。
对光子 m₀ = 0，得 E = pc，即光子能量与动量成正比。光子以光速运动且无静质量，其能量完全来自动量。

How：相对论运动学解题流程

相对论运动学问题的通用四步法：

第一步：确定两个惯性参考系 -- 搞清楚谁是 S，谁是 S'，相对速度 u 是多少，沿什么方向运动。

第二步：确定一个或两个（空间分离的）事件 -- 明确事件在哪个参考系的时空坐标（x, t）或（x', t'）。

第三步：判断「原」量在哪个参考系 -- 固有时？问：「这两个事件在哪个参考系中是同一地点发生的？」那个系测的时间就是固有时。固有长度？问：「这个物体在哪个参考系中是静止的？」那个系测的长度就是固有长度。确定了就直接用 Δt = γΔt₀ 或 L = L₀/γ。

第四步：洛伦兹变换贯通全题 -- 如果题目给的是不同位置的时空坐标（两个位置+两个时间），必须用完整的洛伦兹变换，不能只用时间膨胀和长度收缩的简化公式。

终极易错清单：
① 固有时（proper time）是 Δx' = 0 的时间间隔——两个事件在同一地点发生的参考系中测得的时间间隔。问自己：「这两个事件在哪个参考系中是同一地点？」
② γ ≥ 1 → Δt = γΔt₀ 让时间间隔变大（动钟变慢 → 外力观测到的时间拉长 → 数值更大）；L = L₀/γ 让长度变小（动尺变短 → 运动方向上压缩 → 数值更小）。
③ 长度收缩只发生在运动方向上，垂直于运动方向不变。
④ 不能同时用时间膨胀和长度收缩处理同一个量——要分清哪个量在哪个参考系是「原」的，决定用哪个公式。
⑤ 速度叠加公式的分母永远不要忘了！y、z 分量的变换有额外的 √$1−u²/c²$ 因子。
⑥ 同时性是相对的（异地同时不具不变性），但因果关系是绝对的——因为信号传递速度 ≤ c。
⑦ 动能是 E_k = $γ−1$m₀c²，不是 ½m₀υ²！高速下差距极大。
⑧ 核反应释放能量用静质量亏损 Δm₀（静质量之差），不是动质量之差。
⑨ E = mc² 中 m 是动质量 γm₀，E₀ = m₀c² 是静能。
