# Chapter 4 Special Relativity 狭义相对论

## 核心公式速查

| Concept | Formula | Notes/Conditions |
|---------|---------|------------------|
| Lorentz Factor | `gamma = 1/sqrt(1-v^2/c^2)` | gamma >= 1; gamma ~ 1 when v << c |
| Lorentz Transform | `x' = gamma*(x-vt), t' = gamma*(t-vx/c^2)` | Reduces to Galilean when v << c |
| Time Dilation | `Delta_t = gamma * Delta_t0` | Moving clocks run slow; Delta_t0 is proper time (shortest) |
| Length Contraction | `L = L0/gamma` | Only along direction of motion; L0 is proper length (longest) |
| Relativistic Velocity Addition | `u_x' = (u_x - v)/(1 - u_x*v/c^2)` | Ensures nothing exceeds c |
| Relativistic Mass | `m = gamma * m0` | Increases with speed |
| Mass-Energy Relation | `E = m*c^2` | Rest energy E0 = m0*c^2 |
| Relativistic Kinetic Energy | `Ek = (gamma-1)*m0*c^2` | NOT mv^2/2! For v << c, reduces to mv^2/2 |
| Energy-Momentum Relation | `E^2 = p^2*c^2 + m0^2*c^4` | p = gamma*m0*v |
