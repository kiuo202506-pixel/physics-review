一、刚体运动的描述 页1页2页3页4页5页6页7页8页9页10页11

这是力学从「质点」走向「真实物体」的关键一步。读完本节你应能回答：刚体是什么模型？它有几种基本运动形式？如何用角量（角度、角速度、角加速度）描述转动？角量与线量如何转换？

1.1 刚体模型与运动分类 页1页2页3页4页5页6页7 必考

Hook：推门的时候，门上的每一点运动轨迹都不一样——靠近合页的点转得慢，远离合页的点转得快。但我们只需要一套「角量」（角度、角速度、角加速度）就能同时描述所有点的转动。这就是刚体转动学的第一层智慧。

刚体是理想化的力学模型：在任意外力作用下，形状和大小都不变的物体，体内任意两点间距离始终不变。刚体是特殊的质点系。刚体的基本运动形式有两种：平动（所有点轨迹完全相同）和定轴转动（所有点绕同一轴做圆周运动）。更复杂的运动（如滚动）可视作「质心平动 + 绕质心转动」的叠加。

Why：从第 2 章的质点到本章的刚体，物理模型在逼近真实世界。一个门、一个车轮、一个陀螺都不能简化为质点，但可以视作无数质点的集合——其中每对质点间距不变。这种「刚性约束」使刚体的运动自由度从 3N 降为 6（3 个平动 + 3 个转动），使精确描述成为可能。

What：

运动类型特征举例
平动 Translation体内所有点的位移、速度、加速度完全相同；任意两点连线始终平行于初始位置抽屉的推拉、电梯升降
定轴转动 Fixed-axis Rotation所有点绕同一固定轴做圆周运动；各点的角量相同，线量 $v=r\omega$ 不同门的开合、飞轮、定滑轮
平面平行运动质心平动 + 绕质心转动（二者叠加）纯滚动（车轮）、台球

How：刚体运动的「万能拆解」——任何复杂运动 = 质心的平动 + 绕质心的转动。平动用质心速度描述（第 2 章的工具），转动用角量描述（本章的工具）。这种「分解」是后面转动定律、滚动动能计算的认知基础。

易错：把「刚体定轴转动」和「质点圆周运动」混淆。质点圆周运动的向心力总是 $mv^2/r$；刚体上的质元虽然也在做圆周运动，但其向心力由刚体内力（约束力）自动提供，我们常不直接写它，而是用转动定律 $M=J\alpha$ 绕开内力的海量求解。
为什么说刚体是「特殊的质点系」？它与随意一堆离散质点（如沙粒）的根本区别是什么？
参考答案刚体内任意两点间距不变（刚性约束），自由度为 6；而 N 个离散质点的自由度为 3N。刚性约束大幅降低了问题的数学复杂度，使得用角量统一描述全体质元的运动成为可能。

1.2 角位置、角位移、角速度、角加速度 页8页9页10页11 必考

Hook：直线运动中用 $x, v, a$ 描述一个点的运动；转动中对应的「角家族」是 $\theta, \omega, \alpha$。这两套语言完全平行——你已会前者，换几个符号就能用后者。

角位置 $\theta$ 是刚体相对于参考方向的转角（逆时针为正）；角速度 $\omega=\frac{d\theta}{dt}$ 描述转动快慢，方向由右手法则确定（四指弯曲方向为转动方向，拇指指向为 $\vec{\omega}$ 方向）；角加速度 $\alpha=\frac{d\omega}{dt}=\frac{d^2\theta}{dt^2}$ 描述角速度变化快慢。匀变速转动的三条公式与匀变速直线运动一一对应。

Why：角量系统是描述「旋转类运动」的专用语言。它存在的根本原因是一个物理事实：旋转的效果取决于转了多少角（$\theta$），而不是转了多少距离。一台发动机的转速用 rpm（转/分钟）不用 m/s，正是这个道理。

What：

$$\omega = \lim_{\Delta t \to 0}\frac{\Delta\theta}{\Delta t} = \frac{d\theta}{dt}, \qquad \alpha = \frac{d\omega}{dt} = \frac{d^2\theta}{dt^2}$$

其中 $\omega$ 的单位是 rad/s，$\alpha$ 的单位是 rad/s²。方向约定：右手螺旋法则——右手四指沿转动方向弯曲时，拇指指向即为 $\vec{\omega}$ 方向（沿转轴）。

匀变速转动（$\alpha$ = 常数）与匀变速直线运动的类比：

物理量匀变速直线运动匀变速转动
速度 / 角速度$v = v_0 + at$$\omega = \omega_0 + \alpha t$
位移 / 角位移$x = x_0 + v_0 t + \frac{1}{2}at^2$$\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2$
无 t 公式$v^2 = v_0^2 + 2a$x-x_0$$$\omega^2 = \omega_0^2 + 2\alpha$\theta-\theta_0$$

记忆诀窍：把 $x \to \theta$, $v \to \omega$, $a \to \alpha$，全部照搬！

角量与线量的关系（质元 r 处）：

$$v = r\omega, \quad a_t = r\alpha, \quad a_n = r\omega^2 = \frac{v^2}{r}$$

其中 $v$ 为线速度（切向），$a_t$ 为切向加速度（改变速度大小），$a_n$ 为法向加速度（改变速度方向）。总加速度 $\vec{a} = r\alpha\,\vec{e}_t + r\omega^2\,\vec{e}_n$。

How：三类典型匀变速转动问题——$1$ 已知 $\omega_0, \alpha, t$ 求 $\theta, N$（转数）；$2$ 已知 $\omega_0, \omega, t$ 求 $\alpha$；最常考：$3$ 飞轮刹车模型 — 已知初始转速和停止时间，求角加速度、转过的圈数、某时刻边缘点的线速度/切向加速度/法向加速度。

关键计算步骤：角速度换算（rpm → rad/s：$1\,\text{r/min} = \frac{2\pi}{60} \approx 0.1047\,\text{rad/s}$）→ 用公式求 $\alpha$ → 用 $N = \theta/$2\pi$$ 换算转数 → 边缘线量用 $v=r\omega$、$a_t=r\alpha$、$a_n=r\omega^2$。

易错：$1$ 停转时 $\omega=0$ 但 $\alpha$ 未必为零（刹车瞬间角加速度仍在）；$2$ 角速度与转速的换算 $1\,\text{rpm}=2\pi/60\,\text{rad/s}$，考试中经常出现单位陷阱；$3$ 转数 $N=\theta/$2\pi$$ 不是 $\theta$——$\theta$ 是弧度，$N$ 是圈数，差 $2\pi$ 倍。
一飞轮半径 0.3 m，以 600 rpm 匀速旋转。求：$1$ 角速度；$2$ 轮缘上一点的线速度；$3$ 该点的法向加速度。
参考答案$1$ $\omega = 600 \times 2\pi/60 = 20\pi \approx 62.8\,\text{rad/s}$；$2$ $v = r\omega = 0.3 \times 20\pi = 6\pi \approx 18.85\,\text{m/s}$；$3$ $a_n = r\omega^2 = 0.3 \times $20\pi$^2 = 120\pi^2 \approx 1184\,\text{m/s}^2$，约为 120g，可见高速旋转的离心效应极大。
一飞轮从 15 rad/s 经 5 s 均匀减速到 10 rad/s，求：$1$ 角加速度；$2$ 这 5 s 内转了多少圈？
参考答案$1$ $\alpha = $10-15$/5 = -1\,\text{rad/s}^2$（负号表示减速）；$2$ 用 $\omega^2 - \omega_0^2 = 2\alpha\Delta\theta$：$\Delta\theta = $10^2-15^2$/$2\times(-1$) = 62.5\,\text{rad}$，转数 $N = 62.5/$2\pi$ \approx 10$ 圈。

这是本章的「心脏」——把牛顿第二定律 $F=ma$ 搬到转动世界中，得到 $M=J\alpha$。读完本节你应能回答：力矩如何定义？转动惯量是什么？转动定律怎么用？平行轴定理何时出手？

2.1 力矩（Torque）页17页18页19 必考

Hook：开门时，力作用在门轴附近——推不动；同样的力作用在把手处——轻松推开。这说明转动的效果不仅取决于力的大小，还取决于力的作用点和方向。力矩正是量化「力使物体转动效果」的物理量。

力矩是力对转轴的转动效果的量度。定义式 $\vec{M} = \vec{r} \times \vec{F}$，大小 $M = Fr\sin\theta$（$\theta$ 为 $\vec{r}$ 与 $\vec{F}$ 的夹角）。只有与 $\vec{r}$ 垂直的力分量（$F_\perp = F\sin\theta$）才对转动有贡献。对定轴转动，总外力矩 $M_z = \sum r_i F_{i\perp}\sin\theta_i$。

Why：力矩是转动中的「力」。在平动中 F 决定加速度 a；在转动中 M 决定角加速度 $\alpha$。力矩将力的大小、方向、作用点三者融合为一个转动效果指标——这是用单一公式 $M=J\alpha$ 描述复杂转动的关键纽带。

What：

$$\vec{M} = \vec{r} \times \vec{F}, \qquad M = Fr\sin\theta$$

其中 $\vec{r}$ 是从转轴（或参考点）到力作用点的位矢，$\theta$ 是 $\vec{r}$ 与 $\vec{F}$ 的夹角。方向由右手螺旋法则判定。力矩的单位是 N·m。

特别注意：对转轴起作用的，只有力在垂直于 $\vec{r}$ 方向的分量。平行于 $\vec{r}$ 的分力（如沿杆方向的拉力）力矩为零。

How：定性判断力矩方向——右手四指从 $\vec{r}$ 弯向 $\vec{F}$，拇指即 $\vec{M}$ 方向。定量计算：找力臂 $d = r\sin\theta$（转轴到力作用线的垂直距离），则 $M = Fd$。正负约定：使刚体逆时针转动时 $M>0$，顺时针 $M<0$。

易错：力矩 $M=FL$ 只在力垂直于位矢时才成立。力不垂直时，必须乘 $\sin\theta$ 或用「力 × 力臂」计算。不少同学直接用 $M=Fr$ 而不检查 θ 是否为 90°，这是高频失分点。
用扳手拧螺母：扳手柄长 0.3 m，施加 50 N 的力，$1$ 力垂直于手柄时力矩多大？$2$ 力与手柄呈 30° 角时力矩多大？
参考答案$1$ $M = 0.3 \times 50 = 15\,\text{N·m}$；$2$ $M = 0.3 \times 50 \times \sin 30° = 7.5\,\text{N·m}$。角度偏离 90° 后力矩减半，这就是为什么拧螺丝时要把扳手垂直施力。

2.2 转动定律 $M = J\alpha$ 页20页21页22页30页31页32页33 必考

Hook：$F=ma$ 告诉你：「力引起加速度」。那什么引起角加速度？答案是力矩。但同样的力矩作用在不同物体上效果不同——一个实心铁盘和一辆自行车轮对力矩的「抵抗」截然不同。这个抵抗能力的量度就是转动惯量 $J$。

转动定律 $M = J\alpha$：定轴转动刚体的角加速度 $\alpha$，与所受合外力矩 $M$ 成正比，与转动惯量 $J$ 成反比。这是牛顿第二定律在转动中的精确对应：$F \leftrightarrow M$，$m \leftrightarrow J$，$a \leftrightarrow \alpha$。从 $F=ma$ 推导 $M=J\alpha$ 只需对每个质元写切向分量，再对全刚体求和。

Why：转动定律是本章最重要的枢纽公式。有了它，我们就可以像用 $F=ma$ 解质点的力-运动问题一样，用 $M=J\alpha$ 解刚体的力矩-角加速度问题。定滑轮 + 重物、刹车盘、斜坡 + 滑轮等经典题型全部基于此。

What：

$$M = J\alpha$$

其中 $M$ 为合外力矩，$J = \sum \Delta m_i r_i^2$ 为转动惯量（单位 kg·m²），$\alpha$ 为角加速度。这个公式的物理图像：力矩是角加速度的「因」，转动惯量是转动中「惯性的大小」。

How：应用转动定律的标准步骤（五步法）：

$1$ 选定研究对象（刚体 / 物体+滑轮系统）；$2$ 受力分析——画隔离体图，找所有外力对转轴的力矩（注意：转轴处的约束力不产生力矩）；$3$ 列转动方程 $M_{合} = J\alpha$（刚体）+ 牛顿第二定律（质点）；$4$ 补约束方程（绳连接：$a = R\alpha$；纯滚动：$v_c = R\omega$）；$5$ 联立求解。

三类经典题型：

  • 定滑轮 + 悬挂重物：重物列 $mg - T = ma$，滑轮列 $TR = J\alpha = \frac{1}{2}MR^2\alpha$，约束 $a = R\alpha$。联立得 $a = \frac{mg}{m + M/2}$。
  • 刹车飞轮：已知飞轮 $J=\frac{1}{2}mR^2$，摩擦力 $f$，则 $M = -fR = J\alpha$ → $\alpha = -2f/$mR$$。
  • 两根绳子拉同轴双盘：总力矩 $M = F_1R_1 - F_2R_2 = $J_1 + J_2$\alpha$（注意方向正负）。
易错：$1$ 系统由刚体和质点共同组成时，不能只写转动定律而不写 $F=ma$，两类方程必须联立；$2$ 绳与滑轮间无滑动的约束 $a=R\alpha$ 是连接两套方程的桥梁，漏掉就列不出完整方程组；$3$ 绳子经过质量不同的滑轮时，两端张力不一定相等——同一条绳上张力处处相等的结论只适用于「轻绳」（$m=0$）。
质量为 M、半径为 R 的均匀圆盘定滑轮，用轻绳挂一质量为 m 的重物。系统从静止释放，求重物的加速度和绳中张力。
参考答案对 m：$mg - T = ma$。对滑轮：$TR = J\alpha$，其中 $J = \frac{1}{2}MR^2$，约束 $a = R\alpha$。代入 $TR = \frac{1}{2}MR^2 \cdot a/R$ → $T = \frac{1}{2}Ma$。联立 $mg - \frac{1}{2}Ma = ma$ → $a = \frac{mg}{m + M/2}$。$T = \frac{Mmg}{2m + M}$。若 M=0（滑轮质量忽略），则 $a=g$、$T=0$（自由落体）。

2.3 转动惯量与平行轴定理 页24页25页26页27页28页29 必考

Hook：跷跷板上坐一个小孩，与坐一个大人——同样的推力，前者飞快摆动，后者几乎不动。质量分布不同，转动的「顽固度」就不同。转动惯量 $J$ 正是这种「转动惯性」的度量，它就像转动中的「质量」——但不同于质量只取决于「多重」，转动惯量还取决于「质量分布多远」。

转动惯量 $J = \sum m_i r_i^2$（离散）或 $J = \int r^2 dm$（连续），是刚体对转轴的转动惯性量度。它取决于三个因素：总质量、质量分布（几何形状）、转轴位置。常见形状的转动惯量必须熟记。平行轴定理 $J = J_C + md^2$：刚体对任意轴的转动惯量 = 对过质心平行轴的转动惯量 + 总质量 × 两轴距离的平方。

Why：转动惯量 $J$ 在转动中的角色完全等同于质量 $m$ 在平动中的角色——它是惯性的量度。$J$ 越大，同样的力矩产生的角加速度越小。理解 $J$ 是理解「为什么质量分布影响转动难度」的关键。平行轴定理则让你只需要记住过质心的 $J_C$，就能求出任意平行轴的 $J$。

What：

$$J = \sum \Delta m_i r_i^2 \quad $\text{离散}$ \qquad J = \int r^2\,dm \quad $\text{连续}$$

其中 $r_i$ 是质元 $\Delta m_i$ 到转轴的垂直距离。连续质量分布中 $dm = \lambda dl$（线密度）、$dm = \sigma dS$（面密度）、$dm = \rho dV$（体密度）。

必背常见转动惯量（过质心轴）：

形状转轴位置转动惯量 $J$
均匀细杆（长 L）过质心，垂直杆$\frac{1}{12}mL^2$
均匀细杆（长 L）过端点，垂直杆$\frac{1}{3}mL^2$（平行轴：$\frac{1}{12}mL^2 + m$L/2$^2$）
均匀圆盘/圆柱（半径 R）过质心，垂直盘面$\frac{1}{2}mR^2$
均匀圆环（半径 R）过质心，垂直环面$mR^2$
均匀实心球（半径 R）过球心$\frac{2}{5}mR^2$
均匀薄球壳（半径 R）过球心$\frac{2}{3}mR^2$

平行轴定理：

$$J = J_C + md^2$$

其中 $J_C$ 为对过质心轴的转动惯量，$d$ 为两平行轴之间的距离。注意：$J$ 永远大于等于 $J_C$（$d^2 \ge 0$），即过质心的转动惯量最小。

How：计算转动惯量的两种方法：$1$ 积分法——取微元 $dm$，写 $dJ = r^2 dm$，在几何范围内积分；$2$ 平行轴定理——适用于轴从质心平移到任意平行位置。组合体的转动惯量 = 各部分转动惯量之和（各部分的轴必须相同）。

棒 + 盘的组合体（如摆锤）——$J_{总} = J_{棒$端点轴$} + J_{盘$平行轴$} = \frac{1}{3}m_{棒}L^2 + \left1455L55$。考试常考这种「多部件 + 平行轴」的组合计算。

易错：$1$ 转动惯量不是物体的固有属性，它依赖转轴——同一根杆，转轴过中心和过端点，$J$ 差 4 倍；$2$ 平行轴定理必须用「过质心轴的 $J_C$」加 $md^2$，不能用其他任意轴的 $J$ 加 $md^2$；$3$ $J$ 的单位是 kg·m²，不是 kg·m——$r^2$ 不要忘；$4$ 组合体时别忘了平行轴定理——盘装在杆的一端，盘的 $J$ 不只是 $\frac{1}{2}mR^2$，还要加 $m$L+R$^2$。
一根均匀细杆长 L=1 m，质量 m=2 kg。求 $1$ 转轴过杆中心垂直杆的 $J$；$2$ 转轴过端点垂直杆的 $J$；$3$ 用平行轴定理验证 $2$。
参考答案$1$ $J_C = \frac{1}{12}mL^2 = \frac{1}{12}\times 2 \times 1^2 = \frac{1}{6}\,\text{kg·m}^2$；$2$ $J_{端} = \frac{1}{3}mL^2 = \frac{2}{3}\,\text{kg·m}^2$；$3$ $J_{端} = J_C + md^2 = \frac{1}{6} + 2 \times $0.5$^2 = \frac{1}{6} + \frac{1}{2} = \frac{2}{3}\,\text{kg·m}^2$，验证通过。
均匀圆盘质量 3 kg，半径 0.4 m。求转轴在盘边缘、平行于过中心垂直盘面轴的 $J$。
参考答案过质心轴（垂直盘面）：$J_C = \frac{1}{2}mR^2 = \frac{1}{2}\times 3\times 0.4^2 = 0.24\,\text{kg·m}^2$。平行轴：$d = R = 0.4$，$J = 0.24 + 3\times 0.4^2 = 0.24 + 0.48 = 0.72\,\text{kg·m}^2$。注意这里的轴仍是垂直盘面，并非平行于盘面——轴方向不变，只是位置平移。

这是转动动力学的高潮。读完本节你应建立起「平动动量守恒 → 角动量守恒」的类比，并能用角动量守恒解释花样滑冰、跳水、陀螺等美妙现象。

3.1 角动量定理 页38页39页40页41页43 必考

Hook：平动中，动量定理告诉你「力对时间的积累等于动量的变化」。转动中，「力矩对时间的积累等于角动量的变化」——两个定理的逻辑结构完全相同，你只需做变量替换。

角动量 $\vec{L} = \vec{r} \times m\vec{v}$（质点）或 $L = J\omega$（刚体定轴转动），是转动中动量概念的对应物。角动量定理 $\vec{M} = d\vec{L}/dt$：外力矩等于系统角动量的变化率。积分形式 $\int_{t_1}^{t_2} M\,dt = L_2 - L_1$，其中 $\int M\,dt$ 称为冲量矩。

Why：角动量定理是力矩在时间维度「积累效应」的体现。就像冲量 $\int F\,dt$ 改变动量 $p$，冲量矩 $\int M\,dt$ 改变角动量 $L$。它为角动量守恒定律提供了理论基础，并为碰撞 + 转动问题提供了解题武器。

What：

质点角动量：$\vec{L} = \vec{r} \times \vec{p} = \vec{r} \times m\vec{v}$，大小 $L = rmv\sin\theta$。

刚体定轴转动角动量：$L_z = J\omega$。

角动量定理（微分形式）：$M = \frac{dL}{dt}$。

积分形式：$\int_{t_1}^{t_2} M\,dt = J\omega_2 - J\omega_1$。

对比记忆：

平动转动
动量$\vec{p} = m\vec{v}$$L = J\omega$
定理（微分）$\vec{F} = d\vec{p}/dt$$M = dL/dt$
定理（积分）$\int \vec{F}dt = \Delta\vec{p}$$\int M\,dt = \Delta L$

How：角动量定理在碰撞 + 转动问题中特别有用——当子弹射入木棒、碰撞时间极短时，用角动量定理的积分形式比逐时刻解微分方程方便得多。注意：内力矩之和为零，只有外力矩才能改变系统的总角动量。

易错：角动量和力矩必须是对同一参考点（或同一轴）定义的。向不同轴求角动量后再相加是无意义的——这与「不同方向的速度不能代数相加」是同一逻辑。
质点 m=0.5 kg 以速度 v=4 m/s 沿直线运动，该直线距 O 点距离 d=2 m。求质点对 O 点的角动量。
参考答案$L = rmv\sin\theta = dmv = 2 \times 0.5 \times 4 = 4\,\text{kg·m}^2/\text{s}$。注意质点虽做直线运动，但由于位矢与速度不平行，仍有角动量。

3.2 角动量守恒定律 页42页43页44页45页46页47 必考

Hook：花样滑冰运动员收臂后转速暴增——同样的角动量，转动惯量减小，角速度必然增大。这不是「额外的力在推动」，是角动量守恒在起作用。平动中合外力为零 → 动量守恒；转动中合外力矩为零 → 角动量守恒。两个守恒律在逻辑上是一对孪生兄弟。

角动量守恒定律：当系统所受合外力矩为零时，系统的总角动量保持不变。即 $M_{外}=0 \implies L = \text{常数}$。对定轴转动刚体，$J\omega = \text{常数}$——转动惯量增大，角速度减小；转动惯量减小，角速度增大。

Why：角动量守恒是自然界最深层的守恒律之一（来自空间的旋转对称性）。它的威力在于：只要合外力矩为零，不管系统内部如何复杂变化（收臂、碰撞、人在转盘上走），总角动量都不变。这为「碰撞 + 转动」综合题提供了最强的解题武器。

What：

$$M_{外} = 0 \implies L_{总} = \text{常矢量}$$

对定轴转动的刚体系统：$J_1\omega_1 + J_2\omega_2 + \cdots = \text{常数}$。

守恒的条件记忆要点：合外力矩为零，而非合外力为零。一个旋转的陀螺，受重力（合外力不为零），但重力对支撑点的力矩水平方向——铅直方向的角动量分量仍然守恒。

经典实例：

  • 花样滑冰收臂加速：初始 $\omega_1$ 小，$J_1$ 大（手臂伸展）→ 收臂后 $J_2$ 变小 → $J_1\omega_1 = J_2\omega_2$ → $\omega_2 = \omega_1$J_1/J_2$ > \omega_1$。角加速度 $\alpha > 0$ 但外力矩 $M=0$！不要认为有角加速度就一定要有力矩——$J$ 在变时，$\alpha = d\omega/dt = -$\omega/J$\,dJ/dt$。
  • 子弹打木棒：子弹射入悬挂棒的下端并嵌在其中——这是完全非弹性碰撞 + 角动量守恒的经典综合题。碰撞瞬间对悬挂点 O 的外力矩为零 → 对 O 轴的角动量守恒：$mv_0 l = J_{总}\omega$，其中 $J_{总} = \frac{1}{3}Ml^2 + ml^2$。
  • 人在转盘上走：人-盘系统对转轴的合外力矩为零 → $J_{人}\omega_{人} + J_{盘}\omega_{盘} = 0$（初始静止）。人相对盘走一圈，盘反向转过一定角度。解题时 $\omega_{人地} = \omega_{人盘} + \omega_{盘地}$，相对位移条件 $\theta_{人盘} = 2\pi$。

How：解题流程：$1$ 确认系统、检查外力矩是否为零；$2$ 写角动量守恒方程 $L_{初} = L_{末}$；$3$ 写出初末态各部分的 $J$ 和 $\omega$；$4$ 有碰撞时：碰撞过程角动量守恒，碰撞后若外力矩不为零则用转动定律或能量法。

易错：$1$ 角动量守恒是系统合外力矩为零，不是合外力为零——这两个条件截然不同，混淆是高频失分点；$2$ 「子弹打木棒」不能用动量守恒（悬挂点 O 有约束反力），必须用对 O 点的角动量守恒；$3$ $J$ 变化时角速度会变但角动量守恒——不要误以为角速度守恒。
花样滑冰运动员双臂平展时角速度为 2 rev/s，转动惯量为 3 kg·m²。收臂后转动惯量变为 1.5 kg·m²。求收臂后的角速度。
参考答案$J_1\omega_1 = J_2\omega_2$ → $3 \times 2 = 1.5 \times \omega_2$ → $\omega_2 = 4\,\text{rev/s}$。转速翻倍！角动量的「守恒」通过「此消彼长」实现：$J$ 减半，$\omega$ 翻倍。
质量为 m 的子弹以水平速度 $v_0$ 射入长为 l、质量为 M 的均匀细杆下端并嵌在其中，杆上端悬挂于光滑轴 O。求子弹嵌入后杆-子弹系统开始转动的角速度。
参考答案碰撞瞬间对 O 轴合外力矩为零 → 角动量守恒。碰撞前：$L_{初} = mv_0 l$（子弹对 O 的角动量）。碰撞后：$L_{末} = J_{总}\omega$，其中 $J_{总} = J_{杆} + J_{子弹} = \frac{1}{3}Ml^2 + ml^2$。由 $mv_0 l = $\frac{1}{3}Ml^2 + ml^2$\omega$ → $\omega = \frac{3mv_0}{$3m + M$l}$。

4.1 力矩做功与转动动能定理 页50页51页52页53页54页55 必考

Hook：平动中力做功 $W = \int F\,dx$，动能是 $\frac{1}{2}mv^2$。转动中只需把 $F \to M$，$x \to \theta$，$m \to J$，$v \to \omega$——力矩做功 $W = \int M\,d\theta$，转动动能 $\frac{1}{2}J\omega^2$。你会平动的，就一定会转动的。

力矩做功 $dA = M\,d\theta$，总功 $A = \int_{\theta_1}^{\theta_2} M\,d\theta$。力矩功率 $P = M\omega$。转动动能 $E_k^{旋转} = \frac{1}{2}J\omega^2$。转动动能定理 $A = \frac{1}{2}J\omega_2^2 - \frac{1}{2}J\omega_1^2$：合外力矩做功等于转动动能的增量。

Why：能量法是处理复杂转动问题（特别是非匀变速）的最强武器。当力矩随角度变化时，运动学公式失效，但动能定理仍然有效。此外，机械能守恒（转动动能 + 平动动能 + 势能）让「纯滚动 + 斜面」问题迎刃而解。

What：

$$dA = M\,d\theta, \quad A = \int_{\theta_1}^{\theta_2} M\,d\theta$$ $$P = \frac{dA}{dt} = M\omega$$ $$E_k^{旋转} = \frac{1}{2}J\omega^2$$ $$A = \frac{1}{2}J\omega_2^2 - \frac{1}{2}J\omega_1^2 \quad \text{（转动动能定理）}$$

完整平动-转动能量对照表：

平动（质点）转动（定轴）
功$A = \int \vec{F} \cdot d\vec{r}$$A = \int M\,d\theta$
动能$E_k = \frac{1}{2}mv^2$$E_k^{旋转} = \frac{1}{2}J\omega^2$
动能定理$A = \Delta E_k$$A = \Delta E_k^{旋转}$

How：使用转动动能定理时需注意：系统既有平动又有转动时，总动能 $E_k = \frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2$（质心平动动能 + 绕质心转动动能）。

刚体重力势能：$E_p = mgh_c$，其中 $h_c$ 为质心高度。即刚体的重力势能等于全部质量集中在质心时的势能——这与质点系统完全一致。

机械能守恒（含刚体的系统）：若只有保守内力（重力、弹力）做功，则 $\frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2 + mgh_c = \text{常数}$。

易错：$1$ 纯滚动物体的总动能不是 $\frac{1}{2}mv_c^2$——遗漏了转动动能 $\frac{1}{2}J\omega^2$ 是高频失分点；$2$ $\frac{1}{2}J\omega^2$ 和 $\frac{1}{2}mv^2$ 不要混用——转动用 $J,\omega$，平动用 $m,v$；$3$ 刚体重力势能是 $mgh_c$（质心高度），不是随便取某个点的高度。
均匀圆盘以 $\omega=10\,\text{rad/s}$ 绕中心轴旋转，$m=2\,\text{kg},\,R=0.3\,\text{m}$。求：$1$ 转动动能；$2$ 若要使它在 3 秒内停转，需施加多大的恒定制动力矩？
参考答案$1$ $J = \frac{1}{2}mR^2 = \frac{1}{2}\times 2 \times 0.09 = 0.09\,\text{kg·m}^2$，$E_k = \frac{1}{2}J\omega^2 = \frac{1}{2}\times 0.09 \times 100 = 4.5\,\text{J}$。$2$ 由转动动能定理 $A = M\theta = \Delta E_k$，先求 $\alpha = $0-10$/3 = -3.33\,\text{rad/s}^2$，$M = J\alpha = 0.09 \times $-3.33$ = -0.3\,\text{N·m}$。或直接用 $\Delta E_k/$\omega_{平均}t$ = -4.5/$5 \times 3$ = -0.3\,\text{N·m}$。
一均匀细杆长 L、质量 m，一端悬挂于光滑轴 O，从水平位置静止释放。求杆经过竖直位置时的角速度。
参考答案机械能守恒。初始（水平位置）：$E_p = mg$L/2$$（质心下降参考零点取在竖直位置质心处），$E_k = 0$。竖直位置：$E_p = 0$，$E_k = \frac{1}{2}J\omega^2$ 其中 $J = \frac{1}{3}mL^2$。$mg\cdot\frac{L}{2} = \frac{1}{2}\times\frac{1}{3}mL^2 \times \omega^2$ → $\omega = \sqrt{\frac{3g}{L}}$。

4.2 纯滚动 页55页56页57页58 必考

Hook：一个球在斜面滚下，它既有平动动能又有转动动能——总机械能如何分配？平动和转动各占多少？答案藏在纯滚动条件 $v_c = R\omega$ 中。

纯滚动（无滑滚动）：物体与接触面间无相对滑动。接触点相对于地面的瞬时速度为零。纯滚动条件 $v_c = R\omega$，其中 $v_c$ 是质心速度。纯滚动物体的总动能：$E_k = \frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2 = \frac{1}{2}\left$m + \frac{J_c}{R^2}\right$v_c^2$。

Why：纯滚动是平动和转动「协同运动」的典范——质心以一定速度前进，同时刚体绕质心以特定角速度旋转，使得接触点恰好静止。它统一了平动和转动，是综合考察功-能、动力学的最好题型。

What：

纯滚动约束：$v_c = R\omega$，$a_c = R\alpha$。

总动能：$E_k = \frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2$。

常见刚体纯滚动的有效惯性质量：

  • 均质圆环/薄圆筒：$J_c = mR^2$ → $E_k = mv_c^2$（平动动能=转动动能，各占一半）
  • 均质圆盘/圆柱：$J_c = \frac{1}{2}mR^2$ → $E_k = \frac{3}{4}mv_c^2$
  • 均质实心球：$J_c = \frac{2}{5}mR^2$ → $E_k = \frac{7}{10}mv_c^2$

How：纯滚动斜面问题——设斜面倾角 $\theta$，物体纯滚动下滑。用机械能守恒（无摩擦损耗时的静摩擦不做功）：$mgh = \frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2$，利用 $v_c = R\omega$ 消去 $\omega$，可直接求末速度。也可用动力学法：对质心列 $mg\sin\theta - f = ma_c$，对质心列转动方程 $fR = J_c\alpha$，加 $a_c = R\alpha$，解得 $a_c = \frac{g\sin\theta}{1 + J_c/$mR^2$}$。

关键结论：转动惯量越大的物体，纯滚动下坡加速度越小——圆环比圆盘滚得慢。这可以在斜面竞赛中直观验证。

易错：$1$ 纯滚动时有静摩擦力，但静摩擦力不做功（接触点瞬时速度为零），因此机械能守恒；$2$ 纯滚动的约束条件 $v_c = R\omega$ 只在无滑动时成立——一旦滑动，摩擦力变为滑动摩擦，$v_c \neq R\omega$，且摩擦力做负功；$3$ $\frac{1}{2}J\omega^2$ 中的 $J$ 必须是对质心轴的转动惯量（绕瞬心轴的转动惯量也可用，但要配瞬心角速度，需谨慎使用）。
一均质圆盘（m=1 kg, R=0.2 m）从高 h=0.5 m 的斜面顶端从静止纯滚动下滑。求到达底端时的质心速度。
参考答案机械能守恒：$mgh = \frac{1}{2}mv_c^2 + \frac{1}{2}J_c\omega^2$。$J_c = \frac{1}{2}mR^2$，$\omega = v_c/R$。代入：$mgh = \frac{1}{2}mv_c^2 + \frac{1}{2}\times\frac{1}{2}mR^2 \times $v_c/R$^2 = \frac{1}{2}mv_c^2 + \frac{1}{4}mv_c^2 = \frac{3}{4}mv_c^2$。$v_c = \sqrt{\frac{4}{3}gh} = \sqrt{\frac{4}{3}\times 9.8 \times 0.5} \approx 2.56\,\text{m/s}$。若是无摩擦滑动（不是滚动），$v = \sqrt{2gh} \approx 3.13\,\text{m/s}$——转动「消耗」了部分能量，部分势能转化为转动动能而非平动动能。

力矩（τ）：外界环境的作用力
力矩象征着外界环境对个人或集体的作用力，如机遇、压力、社会需求等，这些因素
推动或阻碍着人生的发展方向。

转动惯量（I）：内在属性的稳定性
转动惯量代表着个人或集体的内在属性，如能力积累、文化底蕴、制度稳定性等，它
决定了改变现状的难易程度。
例如，一个文化底蕴深厚的企业，其制度和文化具有较高的稳定性，改变方向需要较
大的外力和内部共识。
角加速度（α）：突破现状的速度
角加速度反映了突破现状的速度，即个人或集体的创新力、改革成效等，是衡量发展
快慢的重要指标。例如，科技公司通过持续创新，快速适应市场变化，实现业务的快
速增长和突破。
