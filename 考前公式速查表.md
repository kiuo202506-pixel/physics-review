# 大学物理A1 — 期末考前公式速查表（一页打印版）

## 力学 (Ch1-3)

| # | 公式 | 用途 |
|---|------|------|
| 1 | v = dr/dt, a = dv/dt | 速度/加速度定义 |
| 2 | a_t = dv/dt, a_n = v^2/R | 切向/法向加速度 |
| 3 | x=v_0*cos(theta)*t, y=v_0*sin(theta)*t - gt^2/2 | 抛体运动 |
| 4 | v = R*omega, a_t = R*alpha | 线-角量关系 |
| 5 | F_net = m*a | 牛顿第二定律(惯性系) |
| 6 | F_AB = -F_BA | 牛顿第三定律(不同物体) |
| 7 | I = integral(F*dt) = delta_p | 动量定理(力对时间积累) |
| 8 | sum(F_ext)=0 => sum(p)=const | 动量守恒(碰撞问题首选) |
| 9 | W = integral(F dot dr) | 功(力对空间积累，过程量) |
| 10 | W_net = delta_Ek = m*v2^2/2 - m*v1^2/2 | 动能定理(标量方程) |
| 11 | W_nc=0 => Ek+Ep=const | 机械能守恒(仅保守力做功) |
| 12 | Ep=mgh or k*x^2/2 or -GMm/r | 重力/弹性/引力势能 |
| 13 | I = sum(m_i*r_i^2), I=I_C+m*d^2 | 转动惯量+平行轴定理 |
| 14 | M=I*alpha (M=r x F) | 转动定律(类比F=ma) |
| 15 | L=I*omega, sum(M_ext)=0 => L=const | 角动量+守恒(收臂加速) |
| 16 | Ek = I*omega^2/2, W = integral(M*d_theta) | 转动动能+力矩做功 |
| 17 | v_c = R*omega (pure rolling) | 纯滚动不打滑条件 |

## 相对论 (Ch4)

| # | 公式 | 用途 |
|---|------|------|
| 1 | gamma = 1/sqrt(1-v^2/c^2) | 洛伦兹因子(gamma>=1) |
| 2 | x'=gamma*(x-vt), t'=gamma*(t-vx/c^2) | 洛伦兹变换 |
| 3 | delta_t = gamma*delta_t0 | 时间膨胀(固有时最短) |
| 4 | L = L0/gamma (运动方向) | 长度收缩(固有长度最长) |
| 5 | u_x'=(u_x-v)/(1-u_x*v/c^2) | 相对论速度叠加(不超光速) |
| 6 | E=m*c^2, E0=m0*c^2 | 质能关系 |
| 7 | Ek=(gamma-1)*m0*c^2 | 相对论动能(非mv^2/2!) |
| 8 | E^2=p^2*c^2+m0^2*c^4 | 能量-动量关系 |

## 振动与波 (Ch5-6)

| # | 公式 | 用途 |
|---|------|------|
| 1 | x=A*cos(omega*t+phi), omega=sqrt(k/m) | 简谐振动方程(弹簧振子) |
| 2 | T=2*pi/omega, v=-omega*A*sin(...), a=-omega^2*x | 周期+速度+加速度 |
| 3 | E=k*A^2/2 (总能量正比于A^2) | 简谐振动能量守恒 |
| 4 | A=sqrt(A1^2+A2^2+2A1A2*cos(delta_phi)) | 同方向同频振动合成 |
| 5 | u=lambda*f=omega/k | 波速-波长-频率关系 |
| 6 | y=A*cos[omega*(t-x/u)+phi] (+x方向) | 平面简谐波波函数 |
| 7 | delta_phi=2*pi*delta_x/lambda | 空间相位差(波程差->相位差) |
| 8 | 加强:delta_phi=2k*pi, 减弱:delta_phi=(2k+1)*pi | 干涉加强/减弱条件 |
| 9 | y=2A*sin(kx)*cos(omega*t) (固定端反射) | 驻波(半波损失!) |
| 10 | nu'=nu0*(u +/- v_R)/(u -/+ v_S) | 多普勒效应(分子v_R/分母v_S) |

## 热学 (Ch7-8)

| # | 公式 | 用途 |
|---|------|------|
| 1 | p*V = nu*R*T (R=8.31) | 理想气体状态方程 |
| 2 | p = n*m0*<v^2>/3 | 压强微观公式(分子碰壁统计平均) |
| 3 | m0*<v^2>/2 = 3*k*T/2 | 温度=分子平均平动动能量度 |
| 4 | E = i*nu*R*T/2 (i:自由度) | 理想气体内能(仅取决于T) |
| 5 | v_rms=sqrt(3RT/M) > v_avg > v_p | 三种速率大小关系 |
| 6 | lambda_bar = kT/(sqrt(2)*pi*d^2*p) | 平均自由程(与p成反比) |
| 7 | delta_U = Q + W (W:外界对系统做功) | 热力学第一定律 |
| 8 | 等容:W=0,Q=delta_U ; 等压:W=-p*delta_V ; 等温:delta_U=0 ; 绝热:Q=0 | 四大过程速查 |
| 9 | p*V^gamma=C, T*V^(gamma-1)=C, gamma=Cp/Cv=(i+2)/i | 绝热过程方程 |
| 10 | eta_carnot = 1 - T2/T1 (最大效率) | 卡诺效率(仅取决于热源温度) |
| 11 | dS = dQ_rev/T, delta_S_isolated >= 0 | 熵+熵增加原理 |
