# Chapter 8 Thermodynamics 热力学基础

## 核心公式速查

| Concept | Formula | Notes/Conditions |
|---------|---------|------------------|
| First Law of Thermodynamics | `Delta_E = Q + W` | Q>0: system absorbs heat; W>0: work done ON system |
| Isochoric Process (V=const) | `W=0, Q=Delta_E=nu*C_V*Delta_T` | C_V = i*R/2 |
| Isobaric Process (p=const) | `W=-p*Delta_V, Q=nu*C_p*Delta_T` | C_p = C_V + R |
| Isothermal Process (T=const) | `Delta_E=0, Q=-W=nu*R*T*ln(V2/V1)` | Ideal gas: Delta_E depends only on Delta_T |
| Adiabatic Process (Q=0) | `p*V^gamma=C, T*V^(gamma-1)=C` | gamma = C_p/C_V = (i+2)/i |
| Carnot Efficiency | `eta = 1 - T2/T1` | Maximum possible efficiency; depends only on reservoir temps |
| Carnot COP (Refrigerator) | `epsilon = T2/(T1-T2)` | Performance coefficient for refrigeration cycle |
| 2nd Law - Kelvin Statement | `Cannot convert heat entirely to work from single reservoir without other effects` | Perpetual motion machine of the 2nd kind is impossible |
| 2nd Law - Clausius Statement | `Heat cannot spontaneously flow from cold to hot body` | Note the word "spontaneously" |
| Entropy (reversible) | `dS = dQ_rev/T` | State function, unit: J/K |
| Entropy Increase Principle | `Delta_S_isolated >= 0` | Entropy of isolated system never decreases |
